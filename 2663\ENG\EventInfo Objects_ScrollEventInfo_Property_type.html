Property Type
Enum Expr Control Hidden ReadOnly Bind Collection StringResource
Description 
This property holds the cause of the event related to the scrolling in the component.
Syntax 
ScrollEventInfo.type
Setting Syntax 
var strType = e.type;
"first" 
When the scrolling position reaches the minimum point
"firstover" 
When scrolling is attempted at the minimum scrolling position
"last" 
When the scrolling position reaches the maximum point
"lastover" 
When scrolling is attempted at the maximum scrolling position
"lineup" 
When scrolled upward at intervals of a line in vertical scrolling
"linedown" 
When scrolled downward at intervals of a line in vertical scrolling
"lineleft" 
When scrolled leftward at intervals of a line in horizontal scrolling
"lineright" 
When scrolled rightward at intervals of a line in horizontal scrolling
"pageup" 
When scrolled upward at intervals of a page in vertical scrolling
"pagedown" 
When scrolled downward at intervals of a page in vertical scrolling
"pageleft" 
When scrolled leftward at intervals of a page in horizontal scrolling
"pageright" 
When scrolled rightward at intervals of a page in horizontal scrolling
"track" 
When scrolled by a track button regardless of the direction
"trackfirst" 
When scrolled by a track button, and the scrolling position reaches the minimum point
"trackfirstover" 
When scrolled by a track button, and scrolling is attempted at the minimum scrolling position
"tracklast" 
When scrolled by a track button, and the scrolling position reaches the maximum point
"tracklastover" 
When scrolled by a track button, and scrolling is attempted at the maximum scrolling position
"trackstart" 
When scrolling starts by using a track button
"trackend" 
When scrolling ends by using a track button
"wheelup" 
When scrolled upward by a mouse wheel
"wheeldown" 
When scrolled downward by a mouse wheel
"wheelfirst" 
When the scrolling position reaches the minimum point by using a mouse wheel
"wheelfirstover" 
When scrolling is attempted at the minimum scrolling position by using a mouse wheel
"wheellast" 
When the scrolling position reaches the maximum point by using a mouse wheel
"wheellastover" 
When scrolling is attempted at the maximum scrolling position by using a mouse wheel
"slidefirst" 
When the scrolling position reaches the minimum point due to a sliding or fling action
"slidefirstover" 
When scrolling is attempted at the minimum scrolling position by a sliding or fling action
"slidelast" 
When the scrolling position reaches the maximum point due to a sliding or fling action
"slidelastover" 
When scrolling is attempted at the maximum scrolling position by a sliding or fling action
"slidescrollup" 
When scrolled upward by a sliding or fling action
"slidescrolldown" 
When scrolled downward by a sliding or fling action
"slidescrollright" 
When scrolled rightward by a sliding or fling action
"slidescrollleft" 
When scrolled leftward by a sliding or fling action
"none" 
When scrolled by any undefined action

For example, this property has "none" as its value when the value of a position-related property is changed by the use of a method. Another example is when a component is scrolled because pressing an arrow key causes the change in selection.
