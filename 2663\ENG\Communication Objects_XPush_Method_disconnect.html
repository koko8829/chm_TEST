Description 
This method is used to disconnect from an X-PUSH server.
Syntax 
XPush.disconnect()
Parameters
Sample Call:
this.XPush00.disconnect();
Return
None
Remark 
-If this method successfully breaks the connection with an X-PUSH server, the onsuccess event will occur.
 Otherwise, the onerror event will occur.
