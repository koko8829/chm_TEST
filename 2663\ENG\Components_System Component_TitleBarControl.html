TitleBarControl
Supported Environments
[Desktop]
Nexacro Runtime Environment (NRE)
Windows macOS
Web Runtime Environment (WRE)
Edge Chrome Safari Firefox Opera
[Mobile]
Nexacro Runtime Environment (NRE)
Android iOS/iPadOS
Web Runtime Environment (WRE)
Android Default Web Browser iOS/iPadOS Default Web Browser
Description 
It is Component which consists of StatusBar of Frame.
Remark 
It is seen when showstatusbar of each Frame is set.
maxbutton and restorebutton are seen according to status of each frame.
Structure
Property
Name Description
background The attribute to set the background area of the TitleBarControl.
border-radius This property is used to set the shape of the corners of a TitleBarControl.
buttongap This property is used to set the gap between the system buttons on a TitleBarControl.
closebutton This read-only property holds a Close button, one of the system buttons placed on a TitleBarControl.
color This property is used to set the color of the text displayed in a TitleBarControl.
cursor This property is used to set the type of a mouse pointer that appears when the mouse cursor is located in a TitleBarControl.
font This property is used to set the font used in TitleBarControl.
letter-spacing This property is used to set the spacing between characters (kerning) displayed in a TitleBarControl.
maxbutton This read-only property holds a Maximize button, one of the system buttons placed on a TitleBarControl.
minbutton This read-only property holds a Minimize button, one of the system buttons placed on a TitleBarControl.
name This property is used to specify the name of a TitleBarControl.
-nexa-border This property is used to set the border of a TitleBarControl.
-nexa-edge This property is used to set the image of the edges that are displayed inside the border of an TitleBarControl.
-nexa-padding This property is used to set the padding, an inner margin, of a TitleBarControl.
-nexa-rtl-background-image This property is used to set an image that will be displayed as the background of a TitleBarControl. This property is applicable when the rtl property of the TitleBarControl is set to "true".
-nexa-rtl-edge-image This property is used to set an edge image that is displayed inside the border of a TitleBarControl. This property is applicable when the rtl property of the TitleBarControl is set to "true".
-nexa-text-align This property is used to set the horizontal alignment of the text or contents displayed in a TitleBarControl.
-nexa-vertical-align This property is used to set the vertical alignment of the text or contents displayed in a TitleBarControl.
normalbutton This read-only property holds a Restore button, one of the system buttons placed on a TitleBarControl.
opacity This property is used to set the opacity of a TitleBarControl.
word-spacing This property is used to set the spacing between words displayed in a TitleBarControl.
