Description 
The Div is a component used to compose multiple subscreens on one screen.

A screen can be composed independently or connected with the main screen if it is composed with the Div.
The Div can be used in connection with other forms.
The Div areas can be divided using borders or scroll bars.
Remark 
- The Div can make screen composition easy and increase the reusability of a form by dividing the screen.
  However, if the number of stages of the Div increases, the screen composition slows down and the initialization time increases.

- It is efficient to use the Div with a depth of around 2 to 3 stages.
  It is faster to use the Div with 2 to 3 stages on one screen than with 4 to 5 stages when composing a screen.

Constructor 
Div.init( strName, vLeft, vTop , vWidth, vHeight [, vRight, vBottom, [vMinWidth, [vMaxWidth, [vMinHeight, [vMaxHeight]]]]] )
Parameters Type Description
strName String 
It sets the ID of a Div in the form of a string.
vLeft String, Number 
It specifies the value of the left property of a Div in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the right property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vTop String, Number 
It specifies the value of the top property of a Div in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the bottom property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vWidth String, Number 
It specifies the value of the width property of a Div in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vHeight String, Number 
It specifies the value of the height property of a Div in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vRight String, Number 
It specifies the value of the right property of a Div in pixel or percent (%).

This parameter will be ignored if both the vLeft and vWidth parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the left property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vBottom String, Number 
It specifies the value of the bottom property of a Div in pixel or percent (%).

This parameter will be ignored if both the vTop and vHeight parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the top property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vMinWidth String, Number 
It specifies a Div's minimum width that can be displayed in the Form.
vMaxWidth String, Number 
It specifies a Div's maximum width that can be displayed in the Form.

If this value is smaller than vMinWidth, it will be replaced by the value of vMinWidth.
vMinHeight String, Number 
It specifies a Div's minimum height that can be displayed in the Form.
vMaxHeight String, Number 
It specifies a Div's maximum height that can be displayed in the Form.

If this value is smaller than vMinHeight, it will be replaced by the value of vMinHeight.
Sample Call:
var objComp = new Div();

objComp.init( "Div00", 30, 120, 196, 46 );
objComp.init( "Div00", 30, 120, 196, 46, null, null );
objComp.init( "Div00", null, null, 300, "400px", "80%", 300 );
objComp.init( "Div00", 0, 0, 200, 100, null, null, 300, 500, 200, 500 );
objComp.init( "Div00", "Div22:10", 300, null, null, "Div33:10", "20%", 300, 500, 200, 500 );
 
// Create Object  
var objDiv = new Div();  
objDiv.init("Div", 30, 120, 196, 46, null, null);
objDiv.init("Div", null, null, 300, "400px", "80%", 300);
or
var objDiv = new Div("Div", 30, 120, 196, 46, null, null);
var objDiv = new Div("Div", null, null, 300, "400px", "80%", 300);
 
// Add Object to Parent Form  
this.addChild("Div", objDiv); 
 
// Insert Object to Parent Form  
this.insertChild(1, "Div", objDiv); 
 
// Remove Object form Parent Form  
this.removeChild("Div"); 
 
// Show Object  
objDiv.show(); 
 
// Destroy Object  
objDiv.destroy(); 
objDiv = null;

getOffsetRight This method is used to obtain the right position of a Div in pixel, based on the parent component's left side.
