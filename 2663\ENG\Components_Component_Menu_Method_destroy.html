Description 
This method is used to delete a Menu that is dynamically created through a script.
Syntax 
Menu.destroy()
Parameters
Sample Call:
var bSucc = this.Menu00.destroy();
Return
Type Description
Boolean 
It will return "true" if a Menu is successfully removed.

Otherwise, it will return "false".
Remark 
- You should call the destroy() method to completely remove a dynamically-created Menu.
