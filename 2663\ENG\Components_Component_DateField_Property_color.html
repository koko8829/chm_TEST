Description 
This property is used to set the color of the text displayed in a DateField.
Syntax 
DateField.color[= strcolor]
Setting Syntax 
strcolor ::= <NamedColor> | <NumericColor>
 
* XCSS
color : #999999;
color : rgb ( 255, 0, 0 );
color : red;

* Script ( normal property )
this.DateField00.color = "#999999";
this.DateField00.color = "rgb(255,0,0)";
this.DateField00.color = "red";
<NamedColor> 
You can specify a predefined color name in the form of a string like "red" and "blue".
<NumericColor> 
This is a code that indicates the color to display.

The code is a hexadecimal values that begins with "#" and comprises the following 6-digit value (eg. #FF00FF).
You can specify a color code with the rgb() method whose parameters are red, green and blue values. (eg. rgb(255,0,255))
You can specify a color code with the rgba() method whose parameters are red, green, blue and alpha values. (eg. rgba(255,0,255,0.3))
You can specify a color code with the hsl() method whose parameters are hue, saturation and lightness values. (eg. hsl(120,50%,50%))
You can specify a color code with the hsla() method whose parameters are hue, saturation, lightness and alpha values. (eg. hsla(120,50%,50%,0.8))
Remark 
- If you do not set this property, the property's value will be undefined with the property working as if its value is #000000.
