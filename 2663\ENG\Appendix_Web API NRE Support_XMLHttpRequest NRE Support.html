XMLHttpRequest NRE Support
Overview 
The XMLHttpRequest object can be usued in the NRE.
Describes whether functions (properties, methods) are supported depending on the operating system running the app.

* Please refer to the link below for whether the XMLHttpRequest object function is supported according to the browser version in the Web Runtime Environment.
https://developer.mozilla.org/en-US/docs/Web/API/XMLHttpRequest#browser_compatibility

For detailed information on each function, please refer to the specification document below:
https://xhr.spec.whatwg.org/
Restrictions in iOS/iPad NRE 
The XMLHttpRequest object cannot be used when the Update Type is specified as "Update" or "Local" when packing the app.
According to WKWebView policy, files stored locally cannot handle XMLHttpRequest communication (an error occurs when executed).
Whether to support the functions below is only applicable if the Update Type is set to "Server".
