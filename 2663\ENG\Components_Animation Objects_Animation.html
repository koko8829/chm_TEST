Description 
This object applies animation effects to the components registered as items.
Remark 
- You cannot create an Animation object when you are designing an application through Nexacro Studio.
   You should first create the object dynamically through a script and then add the object with the addChild() method.
   You must specify the class "nexacro." when writing a statement to create an Animation object dynamically.

- You can register multiple AnimationItem objects to an Animation object by calling the addTarget() method.

- When an Animation object is executed, animation effects are applied simultaneously to the components registered as items.
Constructor 
Animation( strName, objParent )
 
// Create Object  
var objAni = new nexacro.Animation("Animation00", this);  

// Add Object to Parent Form  
this.addChild( "Animation00", objAni );

// Set Object 
objAni.duration = 1000;
objAni.easing = "linear";
objAni.setEventHandler( "onbegin", this.Animation00_onbegin, this);

// Add Item to Object
objAni.addTarget( "AniItem00", this.Button00, "left:600" );

// Play Object
objAni.play();
 
// Remove Object form Parent
var objAniItem = objAni.removeTarget( "AniItem00" );
this.removeChild( "Animation00" );
 
// Destroy Object
objAniItem.destroy();
objAni.destroy();
objAniItem = null;
objAni = null;
Property
Name Description
beziercurve This property is used to set a Bezier curve applied to an animation. This property is applicable only when the easing property is set to "custom".
delay This property is used to specify the length of a delay to an animation.
direction This property is used to specify the direction of applying an animation.
duration This property is used to specify the duration of an animation.
easing This property is used to specify an interpolation function applied to an animation.
elasticity This property is used to specify elasticity applied to an animation.
id This property is used to specify the ID of an Animation object.
items This read-only property holds all the AnimationItem objects that are register to an Animation object.
loop This property is used to specify whether to repeat an animation.
loopcount This property is used to specify the number of times that an animation is repeated with the loop property set to "true".
name This property is used to specify the name of an Animation object.
parent This read-only property holds the parent object of a Animation.
Method
Name Description
addTarget This method is used to designate a component that will be a target of an animation. The component will be added as an item of an Animation object.
destroy This method is used to delete a Animation that is dynamically created through a script.
pause This method is used to pause an ongoing animation.
play This method is used to start an animation.
removeTarget This method is used to remove an AnimationItem object that is added as an item of an Animation object.
reverse This method is used to reverse the direction of an animation.
seek This method is used to move the playback position of an animation to a certain point.
stop This method is used to stop an ongoing animation.
Event
Name Description
onbegin This event occurs when an animation begins to take effect.
oncomplete This event occurs when an animation finishes.
onrun This event occurs for every tick when the animation is being performed.
onupdate This event occurs for every tick after the calling of the play() method and until the animation finishes.
