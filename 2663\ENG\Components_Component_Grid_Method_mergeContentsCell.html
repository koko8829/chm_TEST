mergeContentsCell

Description 
This method is used to merge cells based on the Grid format as displayed.
Syntax 
Grid.mergeContentsCell( enumBand, nStartSubRow, nStartCol, nEndSubRow, nEndCol, nFirstCell, bKeepSubCell )
Parameters
Parameters Type Description
enumBand String 
This parameter is a string representing the band that includes target cells.

If you specify "head", the head band includes the cells.
If you specify "body", the body band includes the cells.
If you specify "summary", the summary band includes the cells.
nStartSubRow Number 
This is the sub-row index number of the top-left most cell among those will be merged.

The sub-row index is based on the band designated by the enumBand parameter.
If a specified index number is invalid, it will be replaced with the closest valid value.
nStartCol Number 
This is the column index number of the top-left most cell among those will be merged.

The column index is based on the whole Grid.
If a specified index number is invalid, it will be replaced with the closest valid value.
nEndSubRow Number 
This is the sub-row index number of the bottom-right most cell among those will be merged.

The sub-row index is based on the band designated by the enumBand parameter.
If a specified index number is invalid, it will be replaced with the closest valid value.
nEndCol Number 
This is the column index number of the bottom-right most cell among those will be merged.

The column index is based on the whole Grid.
If a specified index number is invalid, it will be replaced with the closest valid value.
nFirstCell Number 
This is the index number of a cell that will be a base for the properties of the merged cell.

The cell index is based on the band designated by the enumBand parameter.
bKeepSubCell Boolean 
If you specify "true", original cells will maintain their own properties while sharing the index number of the resulting merged cell.
If you specify "true", the background property of the cell defined by the nFirstCell parameter will apply to all the sub-cells.

If you specify "false", the merged cell will be treated as one cell so that the index number and properties will be managed as a batch.
If you specify "false", all the properties of the cell defined by the nFirstCell parameter will be applied.
Sample Call:
var nCell = this.Grid00.mergeContentsCell("body",0,3,0,4,3,true);
Return
Type Description
Number 
This method returns the index number of the merged cell, calculated based on the band where the cell belongs.

If this method fails to merge cells, it will return -1.
