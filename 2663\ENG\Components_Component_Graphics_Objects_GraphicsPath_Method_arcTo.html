arcTo

Description 
This method is used to add a path of an elliptical arc to a GraphicsPath. The arc stretches from the current position to the position delivered as arguments.
Syntax 
GraphicsPath.arcTo(  nX, nY, nRadiusX, nRadiusY, nRotation, bClockwise, bLarge )
Parameters
Parameters Type Description
nX Number 
It specifies a pixel to designate the x-coordinate of a point to add to the path.
nY Number 
It specifies a pixel to designate the y-coordinate of a point to add to the path.
nRadiusX Number 
It specifies the horizontal radius of an elliptical arc in pixel.
nRadiusY Number 
It specifies the vertical radius of an elliptical arc in pixel.
nRotation Number 
It specifies an angle at which the x-axis will rotate to draw an elliptical arc.

If you specify a 180 degree, the result will be the same as a 0 degree.
bClockwise Boolean 
It specifies the direction of drawing elliptical arcs.

For "true", the GraphicsControl will draw arcs clockwise, based on the straight line from the current position to the position represent by the values of nX and nY.
For "false", the GraphicsControl will draw arcs counterclockwise, based on the straight line from the current position to the position represent by the values of nX and nY.
bLarge Boolean 
It chooses between the bigger arc and smaller arc.

For "true", the GraphicsControl will display the bigger arc in the direction indicated by bClockwise.
For "false", the GraphicsControl will display the smaller arc in the direction indicated by bClockwise.
Sample Call:
var objGPath = new nexacro.GraphicsPath();

objGPath.moveTo( 100,100 );
objGPath.arcTo( 100, 50, 0, false, true, 200, 200 );
Return
None
Remark 
- After the execution of arcTo(), the current position will change to the position represented by nX and nY.

- The path may not be displayed properly if you do not specify the beginning point of a GraphicsPath.

- The path of an elliptical arc are determined through the below process.
  1. The arc stretches from the current position to the position represented by nX and nY.
  2. The arc has the values of nRadiusX and nRadiusY as its horizontal and vertical radiuses respectively.
  3. The x-axis of the arc rotates at an angle indicated by nRotation.
  4. Proceeding up to the step 3 will result in two or four elliptical arc paths.
  5. One of them will be chosen finally, determined by the values of bClockwise and bLarge.
