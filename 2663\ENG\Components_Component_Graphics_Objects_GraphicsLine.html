Description 
This object is used to display a straight line in a Graphics component.
Remark 
- You can create this object dynamically through a script. You cannot design this object in Nexacro Studio. 

- This object is designed to display a straight line conveniently.

- This object is basically same as the GraphicsPath object while it has the x1, y1, x2 and y2 properties to specify a straight line.
Property
Name Description
id This property is used to set the identification of a GraphicsLine.
opacity This property is used to set the opacity of a GraphicsLine.
parent This read-only property holds the parent object of a GraphicsLine.
strokecap This property is used to set the cap of a straight line displayed in a GraphicsLine.
strokepen This property is used to set the color and type of the straight line of a GraphicsEllipse.
visible This property is used to determine whether to make a GraphicsLine visible or not.
x This property is used to set the x-coordinate of a position where a GraphicsLine is displayed. The coordinate is calculated based on the position of the parent object.
x1 This property is used to set the x-coordinate of a position where a straight line begins.
x2 This property is used to set the x-coordinate of a position where a straight line ends.
y This property is used to set the y-coordinate of a position where a GraphicsLine is displayed. The coordinate is calculated based on the position of the parent object.
y1 This property is used to set the y-coordinate of a position where a straight line begins.
y2 This property is used to set the y-coordinate of a position where a straight line ends.
Method
Name Description
destroy This method is used to completely remove a GraphicsLine from a memory.
getBoundRect This method returns the area where a GraphicsLine is displayed, including the border of the GraphicsLine. The area is calculated based on the Graphics component. The border is set by the strokepen property.
getNextSibling This method returns the next object among those under the same parent as a GraphicsLine.
getOffsetBoundRect This method returns the area where a GraphicsLine is displayed, including the border of the GraphicsLine. The area is calculated based on the parent object. The border is set by the strokepen property.
getOffsetRect This method returns the area where a GraphicsLine is displayed, with the area calculated based on the parent object.
getOffsetRegion This method returns the area where a GraphicsLine is displayed, excluding the border of the GraphicsLine. The area is calculated based on the parent object. The border is set by the strokepen property.
getPreviousSibling This method returns the previous object among those under the same parent as a GraphicsLine.
getRect This method returns the area where a GraphicsLine is displayed, with the area calculated based on the Graphics component.
getRegion This method returns the area where a GraphicsLine is displayed, excluding the border of the GraphicsLine. The area is calculated based on the Graphics component. The border is set by the strokepen property.
isHitTest This method finds the position marked by the coordinates delivered as arguments and returns whether the position is included in the area of a GraphicsLine.
isPointInPath This method returns whether the position of a certain point is included in the area of a GraphicsLine.
rotate This method is used to rotate a GraphicsLine.
scale This method is used to scale a GraphicsLine up/down.
setClipRect This method is used to set coordinates for clipping a certain region of a GraphicsLine.
setTransform The method to apply the transform-related effect consecutively based on the original location of the GraphicsLine.
shear This method is used to shear a GraphicsLine.
toString This method returns a string representing the type that constitutes a GraphicsLine.
transform This method is used to apply transformational methods to a GraphicsLine.
translate This method is used to move the position of a GraphicsLine.
