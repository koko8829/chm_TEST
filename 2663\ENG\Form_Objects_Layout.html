Description 
This object is a subordinate element of a Form.
Remark 
- At least one Layout object will be defined automatically while multi-layout structure refers to multiple Layout objects defined under a Form.

- The properties under a Layout object are listed in the "Layout Information" group, which appears on the Properties pane. If you set those properties, the values will be applied to the relevant Layout object.
 As for the multi-layout structure, the "Layout Information" group will show the information on a currently-selected layout.

- You can set layout-related properties only in Nexacro Studio while those properties cannot be modified in the process of executing an application.

- The below steps will be applied when Nexacro selects a Layout to apply to a selected screen.

  1. Nexacro will choose a Layout whose value of the screenid property matches the ID of the selected screen.
  2. If more than two Layouts are selected as a result of Step1, the platform will choose a Layout whose width shows the narrowest difference compared with the selected screen.
  3. If more than two Layouts are selected as a result of Step2, the platform will choose a Layout whose height shows the narrowest difference compared with the selected screen.
  4. If more than two Layouts are selected as a result of Step3, the platform will choose the Layout that is defined first.

  * If no Layout is found in each step, the default Layout will be applied.
  * In the mobile environment, a Layout can be selected only when the orientation of an actual device is the same as the one defined in the Layout.

- In the portrait orientation, the resolutions of the "mobile_small", "mobile_medium", and "mobile_large" screens are 480x768, 640x1024 and 800x1280 respectively.
Property
Name Description
description This property is used to define the user description of a Layout defined in a Form.
flexcrossaxisalign This property is used to set the alignment method of components based on the cross axis (the vertical axis, if the type property value is "horizontal" and the the horizontal axis if the type property value is "vertical") within the Layout container.
flexcrossaxiswrapalign This property is used to set the alignment method of components based on the cross axis (the vertical axis, if the type property value is "horizontal" and the the horizontal axis if the type property value is "vertical") within the layout container, when the flexwrap property value is set to "wrap".
flexmainaxisalign This property is used to set the alignment method of components based on the main axis (the horizontal axis, if the type property value is "horizontal" and the vertical axis if the type property value is "vertical") within the layout container.
flexwrap This property is used to set whether or not to wrap when the component is placed out of the layout container area.
height This property is used to specify the height of the current layout defined in a form.
horizontalgap This property is used to set the horizontal axis spacing of components (or cells) placed in the layout container.
initvalueid This property is used to set the ID of an InitValue to be applied to an Layout.
mobileorientation This property is used to set the orientation for a layout that will be displayed in a mobile device.
name This property is used to define the name of a Layout.
screenid This property is used to set the IDs of screens to which a layout will apply.
spacing This property is used to set the space between the layout container and the component.
stepcount This property holds the number of step pages that comprise a layout.
stepindex This property holds the index of the current step page within a certain layout.
tablecellalign This property is used to set the alignment method of virtual table cells within the layout container.
tablecellincompalign This property is used to set the arrangement method of components within the virtual table cell area in the layout container.
tabledirection This property is used to set the direction in which components are aligned within the virtual table cell in the layout container.
tabletemplate This property is used to set the virtual table cells within the layout container.
tabletemplatearea This property is used to set the name of a virtual table cell in the layout container.
type This property is used to set the criteria for placing components within the layout container.
verticalgap This property is used to set the vertical axis spacing of components (or cells) placed in the layout container.
width This property is used to specify the width of the current layout defined in a form.
