Description 
The attribute to set to be able to change the column order by dragging the head area of the column in the Grid.
Syntax 
Grid.cellmovingtype[= enumMovingtype]
Setting Syntax 
this.Grid00.cellmovingtype = "col";
"none" 
You cannot change the order of columns.
"col" 
You can change the order of columns in a Grid by dragging a column from the head band.

You can change the order regardless of the value of the band of a GridColumnControl.
"col,band" 
You can change the order of columns in a Grid by dragging a column from the head band.

A column can move within the same band.
A column whose band property is set to "body" cannot move to the areas whose band properties' values are "left" or "right".
Remark 
- It is applied as “none” if the cellmovingtype attribute value is not set.

- The entire column can be moved by selecting the head area of the grid with the mouse and dragging it to the desired place.
   Only one column can be moved at a time.

- The column cannot be moved if there is a merged cell or a sub cell in the column.

- This attribute is applied only when the selectscrollmode attribute value of the Grid is “select”.
   If the selectscrollmode attribute value is not set in the mobile, it is applied as “scroll” and the cellmovingtype attribute value is not applied.

- If the attribute value is “col”, the ondrag event occurs, which is bubbled as parent. 
  ondragmove, ondragenter, ondragleave, and ondrop events do not occur.
