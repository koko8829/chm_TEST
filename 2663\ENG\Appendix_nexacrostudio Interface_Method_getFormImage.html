getFormImage
Description 
The method to request, return, and process the design image of a Form file.
It can be used only when it is executed in the Add on type inside of Nexacro Studio.
It can be used to check or print the design image of a Form file in advance.
The returned value after the execution of the method is a string type in the image data string form that is encoded with Base64.
The prefixurl path used as the parameter can be checked in the prefixurl value after calling the nexacrostudio.request(requesttype.forms) method.
Syntax 
nexacrostudio.getFormImage( strFormurl )
Parameters 
	

		Parameters
		Description
	


	

		
strFormurl

		
Sets the url of the form to bring the image.
Use the prefixurl path.
Return 
	

		Return Type
		Description
	


	

		
String

		
Returns the image of the Form execution screen requested as the strFormurl parameter value in the image data string format encoded by Base64
