Property Type

Description 
This property sets the default input language to be used in the edit window when subcell is in the MultiCombo format.
Syntax 
CellSubCellControl.multicomboimemode[= enumImemode]
Setting Syntax 
enumImemode ::= 'none' | 'alpha' | 'alpha,full' | 'hangul' | 'hangul,full' | 'katakana' | 'katakana,full' | 'hiragana' | 'direct'
 
this.Grid00.setSubCellProperty( "body", 0, 0, "multicomboimemode", "alpha" );
"none" 
This does not change the default input language and retains the language set in the current system.
"alpha" 
This sets the input language to English.

This is applied only when the IME is in East Asian languages (Korean/Japanese/Chinese Characters).
"alpha,full" 
This sets the input language to English fullwidth.

This is applied only when the IME is in East Asian languages (Korean/Japanese/Chinese Characters).
"hangul" 
This sets the input language to Korean.

This is applied only when the IME is in Korean.
"hangul,full" 
This sets the input language to Korean fullwidth.

This is applied only when the IME is in Korean.
"katakana" 
This sets the input language to Japanese kana.

This is applied only when the IME is in Japanese.
"katakana,full" 
This set the input language to Japanese kana fullwidth.

This is applied only when the IME is in Japanese.
"hiragana" 
This set the input language to Japanese hiragana fullwidth.

This is applied only when the IME is in Japanese.
"direct" 
This sets the input language to Japanese direct input.

This is applied only when the IME is a IME2002 Japanese.
This is applied as “alpha” if the IME is IME2007 and 2010.
Remark 
- If the multicomboimemode property value is not set, it is applied as "none".

- The IME is reset when the subcell receives focus and the editing window is activated.

- This property is supported only in some environments, so check the supported environments above.
   > This property is supported only in Windows NRE, Android NRE.
   > This property is not supported in macOS NRE, iOS/iPadOS NRE, or WRE.
