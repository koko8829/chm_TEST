-nexa-edge

Property Type
Enum Expr Control Hidden ReadOnly Bind Collection StringResource
Description 
This property is used to set the image of the edges that are displayed inside the border of an ListBoxItemControl.
Syntax 
ListBoxItemControl.-nexa-edge
Setting Syntax 
-nexa-edge ::= 'none' | <image> [<fixedwidth>'px' <fixedheight>'px']

<image> ::= 'URL("' <Theme-Image> | <Web-Image> '")'
 
* XCSS
-nexa-edge : URL("./images/border.png") 5px 5px;
"none" 
It does not specify the image of the edges.
<image> 
It designates an image from a theme file or the Web, with the format "URL('[image location]').

The image is displayed in the form of the edges inside the border.
<Theme-Image> 
It designates an image that is declared in a theme file, with the format "theme://images\image name"

The target image must be registered in the theme.
<Web-Image> 
It designates an image that is located in the Web, with the format "http://path/image name" or "https://path/file name".

Your environment must be web-accessible.
<fixedwidth> 
It specifies the size of the left/right side where an image will be fixed. The size will be specified in pixel.
If the size is specified as 0px, the image will not fixed.

The default value is 0px.
<fixedheight> 
It specifies the size of the top/bottom side where an image will be fixed. The size will be specified in pixel.
If the size is specified as 0px, the image will not fixed.

The default value is 0px.
Remark 
- The default value of this property is "undefined" while it behaves as "none".

- This property is used to set the image of the edges that are displayed inside the border of a ListBoxItemControl.

- To set this property, you should fix certain areas of top/bottom/left/right side  (values of <fixedwidth> and <fixedheight>). Then, you should stretch the inner rectangle area.

- Even if background image or color is already set by the background property, the image that is designated by the -nexa-edge property will top it.


◆ Restrictions in the web runtime environment

- The presentation of the image may differ according to web browsers if the values of <fixedwidth> and <fixedheight> exceeds the half of the image's size.

- This property may not function properly in the web browser versions that do not support CSS3.
