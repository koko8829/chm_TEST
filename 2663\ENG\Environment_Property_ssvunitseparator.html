Description 
The attribute to set the column delimiter of the SSV data when communicating in SSV format.
Syntax 
Environment.ssvunitseparator
Setting Syntax 
"33"
"0x1E"
"33,0x1E,34"
"0x1E,0x1F"
"33,34,35,36,37"
ssvunitseparator 
Set the ASCII character in decimal or hexadecimal form to use it as a column delimiter for data.

-ASCII characters in the range of 0x00(0) to 0x7F(127) can be set by combining up to 5 characters using commas (",").
Values greater than 0x80(128) cannot be set and if setting more than 5 characters, it is applied only up to 5 characters.

- Among the ASCII characters in the range of 0x00(0) to 0x7F(127), you cannot use the characters below because they are being used by the SSV or used as Tokens in the product.
However, the characters below can be included when using two or more characters in combination.
0x00(00) - NUL  (Null char.)
0x03(03) - ETX  (End of Text)
0x1D(29) - GS  (Group Separator)
0x28(40) - ( (left opening parenthesis)
0x29(41) - ) (right closing parenthesis)
0x3A(58) - : (colon)
0x3D(61) - = (equal sign)
Remark 
- This property can be set only in Nexacro Studio and cannot be modified during execution of the application.

- The EmptyString is set and the ASCII character 0x1F(31)-US is used as delimiter if the ssvunitseparator attribute value is not set.
