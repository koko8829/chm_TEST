Description 
This object represents the system tray used in an application. Multiple trays can be registered.
Constructor 
Tray.init(strID, icon, tooltip);
Parameters Type Description
strID String 
It sets the ID of a dynamically-created Tray.
icon String 
It sets an icon that will be used in the Tray.
tooltip String 
It sets the text that will be displayed as a tooltip in the Tray.
Sample Call:
var objApp = nexacro.getApplication() ;
var objTray = new nexacro.Tray("Tray00", objApp);

objTray.init("Tray00", "information", "Tray Tooltip");

var nIndex = objApp.addTray("Tray00", objTray);
objTray.show();
 
// Get Appication
var objApp = nexacro.getApplication();

// Create Object
var objTray = new nexacro.Tray("Tray00", objApp);

// Init Object	
objTray.init("Tray00", "information", "Tray ToolTip");

//Add Object to Application
var nIndex = objApp.addTray("Tray00", objTray);

// Show Object
objApp.trays["Tray00"].show();

// Remove Object from Application
objApp.removeTray("Tray00"); 
 
// Destroy Object 
objTray.destroy(); 
objTray = null;
Property
Name Description
icon This property is used to set an icon image on the tray.
items This read-only property holds the TrayPopupMenu objects registered in a Tray.
name This property is used to specify the name of a Tray.
tooltip This property is used to specify the text of a tooltip that apeears when a mouse pointer hovers over a tray icon.
Method
Name Description
addItem This method is used to add a TrayPopupMenu object to the items property of a Tray.
deleteItem This method is used to delete a TrayPopupMenu object registered in the items property of a Tray.
destroy This method is used to delete a tray that is dynamically created through a script.
findItem This method returns the index number of a certain TrayPopupMenu object registered in the items property of a Tray.
getItemCount This method returns the number of TrayPopupMenu objects registered in the items property of a Tray.
hideBalloonTip This method is used to hide a balloon tip on a Tray.
init This method is used to initialize a Tray after creating it in a script dynamically.
insertItem This method is used to insert a TrayPopupMenu object to a certain place of the items property of a Tray.
show This method is used to display a tray on the taskbar. The tray mentioned here refers to the one that is dynamically created through a script.
showBalloonTip This method is used to show a balloon tip on a Tray.
Event
Name Description
onballoontipclick This event occurs when the balloon tip displayed on the tray is clicked.
onballoontiphide This event occurs when the balloon tip displayed on the tray is hidden.
onballoontipshow This event occurs when a balloon tip is displayed on the tray.
ondblclick This is an event that occurs when double-clicking a tray icon.
onlbuttonup This event occurs when you click the left button of a mouse within a Tray icon.
onrbuttonup This event occurs when you click the right button of a mouse within a Tray icon.
