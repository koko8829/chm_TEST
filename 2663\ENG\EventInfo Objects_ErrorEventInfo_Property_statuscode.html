Description 
This property holds the code of an error that has occurred.
Syntax 
ErrorEventInfo.statuscode
Setting Syntax
301 
The requested URI has been replaced with a new URI.
302 
The requested URI has been changed temporarily.
305 
You must use a proxy location to access the requested resources.
307 
The requested URI has been changed temporarily.
400 
Cannot be processed due to an invalid request from the client.
Please check the address and try again.
401 
The request for logging on to the server does not match the permissions defined in the server.
402 
It is reserved already.
403 
There is no available default page in the directory that is accessed by the browser.
404 
No document has been found in the requested path.
Please check the address and try again.
405 
The page cannot be displayed on the site.
406 
This request cannot be accepted.
407 
Proxy authorization required
408 
Request timed out
409 
The request cannot be completed since there are conflicts in the request.
410 
The requested page cannot be used anymore.
411 
The server does not accept the request since the Content-Length header is not defined.
412 
The prerequisites have been treated as failure in the server.
413 
The server has not accepted the request since the entity of the request is too big.
414 
The server has not accepted the request since the requested URL is too long.
415 
The server has not accepted the request since the media type is not supported.
416 
The server has not accepted the request since the requested range is too large.
417 
The requirements of the request header cannot be processed.
449 
Retry the request after taking appropriate measures.
500 
The request cannot be processed due to a server error.
501 
The server can neither recognize the request method nor implement the request.
502 
The server is acting as a gateway or proxy, or the server has received an invalid response from the parent server.
503 
The server is unavailable currently.
504 
The server is acting as a gateway or proxy, or the server has not received an appropriate request from the parent server.
505 
The server does not support the version of the HTTP protocol used for the request.
511 
The permission to access the network has not been authenticated.
10400 
The communications have been stopped by the user.
10401 
The transaction has been stopped by the pressing of the Esc key.
10402 
The SSL certificate has failed to be verified.
10403 
The information on the SSL proxy has been changed, or no proxy information has been obtained.
10405 
The same service ID has been called during a transaction.
10408 
A logical timeout has occurred within Nexacro regardless of the state of the transaction.
10499 
An unknown error has occurred within Nexacro regardless of the state of the transaction.

This error occurs in an SSL certificate or HTTP proxy server in the Nexacro runtime environment.
10504 
A logical server timeout has occurred within Nexacro regardless of the state of the transaction.
10599 
The format of the data received through the transaction is not supported by the Nexacro.
Remark 
- All the communications errors fire the onerror event of the Environment object except when they are related to the transaction method.
The errors related to the transaction method will fire the onerror event of the Application or Form object depending on the location where the method was called.
