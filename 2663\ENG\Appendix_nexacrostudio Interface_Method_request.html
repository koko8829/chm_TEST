request
Description 
The method to request, return, and process Nexacro Studio project information.
It can be used only when it is executed in the Add on type inside of Nexacro Studio.
The value to be return after executing the method is a String type.
Syntax 
nexacrostudio.request( requestType[, dataobjuectID] );
Parameters 
	

		Parameters
		Description
	


	

		
requestType

		
Sets the type of information to be imported from the project.

	


	

		
dataobjectID

		
Sets the attribute value of the DataObject id to save the information returned after calling the request method.
The DataObject should be created as an Application Variable, not Form.
Return 
	

		Return Type
		Description
	


	

		
String

		
Returns the information requested according to the parameter value of the requesttype into a JSON format string.
Request Type 
	

		requesttype
		Description
	


	

		
requesttype.all

		
Runs the requesttype below to Requests the project information to be aliened and returned.
- requesttype.general
- requesttype.environment
- requesttype.typedefinition
- requesttype.applicationvariables
- requesttype.applications
- requesttype.forms

	


	

		
requesttype.general

		
Requests the general information of files used in the project
Returns information of the file name, file type, path, and the edit date

	


	

		
requesttype.environment

		
Runs the requesttype below with basic information of Environment (attribute, event, function) to Requests the TypeDefinition information to be aliened and returned.
- requesttype.screeninfo
- requesttype.env_variables
- requesttype.env_cookies
- requesttype.env_httpheader

	


	

		
requesttype.screeninfo

		
Request the ScreenDefinition information of the Environment.

	


	

		
requesttype.env_variables

		
Requests the Variables information of the Environment.

	


	

		
requesttype.env_cookies

		
Requests the Cookies information of the Environment.

	


	

		
requesttype.env_httpheader

		
Requests the HTTP Header information of the Environment.

	


	

		
requesttype.typedefinition

		
Runs the requesttype below to request the TypeDefinition information to be aliened and returned.
- requesttype.modules
- requesttype.objects
- requesttype.services
- requesttype.protocoladaptors
- requesttype.deviceadaptors

	


	

		
requesttype.modules

		
Requests the Modules information registered in the TypeDefinition.

	


	

		
requesttype.objects

		
Requests the Objects information registered in the TypeDefinition.

	


	

		
requesttype.services

		
Requests the Services information registered in the TypeDefinition.

	


	

		
requesttype.protocoladaptors

		
Requests the ProtocolAdators information registered in the TypeDefinition.

	


	

		
requesttype.deviceadaptors

		
Requests the DeviceAdaptors information registered in the TypeDefinition.

	


	

		
requesttype.applicationvariables

		
Runs the requesttype below to request the Application Variables information to be aliened and returned.
- requesttype.app_datasets
- requesttype.app_variables
- requesttype.app_dataobjects

	


	

		
requesttype.app_datasets

		
Requests the Datasets information registered in the Application Variables.

	


	

		
requesttype.app_variables

		
Requests the Variables information registered in the Application Variables.

	


	

		
requesttype.app_dataobjects

		
Requests the DataObjects information registered in the Application Variables.

	


	

		
requesttype.applications

		
Requests the Applications information of the project.

	


	

		
requesttype.forms

		
Requests the information on the xfdl used in the project.
