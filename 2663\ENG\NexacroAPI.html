Description 
These are methods provided by Nexacro for functional extensions.
Remark 
- Since these methods are statically defined in Nexacro, you can use them without any creation and declaration.
Method
Name Description
addErrorMessage This method is used to add a user-defined error message to an application.
base64Decode This method is used to decode base64-encoded data.
base64Encode This method is used to encode data in the Base64 scheme.
ceil This method rounds a numeric value upward to a value of the defined number of digits. Otherwise, this method rounds a numeric value off to its nearest integer.
deleteCacheDB This method is used to delete the cache DB files and related temporary files of a current application.
flashWindow This method is used to flash the icon of an application on the taskbar.
floor This method rounds a numeric value downward up to a value of the defined number of digits. Otherwise, this method rounds a numeric value downward to its nearest integer.
gc This method is used to immediately remove the data that remains unused in a memory.
getApplication This method returns the Application object of a currently-executed application.
getCookieVariable This method returns the value of a cookie defined in the Cookies area in Environment.
getEnableWheelZoom This method returns whether a zoom feature is set for the window, one of whose frame is designated as an argument.
getEnvironment This method returns the Environment object of a currently-executed application.
getEnvironmentVariable This method is used to return the value of a variable defined in the Variables field in Environment.
getHTTPHeaderVariable This method returns a variable defined in the HTTP Header of Environment.
getLogFilePath This is a method of returning the path where the app saves log files.
getPopupFrames This method returns a list of ChildFrames opened in the window that includes the frame designated as an argument. The list is returned in the form of a collection.
getPrivateProfile This is a method of returning the personalized data saved by the application.
getProjectPath This method returns the execution path of an application.
getStringResourceValue This is a method of returning a string corresponding to the key value specified in StringResource.
getTextSize This method returns the size of the area where text is displayed. The area is calculated based on the delivered arguments.
getWheelZoom This method returns the zoom factor of the window, one of whose frame is designated as an argument.
isAlpha This method checks a character code or a string that is delivered as an argument. Then, this method returns whether the argument is an alphabet letter.
isAlphaNumeric This method checks a character code or a string that is delivered as an argument. Then, this method returns whether the argument is alphanumeric.
isExistCookieVariable This method returns the existence of a certain cookie variable defined in the Cookies area in Environment.
isLower This method checks a character code or a string that is delivered as an argument. Then, this method returns whether the argument is lower case.
isNumeric This method checks a character code or a string that is delivered as an argument. Then, this method returns whether the argument is a number.
isSpace This method checks a character code or a string that is delivered as an argument. Then, this method returns whether the argument is a whitespace.
isUpper This method checks a character code or a string that is delivered as an argument. Then, this method returns whether the argument is upper case.
loadStringResource This is a method of loading the StringResource corresponding to the specified language.
loadStyle The method to load and apply an XCSS file to the application.
open This is a method of displaying a dynamically generated ChildFrame as Modeless Window.
parseDate This method is used to parse a date string and returns the number of milliseconds between the date string and midnight of January 1, 1970.
removeCookieVariable This method is used to remove a cookie from the Cookies area in Environment.
removeEnvironmentVariable This method is used to remove a variable defined in the Variables field in Environment.
removeHTTPHeaderVariable This method is used to remove a variable defined in the HTTP Header of Environment.
removePrivateProfile This method is used to delete individualized data stored by the application.
replaceAll This method replaces a certain string with a new string all through the defined range. Then, this method returns the new string entered as the result of the replacement.
round This method rounds a numeric value off to a value of the defined number of digits. Otherwise, this method rounds a numeric value off to its nearest integer.
setCookieVariable This method is used to add a cookie to the Cookies area in Environment.
setEnableWheelZoom This method sets a zoom feature is set for the window, one of whose frame is designated as an argument.
setEnvironmentVariable This method is used to add a variable to the Variables area in Environment.
setHTTPHeaderVariable The method to add the variance of the HTTP Header area of the Environment.
setInitialVariable This method is used in the web browser environment to set a variable in a global area before the loading of an application.
setPrivateProfile This is a method of storing personalization data in the application.
setProtocolVar This method is used to deliver the value of a setting to a communications protocol adapter.
setRenderingType This method is used to determine how to render a screen in the Windows NRE.
setWheelZoom This method sets the zoom factor of the window, one of whose frame is designated as an argument.
stripQuote This method removes the quotation marks surrounding a string delivered as an argument and then returns the result.
toNumber This method returns the result of converting a string delivered as an argument into a number.
trim This method is used to remove white space or predefined characters from both sides of a string that is delivered as an argument.
trimLeft This method is used to remove white space or predefined characters from the left side of a string that is delivered as an argument.
trimRight This method is used to remove white space or predefined characters from the right side of a string that is delivered as an argument.
wrapQuote This method surrounds a string delivered as an argument with quotation marks and then returns the result.
