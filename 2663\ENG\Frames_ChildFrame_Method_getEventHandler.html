getEventHandler

Android Default Web Browser iOS/iPadOS Default Web Browser
Description 
This method returns a specific handler from a certain event of a ChildFrame by specifying the index number of the handler.
Syntax 
ChildFrame.getEventHandler( strEventID, nIdx )
Parameters
Parameters Type Description
strEventID String 
The ID of an event that you search for your sought handler
nIdx Number 
The index number of the event handler you want to find

The index number begins from 0.
Sample Call:
var objFunc = ChildFrame00.getEventHandler( "onmove", 0 );
Return
Type Description
Object 
It returns the object of an event handler whose index number is specified. 

If there is no handler for the specified index number, it will return "null".
