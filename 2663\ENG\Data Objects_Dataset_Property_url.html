Description 
This property is used to specify the URL of a business service that is connected to a Dataset.
Syntax 
Dataset.url[= strUrl]
Setting Syntax 
strUrl ::= <absolute-url> | <relative-url>
 
this.DataSet00.url = "http://localhost:8080/Prj/BService/Serv00.jsp";     // Absolute path

this.DataSet00.url = "../BService/Serv00.jsp";                                     // Relative path
this.DataSet00.url = "Service1::Serv00.jsp";                                         // Path to the Sservices in the TypeDefinition
<absolute-url> 
It specifies a path to a business service with the format "http://path/service name" or "https://path/service name".

You must be able to access the Web, and the use of this property is limited in the cross-domain situations.
<relative-url> 
It specifies a path by using a prefix defined in the services in the TypeDefinition.

Otherwise, it specifies a relative path based on the location of a current location.
Remark 
- When the load() method of a Dataset is called, the Dataset will be loaded onto the service that is specified as the value of the URL property. This will be true even if the preload property of the Dataset is set to "false."

- The onload event will occur if data is loaded to a Dataset.
