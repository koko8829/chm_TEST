Description 
This property is used to specify the text of a tooltip that apeears when a mouse pointer hovers over a grid band.
Syntax 
GridBandControl.tooltiptext[= strToolTipText]
Setting Syntax 
this.Grid00.setBandProperty("body", "tooltiptext", "This is ToolTip Text')";
this.Grid00.setBandProperty("body", "tooltiptext", "')";          // a tooltip will not appear in this case.
this.Grid00.setBandProperty("body", "tooltiptext", null);        // a tooltip defined in the Grid will appear.
strToolTipText 
Text to be shown in a tooltip

A tooltip will not appear when you specify an empty string.
If you specify "null" or "undefined", this property will display a tooltip defined in the Grid.
Remark 
- The default value of this property is "undefined".

- If this property is set to "undefined", the tooltiptext property of a higher level, which is the Grid, will apply instead.
 If that higher level property is also set to "undefined", the same property of a more higher level will apply.

- A tooltip appears when a mouse pointer does not move over a GridBandControl for a certain duration.
