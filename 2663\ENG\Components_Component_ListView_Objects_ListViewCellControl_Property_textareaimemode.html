Property Type

Description 
This property is used to specify a default language used in the editable control of a cell whose edittype property is set to "textarea".
Syntax 
ListViewCellControl.textareaimemode[= enumImemode]
Setting Syntax 
enumImemode ::= 'none' | 'alpha' | 'alpha,full' | 'hangul' | 'hangul,full' | 'katakana' | 'katakana,full' | 'hiragana' | 'direct'
 
this.ListView00.setCellProperty( "body", "col00", "textareaimemode", "alpha" );
"none" 
It does not change the default input language. Instead, it maintains the current language set by the system.
"alpha" 
It sets English as an input language.

This value is applicable only when you have an East Asian IME (Korean/Japanese/Chinese).
"alpha,full" 
It sets full-width English as an input language.

This value is applicable only when you have an East Asian IME (Korean/Japanese/Chinese).
"hangul" 
It sets Korean as an input language.

This value is applicable only when you have a Korean IME.
"hangul,full" 
It sets full-width Korean as an input language.

This value is applicable only when you have a Korean IME.
"katakana" 
It sets Japanese Kana as an input language.

This value is applicable only when you have a Japanese IME.
"katakana,full" 
It sets full-width Japanese Kana as an input language.

This value is applicable only when you have a Japanese IME.
"hiragana" 
It sets Japanese hiragana as an input language.

This value is applicable only when you have a Japanese IME.
"direct" 
It allows you to directly input Japanese.

This value is applicable only when you have IME 2002 for Japanese.
If your IME is IME 2007 or 2010, this value will work the same as "alpha".
Remark 
- The default value of this property is "none".

- Your IME will be reset when the editable field of the cell becomes activated with the cell receiving focus.

- This property is supported by some environments. See the above Supported Environments for reference.
 > Environments supporting the property: Windows NRE, Android NRE.
 > Environments not supporting the property: macOS NRE, iOS/iPadOS NRE and WRE.
