getOffsetBoundRect

Description 
This method returns the area where a GraphicsText is displayed, including the border of the GraphicsText. The area is calculated based on the parent object. The border is set by the strokepen property.
Syntax 
GraphicsText.getOffsetBoundRect( );
Parameters
Sample Call:
var objGroup = this.Graphics00.getObjectByID("GraphicsGroup00");
var objGraphicsText = objGroup.getObjectByID("GraphicsText00");
var objRect = objGraphicsText.getOffsetBoundRect();

trace( "left : " + objRect.left );
Return
Type Description
Object 
This method returns an area in the form of a Rect object after calculating the area based on the top-left of the parent object.
If the GraphicsText has its border, the resulting area will include the border area.

If the strokepen property value (line-width) of the GraphicsEllipse, GraphicsLine, GraphicsPath, GraphicsPaths, or GraphicsRect object is set, the resulting area will include the border area.
If the strokepen property value is not set or in the case of the GraphicsGroup, GraphicsImage or GraphicsText object which is no border information, the resulting area will exclude the border area.

* A Rect object holds the left, top, right, bottom, width and height properties.
Remark 
- This method returns a logical area, regardless of whether the GraphicsText is displayed.
   Accordingly, note that you cannot use a value returned by this method to determine whether the GraphicsText is displayed.

- If you call a transformational method of a GraphicsText such as scale() or rotate(), the getOffsetBoundRect() method will return the area to which the called method has been applied.

- If you want to obtain an area calculated based on the Graphics component, you should use the getBoundRect() method.

- The strokepen property value (line-width) of GraphicsEllipse, GraphicsLine, GraphicsPath, GraphicsPaths, or GraphicsRect object is applied by dividing into the inside and outside based on edges of the object's actual area.
