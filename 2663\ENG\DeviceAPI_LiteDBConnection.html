DeviceAPI > LiteDBConnection
LiteDBConnection
Supported Environments
[Desktop]
Nexacro Runtime Environment (NRE)
Windows macOS
Web Runtime Environment (WRE)
Edge Chrome Safari Firefox Opera
[Mobile]
Nexacro Runtime Environment (NRE)
Android iOS/iPadOS
Web Runtime Environment (WRE)
Android Default Web Browser iOS/iPadOS Default Web Browser
Description 
It shows the connection information of DB supported on mobile devices.
Property
Name Description
async This property is used to determine how a LiteDBConnection object performs a requested task.
busytimeout This property sets a period of time allowed for receiving a response from the database connected to a LiteDBConnection.
datasource This property is used to set the location information of the database to be connected to LiteDBConnection.
id This property is used to set the identification of a LiteDBConnection.
name This property is used to specify the name of a LiteDBConnection.
openflag This property is used to set an option for connecting a LiteDBConnection to a database.
parent This read-only property holds the parent object of a LiteDBConnection.
preconnect This property determines whether to connect a LiteDBConnection to the database when the LiteDBConnection is created.
Method
Name Description
begin This method is used to set a transaction committed to the database connected with a LiteDBConnection.
close This method is used to close the database connected with a LiteDBConnection.
commit This method is used to commit a transaction to the database connected with a LiteDBConnection.
isConnected This method is used to check whether a LiteDBConnection is connected to a database.
open This is a method of connecting a database to LiteDBConnection.
rollback This method is used to roll back a transaction committed to the database connected with a LiteDBConnection.
Event
Name Description
onerror The onerror event occurs when a task request to a LiteDBConnection object fails.
onsuccess This event occurs when a task requested by a LiteDBConnection is successful.
