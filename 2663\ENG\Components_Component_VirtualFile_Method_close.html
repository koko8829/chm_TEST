Web Runtime Environment (WRE)
Android Default Web Browser iOS/iPadOS Default Web Browser
Description 
This method is used to close a file opened by a VirtualFile.
Syntax 
VirtualFile.close()
Parameters
Sample Call:
this.VirtualFile00.close();
Return
None
Remark 
- If this method succeeds in closing a file, it will fire the onsuccess event.
 If it fails, it will fire the onerror event.

- In the web runtime environment, you cannot use a VirtualFile object on its own.
  You can use a VirtualFile together with the filelist property of a FileUpload, FileUpTransfer or DragEventInfo object. A VirtualFile object can also be used with the virtualfiles property of a FileDialogEventInfo object.
