System
Supported Environments
[Desktop]
Nexacro Runtime Environment (NRE)
Windows macOS
Web Runtime Environment (WRE)
Edge Chrome Safari Firefox Opera
[Mobile]
Nexacro Runtime Environment (NRE)
Android iOS/iPadOS
Web Runtime Environment (WRE)
Android Default Web Browser iOS/iPadOS Default Web Browser
Description 
This object provides information about the OS, hardware and input devices.
Remark 
- This global object can be accessed through a script.

- The functions provided by this object vary according to OSes and browsers.
Property
Name Description
computername This read-only property holds the name of the NetBIOS of the computer where an application is running.
cpuarchitecture This read-only property holds the manufacture of the CPU of the computer where an application is running.
cpucount This read-only property holds the number of CPUs in the computer where an application is running.
cputype This read-only property holds the information on the CPU of the computer where an application is running.
locale This property holds the locale code value set by the computer that is running an application.
loginuser This read-only property holds the ID of the user who logs into the computer where an application is running.
mobileorientation This read-only property holds the current screen orientation of the mobile device that runs the application.
mobileproducttype This read-only property holds the model name of the mobile device where an application is running.
monitorcount This read-only property holds the number of monitors connected to the computer where an application is running.
navigatorfullname This read-only property holds a certain type of Windows Nexacro engine where an application runs.
navigatorname This read-only property holds the information on the browser where an application is running.
navigatorversion This read-only property holds the version of the browser where an application is running.
notificationtoken This read-only property holds the value of a device token for receiving push notifications in the mobile device that runs your application.
osversion This read-only property holds the name and version of the OS where an application is running.
taskbarsize This read-only property holds the size of the taskbar (indicator bar for Android) of the mobile device in which an application is running.
Method
Name Description
attachChildFrame This method is used to load an .xfdl file onto a user-DOM in a web browser. The .xfdl file here is generated through Nexacro Studio.
checkDevicePermission The method to check whether to set authority.
clearClipboard The method to delete the content saved on the clipboard of the system.
clientToScreenX This method converts an x-coordinate based on the client of the component into an x-coordinate based on the screen.
clientToScreenY This method converts a y-coordinate based on the client of the component into a y-coordinate based on the screen.
convertRealPath The method to convert the path including the Alias to the actual absolute path.
execBrowser This is a method of launching the browser (or app) according to the delivered string information.
execDefaultBrowser The method to execute the delivered URL from the web browser selected as the basic app.
execNexacro This method is used to execute Nexacro by by specifying an execution path and arguments.
execShell This method is used to execute a program in the Windows shell.
exportPDF The method to output the image of the selected component as a PDF file.
getClipboard The method to return the data saved on the clipboard of the system in a selected format.
getCursorX This method is used to obtain the x-coordinate of the position where a mouse cursor is located.
getCursorY This method is used to obtain the y-coordinate of the position where a mouse cursor is located.
getMonitorIndex This method returns the index number of the monitor closest to a delivered coordinate in the multi-monitor environment.
getPackageVersion This method returns app version information.
getScreenHeight This method returns the height of the screen where the application is running.
getScreenPrintString This method outputs the specified component in HTML string format.
getScreenRect This method returns a Rect object that contains the information on a specified monitor.
getScreenResolution This method returns the resolution of a specified monitor.
getScreenWidth This method returns the width of the screen where the application is running.
isPrimaryMonitor This method returns whether a specified monitor is a main monitor in the multi-monitor environment.
print The method to print out the image of the selected component.
requestDevicePermission The method to request the set authority.
saveToImageBase64String The method to return the image of the selected component as an encoded string with Base 64.
saveToImageFile The method to output the image of the selected component as a file.
screenToClientX This method converts an x-coordinate based on the screen into an x-coordinate based on the client of the component.
screenToClientY This method converts a y-coordinate based on the screen into a y-coordinate based on the client of the component.
setClipboard The method to save the data on the clipboard of the system in a selected format.
showModalSync This method is used to display a dynamically-created ChildFrame in the form of a modal window that works in a synchronous manner.
showModalWindow This method is used to display a dynamically-created ChildFrame in the form of a separate modal window.
