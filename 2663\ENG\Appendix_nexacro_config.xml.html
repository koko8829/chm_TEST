nexacro_config.xml
Overview 
The nexacro_config.xml file sets the operation option to be used in the NRE environment.
The options that can be set may differ according to the OS environment of the app.
File Creation and Path 
The basic file path by OS is as follows. 
  • Windows : %USERAPP%\Config\[Key]\nexacro_config.xml
  • Android : \res\xml\nexacro_config.xml
  • iOS/iPadOS : {asset}/data/nexacro_config.xml
  • macOS : {asset(project)}/data/nexacro_config.xml
- When building Android, iOS/iPadOS, macOS NRE apps using App Builder, nexacro_config.xml file is automatically created and applied according to App Builder  settings.
- Xcode, When creating a project directly in the Android Studio, the nexacro_config.xml file needs to be added manually.
- In the Windows NRE, it is applied only when the file location is specified with the -NC execution option.
