Description 
As a frame subordinate to a MainFrame, a VFrameSet is a basic unit for arranging Form screens in an application.

A VFrameSet can include one or more of FrameSets, HFrameSets, VFrameSets, TileFrameSets and ChildFrames internally.
A VFrameSet automatically arranges its subordinate frames vertically.
Structure
Constructor 
VFrameSet.init( strName, nLeft, nTop, nWidth, nHeight [, nRight, nBottom] )
Parameters Type Description
strName String 
This is a string that represents the ID of a VFrameSet.
nLeft Number 
This parameter is used to set the left property of the VFrameSet based on the top-left side of the parent frame.
The value can be expressed in pixel or percent (%).
nTop Number 
This parameter is used to set the top property of the VFrameSet based on the top-left side of the parent frame.
The value can be expressed in pixel or percent (%).
nWidth Number 
This parameter is used to set the width property of the VFrameSet.
The value can be expressed in pixel or percent (%).
nHeight Number 
This parameter is used to set the height property of the VFrameSet.
The value can be expressed in pixel or percent (%).
nRight Number 
This parameter is used to set the right property of the VFrameSet based on the bottom-right side of the parent frame.
The value can be expressed in pixel or percent (%).
If you set the nLeft and nWidth parameters all, this parameter will be ignored.
nBottom Number 
This parameter is used to set the bottom property of the VFrameSet based on the bottom-right side of the parent frame.
The value can be expressed in pixel or percent (%).
If you set the nTop and nHeight parameters all, this parameter will be ignored.
Sample Call:
var obFrame = new VFrameSet();

obFrame.init( "VFrameSet00", 30, 120, 196, 46, null, null);
obFrame.init( "VFrameSet00", null, null, 300, "400px", "80%", 300);
 
// Create Object  
var objVFrameSet = new VFrameSet();  
objVFrameSet.init("VFrameSet", 30, 120, 196, 46, null, null);
objVFrameSet.init("VFrameSet", null, null, 300, "400px", "80%", 300);
or
var objVFrameSet = new VFrameSet("VFrameSet", 30, 120, 196, 46, null, null);
var objVFrameSet = new VFrameSet("VFrameSet", null, null, 300, "400px", "80%", 300);
 
// Add Object to Parent Frame  
this.addChild("VFrameSet", objVFrameSet); 
 
// Insert Object to Parent Frame  
this.insertChild(1, "VFrameSet", objVFrameSet); 
 
// Remove Object form Parent Frame  
this.removeChild("VFrameSet"); 
 
// Show Object  
objVFrameSet.show(); 
 
// Destroy Object  
objVFrameSet.destroy(); 
objVFrameSet = null;

getNativeHandle This method returns the window handle (HWND) value to which the VFrameSet belongs.
getOffsetBottom This method is used to obtain the bottom position of a VFrameSet in pixel, based on the parent frame's top side.
getOffsetHeight This method is used to obtain the height of a VFrameSet in pixel.
getOffsetLeft This method is used to obtain the left position of a VFrameSet in pixel, based on the parent frame's left side.
getOffsetRight This method is used to obtain the right position of a VFrameSet in pixel, based on the parent frame's left side.

getOffsetWidth This method is used to obtain the width of a VFrameSet in pixel.

move This method is used to change the position and size of a VFrameSet.
removeChild This method is used to remove a frame that has been dynamically added to a VFrameSet.
removeEventHandler This method is used to remove a handler in a certain event of a VFrameSet.

setEventHandler This method is used to change the first handler in a certain event of a VFrameSet.

setFocus This method is used to activate and give a focus to an VFrameSet.
setOffsetBottom This method is used to specify the bottom position of a VFrameSet in pixel when the VFrameSet belongs to a FrameSet.
setOffsetHeight This method is used to specify the height of a VFrameSet in pixel when the VFrameSet belongs to a FrameSet.
setOffsetLeft This method is used to specify the left position of a VFrameSet in pixel when the VFrameSet belongs to a FrameSet.
setOffsetRight This method is used to specify the right position of a VFrameSet in pixel when the VFrameSet belongs to a FrameSet.
setOffsetTop This method is used to specify the top position of a VFrameSet in pixel when the VFrameSet belongs to a FrameSet.
setOffsetWidth This method is used to specify the width of a VFrameSet in pixel when the VFrameSet belongs to a FrameSet.
