XPush
onsuccess Event Action 
	

		Action
		Value
		Description
	


	

		
nexacro.XPushAction.AUTH

		
0

		
When the X-PUSH server has been connected successfully by the connect() method.

	


	

		
nexacro.XPushAction.BYEC

		
1

		
When the X-PUSH server has been disconnected successfully by the disconnect() method.

	


	

		
nexacro.XPushAction.ADDF

		
2

		
When a topic has been registered successfully by the subscribe() method.

	


	

		
nexacro.XPushAction.DELF

		
3

		
When a topic has been removed successfully by the unsubscribe() method.

	


	

		
nexacro.XPushAction.REQD

		
4

		
When an unreceived reliable messages has been requested successfully by the requestMessage() method.

	


	

		
nexacro.XPushAction.RECT

		
5

		
When an unreceived reliable messages has been responded successfully by the sendResponse() method.

	


	

		
nexacro.XPushAction.RGST

		
6

		
When a device token has been successfully registered in an X-PUSH server by the registerDevice() method.

	


	

		
nexacro.XPushAction.UNRG

		
7

		
When a device token has been successfully unregistered from an X-PUSH server by the unregisterDevice() method.

	


	

		
nexacro.XPushAction.ADUI

		
8

		
When a topic of reliable message has been registered successfully by the registerTopic() method.

	


	

		
nexacro.XPushAction.UNUI

		
9

		
When a topic of reliable message has been unregistered successfully by the unregisterTopic() method.

	


	

		
nexacro.XPushAction.MSGC

		
10

		
When unreceived the number of reliable messages has been successfully requested by the requestMessageCount() method.
