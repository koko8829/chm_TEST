Description 
This object provides a Bluetooth connection with an external device.
Remark 
- This object supports Bluetooth Low Energy. It supports the following versions of OSes or higher versions: Windows 8, Android 4.3, iOS 5 and OSX 10.6.

- All the methods of the BluetoothLE object are called in the asynchronous manner.
   The results of calling those methods are delivered to events.
Property
Name Description
id This property is used to set the identification of a BluetoothLE.
name This property is used to specify the name of a BluetoothLE.
Method
Name Description
connect This method is used to connect to a peripheral device.
disconnect This method is used to disconnect from a peripheral device.
discoverService This method is used to request the universally unique identifiers (UUIDs) of the services provided by the connected peripheral device.
readCharacteristic This method requests the connected peripheral device to send a characteristic of a certain service.
scanStart This method is used to begin a scanning process for finding peripheral devices that are sending advertisements.
scanStop This method is used to end a scanning process for finding peripheral devices that are sending advertisements.
subscribe This method requests notification sent to the connected peripheral device when a characteristic of a certain service changes.
unsubscribe This method requests the cancellation of notification sent to the connected peripheral device.
writeCharacteristic This method saves a characteristic of a certain service to the connected peripheral device.
Event
Name Description
onerror This event occurs when a task requested by a BluetoothLE fails.
onscanresult This event occurs when a BluetoothLE object finds a device that provides a certain service through the scanStart() method.
onsubscriberesult This event occurs when a notification is delivered to a device that is connected to a BluetoothLE object.
onsuccess This event occurs when a task requested by a BluetoothLE object succeeds.
