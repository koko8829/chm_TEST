Description 
The Tab is a component that adds multiple TabPageControl.

It is good to use the tab when a large number of input/output components need to be assigned to a narrow screen.
The TabPageControl inside the Tab is composed of the content of the Tab or used by connecting the form.
Remark 
- Since the loading speed of the screen may become slower if using the tab with 4 to 5 stages, use the tab overlapped within 2 to 3 stages.

- The usecontrolkey attribute should be set for the Key Action to be used as combined with the “Control” key.

Basic Key Action
Action Key Description
Move focus Tab 
The focus moves to the next component according to the tab movement order.
Move between TabPages Ctrl + Tab 
Entering this key combination will change the current TabPage control.
Move between TabPages Left/Right arrow key 
Pressing the left/right arrow key will change the current TabPage control when the focusacceptable property is set to "true" and focus falls on a TabButton control.
Click TabButton Ctrl + BackSpace 
Entering this key combination will trigger the onextrabuttonclick event by clicking the TabButton control of a current TabPage control.

Constructor 
Tab.init( strName, vLeft, vTop , vWidth, vHeight [, vRight, vBottom, [vMinWidth, [vMaxWidth, [vMinHeight, [vMaxHeight]]]]] )
Parameters Type Description
strName String 
It sets the ID of a Tab.
vLeft String, Number 
It specifies the value of the left property of a Tab in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the right property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vTop String, Number 
It specifies the value of the top property of a Tab in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the bottom property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vWidth String, Number 
It specifies the value of the width property of a Tab in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vHeight String, Number 
It specifies the value of the height property of a Tab in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vRight String, Number 
It specifies the value of the right property of a Tab in pixel or percent (%).

This parameter will be ignored if both the vLeft and vWidth parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the left property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vBottom String, Number 
It specifies the value of the bottom property of a Tab in pixel or percent (%).

This parameter will be ignored if both the vTop and vHeight parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the top property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vMinWidth String, Number 
It specifies a Tab's minimum width that can be displayed in the Form.
vMaxWidth String, Number 
It specifies a Tab's maximum width that can be displayed in the Form.

If this value is smaller than vMinWidth, it will be replaced by the value of vMinWidth.
vMinHeight String, Number 
It specifies a Tab's minimum height that can be displayed in the Form.
vMaxHeight String, Number 
It specifies a Tab's maximum height that can be displayed in the Form.

If this value is smaller than vMinHeight, it will be replaced by the value of vMinHeight.

var objTab = new Tab();  
objTab.init("Tab", 30, 120, 196, 46, null, null);
objTab.init("Tab", null, null, 300, "400px", "80%", 300);
or
var objTab = new Tab("Tab", 30, 120, 196, 46, null, null);
var objTab = new Tab("Tab", null, null, 300, "400px", "80%", 300);
 
// Add Object to Parent Form  
this.addChild("Tab", objTab); 
 
// Insert Object to Parent Form  
this.insertChild(1, "Tab", objTab); 
 
// Remove Object form Parent Form  
this.removeChild("Tab"); 
 
// Show Object  
objTab.show(); 
 
// Destroy Object  
objTab.destroy(); 
objTab = null;

onsetfocus This event occurs when focus comes into a Tab.
onsize This event occurs when the size of a Tab changes.
ontouchend This event occurs when you touch and release the screen within the area of a Tab.
ontouchmove This event occurs when a touch point is moved along the touch surface within a Tab.
ontouchstart This event occurs when a touch point is placed on the touch surface within a Tab.
