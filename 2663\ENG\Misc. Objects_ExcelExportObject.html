ExcelExportObject
Supported Environments
[Desktop]
Nexacro Runtime Environment (NRE)
Windows macOS
Web Runtime Environment (WRE)
Edge Chrome Safari Firefox Opera
[Mobile]
Nexacro Runtime Environment (NRE)
Android iOS/iPadOS
Web Runtime Environment (WRE)
Android Default Web Browser iOS/iPadOS Default Web Browser
Description 
This object is used to export the contents of a Grid in the form of an Excel or Hancell file.
Property
Name Description
commcompress This property determines whether to compress POST data in the transactions with the server for exporting data.
commdataformat This property determines the data type used to deliver data to the server for exporting a Grid.
exportactivemode This property is used to determine whether to open an exported file automatically after the export process.
exporteventtype This property defines how to present the information on the exporting process and how to fire the onprogress event.
exportfilename This property is used to set the name of a file created as the result of export.
exportfilepassword This property is used to set the password for a file created as the result of export.
exportfilepath This property is used to set an absolute path where an exported file is created.
exportmessagealert This property is used to set the text written in an alert dialog that appears before the exporting process begins.
exportmessagecomplete This property is used to set the text that is written in the progress bar when the exporting process is completed. This property is applied when the exportuitype property is set to "exportprogress".
exportmessageprocess This property is used to set the text that is written in the progress bar in the middle of the exporting process. This property is applied when the exportuitype property is set to "exportprogress".
exportmessageready This property is used to set the text that is written in the progress bar before the exporting process begins. This property is applied when the exportuitype property is set to "exportprogress".
exporttype The attribute to set the file format to be created as a constant value by exporting the ExcelExportObject.
exportuitype This property determines how to show the progress of an export process.
exporturl This property is used to specify the URL from which an ExcelExportObject will export files.
exportwordwrap This property determines whether to apply text line wrap to Grid cells when the Grid is exported.
filefilter This property is used to set the file types applicable to the file dialog displayed for creating a file as a result of an exporting process.
filefilterindex This property is used to determine the default file type presented by a file dialog by specifying an index number.
name This property is used to specify the name of a ExcelExportObject.
quotechar The attribute to set the value to be used as a limiter when exporting.
separator The attribute to set the value to be used as a delimiter when exporting.
transferrowcount This property is used to set the number of rows whose data will be exported as a packet when the entire data is divided.
Method
Name Description
addEvent This method is used to declare a certain event and then add it to a ExcelExportObject.
addEventHandler This method is used to add a handler to a certain event of a ExcelExportObject.
addEventHandlerLookup This method is used to find a function and add it to a certain event of a ExcelExportObject as a handler.
addExportItem The method to add the items to be exported to the ExcelExportObject.
clear This method is used to delete all the items added to an ExcelExportObject.
clearEventHandler This method is used to delete all the event handlers registered in a certain event.
clearExportItems This method is used to delete the items of a certain type from an ExcelExportObject.
count This method returns the number of items registered in an ExcelExportObject.
countExportItems This method returns the number of the items of a certain type in an ExcelExportObject.
delExportItem This method is used to delete a certain item of a certain type from an ExcelExportObject.
destroy This method is used to delete a ExcelExportObject that is dynamically created through a script.
exportData This method is used to export all the items set by an ExcelExportObject.
exportDataEx This method is used to export all the items set by an ExcelExportObject based on the values displayed in the cells.
exportItems This method is used to export the items of a certain type in an ExcelExportObject.
findEventHandler This method finds a handler in a certain event of a ExcelExportObject and returns the index of the handler.
getEventHandler This method returns a specific handler from a certain event of a ExcelExportObject by specifying the index number of the handler.
getExportItem This method returns a certain item of a certain type from an ExcelExportObject.
insertEventHandler This method is used to add a handler to a certain event of a ExcelExportObject at a specified position.
removeEvent This method is used to remove a certain event that is declared in a ExcelExportObject.
removeEventHandler This method is used to remove a handler in a certain event of a ExcelExportObject.
removeEventHandlerLookup This method is used to find a handler in a certain event of a ExcelExportObject and remove it.
setEventHandler This method is used to change the first handler in a certain event of a ExcelExportObject.
setEventHandlerLookup This method finds a function and uses it to replace the first handler in a certain event of a ExcelExportObject.
setExportItem This method is used to add an export item to a certain position of an ExcelExportObject.
Event
Name Description
onerror This event occurs when there is an error during export process.
onprogress This event occurs at each stage of the exporting process.
onsuccess This event occurs when the exporting of all items is completed.
