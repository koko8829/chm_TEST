Description 
This property is used to set the color and type of the outer line of a GraphicsEllipse.
Syntax 
GraphicsStock.strokepen[= strStrokePen]
Setting Syntax 
strStrokePen ::= <line-width> <line-style> <color>

<line-width> ::= <nWidth> ['px']
<line-style> ::= 'solid' | 'dotted' | 'dashed'
<color> ::= <NamedColor> | <NumericColor>
 
var objGObject = new nexacro.GraphicsEllipse();

objGObject.strokepen = "5px solid red";
objGObject.strokepen = "5 dotted #999999";
objGObject.strokepen = "5px dotted rgb(255,0,0)";
<nWidth> 
This is the number of pixels, indicating the width of the outer line.
<line-style> 
This is the style of the outer line.

If you specify "solid", a solid line will be drawn.
If you specify "dotted", a dotted line will be drawn.
If you specify "dashed", a dashed line will be drawn.
<NamedColor> 
You can specify a predefined color name in the form of a string like "red" and "blue".
<NumericColor> 
This is a hexadecimal code that begins with "#" and comprises the following 6-digit value (eg. #FF00FF).

You can specify a color code with the rgb() method whose parameters are red, green and blue values. (eg. rgb(255,0,255))
Remark 
- The strokepen property value is applied by dividing it into the inside and outside based on the actual area of the object.
