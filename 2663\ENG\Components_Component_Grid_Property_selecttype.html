Description 
This property is used to determine the range of selection that you make in a Grid.
Syntax 
Grid.selecttype[= enumType]
Setting Syntax 
enumType ::= 'row' | 'multirow' | 'cell' | 'area' | 'multiarea'
 
this.Grid00.selecttype = "multirow";
"row" 
You can select only one row in the body band of a Grid.

Assume that you hold down the left button of a mouse and drag the pointer up/downward in
 a Grid, with the selectscrollmode property set to "select". In that case, the row where you pressed the left button will be selected.
The rowposition property of a bound Dataset will change its value to a selected row.
"multirow" 
You can select multiple rows in the body band of a Grid.

With the selectscrollmode property set to "select", you can select multiple rows by dragging a mouse pointer in a Grid.
You can select multiple rows by using a mouse with a control or shift key.
The rowposition property of a bound Dataset will change its value to the last-selected row.
"cell" 
You can select only one cell in the body band of a Grid.

Assume that you hold down the left button of a mouse and drag the pointer to a certain direction in a Grid, with the selectscrollmode property set to "select". In that case, the cell where you pressed the left button will be selected.
The rowposition property of a bound Dataset will change its value to the row that contains a selected cell.
"area" 
You can select only one rectangular area in the body band of a Grid.

With the selectscrollmode property set to "select", you can select an area by dragging a mouse pointer in a Grid.
You can select an area by using a mouse with the shift key.
The rowposition property of a bound Dataset will change its value to the last-selected row within a selected area.
"multiarea" 
You can select multiple areas in the body band of a Grid.

With the selectscrollmode property set to "select", you can select an area by dragging a mouse pointer in a Grid.
You can select multiple areas by using a mouse with a control or shift key.
The rowposition property of a bound Dataset will change its value to the last-selected row within the last-selected area.
Remark 
- The default value of this property is "row".

- If you change the value of the rowposition property of a bound Dataset with this property set to "row" or "cell", the relevant row of the Grid will be selected.

- With this property set to a value other than "row" and "cell", a selection made in the Grid will remain unchanged even if the rowposition property of a bound Dataset changes its value.
 In this case, you should be careful that the value of the rowposition property will not match the selection made in the Grid.

- The information on a selected area, which is influence by the value of the selecttype property is applied to the following properties automatically: selectstartrow, selectendrow, selectstartcol, selectendcol, selectstartsubrow and selectendsubrow.

- In the touch-enabled mobile environment, a multiple selection is impossible even with this property set to "multirow" or "multiarea".
 This is not a functional problem but are traits of such a input device.
