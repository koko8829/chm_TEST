Description 
This property sets the action of the Cell when it is in the MultiCombo format.
Syntax 
ListViewCellControl.multicombotype[= enumType]
Setting Syntax 
enumType ::= 'dropdown' | 'search' | 'filter' | 'filterlike' | 'caseisearch' | 'caseifilter' | 'caseifilterlike'
 
this.ListView00.setCellProperty( "body", "cell00", "multicombotype", "caseifilter");
"dropdown" 
All items are displayed in the Combo's item list.

No text can be entered in the edit window of the Cell.
"search" 
The Combo search function is enabled when a string is entered in the edit window of the Cell.

The first item starting with the string entered in the edit window is displayed in the selected item list.
If there is no item starting with the string entered in the edit window, the item list will not be displayed.

All items are displayed in the item list.
The item search is case sensitive.
"filter" 
The Combo filter function is enabled when a string is entered in the Cell edit window.

A list of items is displayed with only the items starting with the string entered in the edit window.
If there is no item starting with the string entered in the edit window, the item list will not be displayed.

The item list is automatically updated when additional characters are entered.
The item filter is case sensitive.
"filterlike" 
The Combo filter function is enabled when a string is entered in the Cell edit window.

A list of items is displayed with only the items including the string entered in the edit window.
If there is no item including the string entered in the edit window, the item list will not be displayed.

The item list is automatically updated when additional characters are entered.
The item filter is case sensitive.
"caseisearch" 
The Combo search function is enabled when a string is entered in the edit window of the Cell.

This behaves the same as the "search" value, and item searches are not case sensitive.
"caseifilter" 
The Combo filter function is enabled when a string is entered in the Cell edit window.

This behaves the same as the “filter" value, and item searches are not case sensitive.
"caseifilterlike" 
The Combo filter function is enabled when a string is entered in the Cell edit window.

This behaves the same as the “filterlike" value, and item searches are not case sensitive.
Remark 
- If the multicombotype property value is not set, it is applied as "dropdown".

- Regardless of the multicombotype property value, an item list with all items is displayed when the Cell's DropButton is clicked.

- Setting the multicombotype property to a value other than "dropdown" enables the search/filter feature.
   The search/filter feature helps the user input and displays item candidate values through the edit area and list window.

- If an item from the item list is selected by using the mouse or keyboard, the input is processed as finished.
