Description 
As a frame subordinate to a MainFrame, an HFrameSet is a basic unit for arranging Form screens in an application.

An HFrameSet can include one or more of FrameSets, HFrameSets, VFrameSets, TileFrameSets and ChildFrames internally.
An HFrameSet automatically arranges its subordinate frames horizontally.
Structure
Constructor 
HFrameSet.init( strName, nLeft, nTop, nWidth, nHeight [, nRight, nBottom] )
Parameters Type Description
strName String 
This is a string that represents the ID of a HFrameSet.
nLeft Number 
This parameter is used to set the left property of the HFrameSet based on the top-left side of the parent frame.
The value can be expressed in pixel or percent (%).
nTop Number 
This parameter is used to set the top property of the HFrameSet based on the top-left side of the parent frame.
The value can be expressed in pixel or percent (%).
nWidth Number 
This parameter is used to set the width property of the HFrameSet.
The value can be expressed in pixel or percent (%).
nHeight Number 
This parameter is used to set the height property of the HFrameSet.
The value can be expressed in pixel or percent (%).
nRight Number 
This parameter is used to set the right property of the HFrameSet based on the bottom-right side of the parent frame.
The value can be expressed in pixel or percent (%).
If you set the nLeft and nWidth parameters all, this parameter will be ignored.
nBottom Number 
This parameter is used to set the bottom property of the HFrameSet based on the bottom-right side of the parent frame.
The value can be expressed in pixel or percent (%).
If you set the nTop and nHeight parameters all, this parameter will be ignored.
Sample Call:
var obFrame = new HFrameSet();

obFrame.init( "HFrameSet00", 30, 120, 196, 46, null, null);
obFrame.init( "HFrameSet00", null, null, 300, "400px", "80%", 300);
 
// Create Object  
var objHFrameSet = new HFrameSet();  
objHFrameSet.init("HFrameSet", 30, 120, 196, 46, null, null);
objHFrameSet.init("HFrameSet", null, null, 300, "400px", "80%", 300);
or
var objHFrameSet = new HFrameSet("HFrameSet", 30, 120, 196, 46, null, null);
var objHFrameSet = new HFrameSet("HFrameSet", null, null, 300, "400px", "80%", 300);
 
// Add Object to Parent Frame  
this.addChild("HFrameSet", objHFrameSet); 
 
// Insert Object to Parent Frame  
this.insertChild(1, "HFrameSet", objHFrameSet); 
 
// Remove Object form Parent Frame  
this.removeChild("HFrameSet"); 
 
// Show Object  
objHFrameSet.show(); 
 
// Destroy Object  
objHFrameSet.destroy(); 
objHFrameSet = null;
