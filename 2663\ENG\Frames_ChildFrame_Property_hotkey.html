Description 
The attribute to set the hotkey to execute the basic actions of the ChildFrame.
Syntax 
ChildFrame.hotkey[= strHotkey]
Setting Syntax 
this.ChildFrame00.hotkey = "G"; 
this.ChildFrame00.hotkey = "CTRL+A"; 
this.ChildFrame00.hotkey = "CTRL+ALT+X";
strHotkey 
A hotkey is a combination of modifier keys (Ctrl, Alt and Shift) and a non-modifier key.
You can use one or more modifier keys in a combination.
Keys are combined by using "+" symbols between them.
Remark 
(1) Basic action of component

The basic action of the component is executed when setting the hotkey attribute value. The basic action of the component is as follows:
- onclick: Button, CheckBox
- Move focus (when setting the accessibility): GroupBox, Static
- onsetfocus: Other Frame objects or components
* For the GroupBox and Static components, the hotkey attribute is not processed unless it is an accessibility setting environment.

(2) Exceptional conditions

(2-1) When there is a component with the same hotkey attribute value
If there are two components or more with the same hotkey attribute value within the form, the hotkey attribute of the component created first is processed first in the index order of the components attribute.

(2-2) When using the input device with a hotkey set
When using the input device with the hotkey setting function, the hotkey attribute is not processed if the hotkey attribute value is set as the same value as the hotkey set from the input device.
For example, if using a keyboard that processes by inverting the “CTRL+M” key input to “ENTER”, the basic action is not executed even if the hotkey attribute value is set as “CTRL+M” and the relevant key is entered.

(2-3) When the upper frame or form is not activated
The hotkey attribute operates only when the upper frame or form is activated. 

(2-4) When the enable attribute value is false
If the enable attribute value of the component is false, the hotkey attribute is not processed.

(2-5) When the hotkeycolumn attribute value is set
The hotkeycolumn attribute is not processed if the hotkey attribute value and the hotkeycolumn attribute value of the menu component are set as the same attribute value.
