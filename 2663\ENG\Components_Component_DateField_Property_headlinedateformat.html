Description 
This property is used to set the format of the date data displayed in the headline in the datetimepicker control.
Syntax 
DateField.headlinedateformat [=strFormat]
Setting Syntax 
strFormat ::= <maskchar> | <dateStyle>

<maskchar> ::= [<Year>] [<Month>] [<Day>] [<Week>]


* <maskchar> is a combination of mask characters for displaying dates, where each element can be reordered and omitted.
 
this.DateField00.headlinedateformat = "yyyy-MM-dd ddd";
this.DateField00.headlinedateformat = "SHORTDATE";
<Year> 
Set the mask characters representing the year.

When set to "yyyy," all four digits of the year will be displayed.
(When "yy" is set, it will operate the same way as "yyyy.")
<Month> 
Set the mask characters representing the month.

When it is set as "M," January to September will be displayed as one digit and October to December will be displayed as two digits.
When it is set as "MM," all months will be displayed as two digits, and January to September will be displayed with "0" as the first digit.
When it is set as "MMM," the month string will be displayed in an abbreviated format according to locale settings.
When it is set as "MMMM"," the month string will be in an expanded format according to locale settings.
<Day> 
Set the mask characters representing the day.

When it is set as "d," the 1st to 9th will be displayed as one digit, and the other days will be displayed as two digits.
When it is set as "dd," all days will be displayed as two digits, and the 1st to 9th will be displayed with "0" as the first digit.
<Week> 
Set the mask characters representing the day of the week.

When it is set as "ddd,," the abbreviated name of the day  of the week will be displayed.
When it is set as "dddd," the full name of the day of the week will be displayed.
<dateStyle> 
Set as the style options used by ECMA Intl API DateTimeFormat() creator. 

"SHORTDATE" - dateStyle: 'short'
"MEDIUMDATE" - dateStyle: 'medium'
"LONGDATE" - dateStyle: 'long'
"FULLDATE" - dateStyle: 'full'
Remark 
- If the headlinedateformat property value is not set, it will be applied in MEDIUMDATE (dateStyle: 'medium') format.
