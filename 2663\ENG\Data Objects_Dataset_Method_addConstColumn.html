Description 
The method to add a const column (constant column) to the DataSet.
Syntax 
Dataset.addConstColumn( strColID, varVal [, strColType [, nColSize [, strDatapath]]] )
Parameters
Parameters Type Description
strColID String 
This is a string that represents the ID of a const column to add to the Dataset.

If there is already a const column whose ID equals the value of the this parameter, the value of the existing const column will be changed.
varVal Variant 
This is a variant that represents the value of a const column to add to the Dataset.
strColType String 
This is the type of a const column to add.

If you omit this parameter, either none or an invalid type can be set.
nColSize Number 
This is the data size of a const column to add.

If you omit this parameter, no data size will be set.
strDatapath String 
Set the datapath to be brought as the column value from the JSON data.
Set a route path that uses $.
Bring the first value if repetitive data is selected using @.
Sample Call:
var nIndex;
nIndex = this.Dataset00.addConstColumn( "ConstColumn1", 1234, "INT", 200 );
nIndex = this.Dataset00.addConstColumn( "ConstColumn2", "TEST", "STRING", 256 );
nIndex = this.Dataset00.addConstColumn( "company", , "STRING", 256, "$.company" );
Return
Type Description
Number 
It returns the index of the Const Column added to the DataSet.

-1 is returned if failing to add the Const Column.
Remark 
- The onrowsetchanged event occurs when adding a Const Column is successful.
   At this time, the e.reason attribute value of the onnrowsetchanged event becomes "REASON_CHANGELAYOUT".

- If the dataobjectpath attribute of the dataset object is selected and the Data Path Editor is executed in the nexacro studio, it sets the datapath attribute value automatically after selecting and adding the item to the column.
- The varVal parameter value is ignored when the parameters for varVal and strDatapath are set together.
