VirtualFile
open Method Option Flags 
	

		Option
		Hex Value
		Description
	


	

		VirtualFile.openRead
		0x0001
		is used when opening a read only file. Without this file, you cannot open a read only file.
	


	

		VirtualFile.openWrite
		0x0002
		is creates a file as write only. Regardless of existence of this file, you can create the write only file.
	


	

		VirtualFile.openAppend
		0x0010
		is used to open a file that exists as addtion only. If the file does not exist, it is created. In case the file is opened, file pointer is moved to the last location.
	


	

		VirtualFile.openText
		0x0100
		is used to open text file.
	


	

		VirtualFile.openBinary
		0x0200
		is used to open binary file. 
	


	

		VirtualFile.openCreate
		0x1000
		is used to open a file by creating when not existing. If the file exists, the existing file shall be opened.
seek Method Option Flags 
	

		Option
		Decimal Value
		Description
	


	

		VirtualFile.seekBegin
		0
		is the initial starting location of file.
	


	

		VirtualFile.seekCurrent
		1
		is the current location of file.
	


	

		VirtualFile.seekEnd
		2
		is the end location of file.
find Method Option flags 
	

		Option
		Hex Value
		Description
	


	

		
VirtualFile.findAll

		
0x01

		
Searches for files and folders belonging to the base folder as the target.

	


	

		
VirtualFile.findFileOnly

		
0x02

		
Searches for files belonging to the base folder as the target.

	


	

		
VirtualFile.findDirectoryOnly

		
0x03

		
Searches for folders belonging to the base folder as the target.

	


	

		
VirtualFile.findRecursive

		
0x08

		
Allow recursive searches of subfolders.
It can be used in combination with other option values.

	


	

		
VirtualFile.findCaseless

		
0x10

		
It is not case sensitive when searching.
It can be used in combination with other option values.
