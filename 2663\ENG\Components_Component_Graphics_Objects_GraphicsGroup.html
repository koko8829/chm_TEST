Description 
This object is used for processing Graphics-related objects as a logical group.
Remark 
- You can create this object dynamically through a script. You cannot design this object in Nexacro Studio. 

- This object can group the following objects, which belong to the Graphics component, regardless of types: GraphicsRect, GraphicsEllipse, GraphicsImage, GraphicsPath, GraphicsPaths, GraphicsLine and GraphicsText.

- If there is a change in a property of a GraphicsGroup object, the change will apply to all the objects grouped by the GraphicsGroup.

- A GraphicsGroup object does not appear on the screen.
Property
Name Description
id This property is used to set the identification of a GraphicsGroup.
parent This read-only property holds the parent object of a GraphicsGroup.
visible This property is used to determine whether to make a GraphicsGroup visible or not.
x This property is used to set the x-coordinate of a position where a GraphicsGroup is displayed. The coordinate is calculated based on the position of the parent object.
y This property is used to set the y-coordinate of a position where a GraphicsGroup is displayed. The coordinate is calculated based on the position of the parent object.
Method
Name Description
addChild This method is used to add a Graphics object as a child to a GraphicsGroup.
clear This method is used to remove an object added to a GraphicsGroup completely from a memory.
destroy This method is used to completely remove a GraphicsGroup from a memory.
getBoundRect This method returns the area where a GraphicsGroup is displayed, including the border of the GraphicsGroup. The area is calculated based on the Graphics component. The border is set by the strokepen property.
getFirstChild This method returns the first object among those that have been added to a GraphicsGroup.
getLastChild This method returns the last object among those that have been added to a GraphicsGroup.
getNextSibling This method returns the next object among those under the same parent as a GraphicsGroup.
getObjectByID This method searches the objects added to a GraphicsGroup and returns the object of a certain ID among them.
getObjects This method returns an array containing the objects added to a GraphicsGroup.
getOffsetBoundRect This method returns the area where a GraphicsGroup is displayed, including the border of the GraphicsGroup. The area is calculated based on the parent object. The border is set by the strokepen property.
getOffsetRect This method returns the area where a GraphicsGroup is displayed, with the area calculated based on the parent object.
getOffsetRegion This method returns the area where a GraphicsGroup is displayed, excluding the border of the GraphicsGroup. The area is calculated based on the parent object. The border is set by the strokepen property.
getPreviousSibling This method returns the previous object among those under the same parent as a GraphicsGroup.
getRect This method returns the area where a GraphicsGroup is displayed, with the area calculated based on the Graphics component.
getRegion This method returns the area where a GraphicsGroup is displayed, excluding the border of the GraphicsGroup. The area is calculated based on the Graphics component. The border is set by the strokepen property.
hasChild This method returns whether a GraphicsGroup has any Graphics object.
indexOf This method finds the object of a certain ID among the objects added to a GraphicsGroup and then returns the index number of the found object.
insertChild This method is used to add a Graphics object as a child to a certain position in a GraphicsGroup.
isHitTest This method finds the position marked by the coordinates delivered as arguments and returns whether the position is included in the area of a GraphicsGroup.
removeChild This method is used to remove a Graphics object of a certain ID among the children of a GraphicsGroup.
rotate This method is used to rotate a GraphicsGroup.
scale This method is used to scale a GraphicsGroup up/down.
setClipRect This method is used to set coordinates for clipping a certain region of a GraphicsGroup.
setTransform The method to apply the transform-related effect consecutively based on the original location of the GraphicsGroup.
shear This method is used to shear a GraphicsGroup.
toString This method is used to shear a GraphicsGroup.
transform This method is used to apply transformational methods to a GraphicsGroup.
translate This method is used to move the position of a GraphicsGroup.
