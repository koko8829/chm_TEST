Description 
This method returns the text property value of a specific cell within the head band area of the Grid component or the HeadValue value managed inside the cell.
Syntax 
Grid.getHeadValue( nCellIdx )
Parameters
Parameters Type Description
nCellIdx Number 
This sets the cell index to acquire the text property value or the HeadValue value managed inside the cell.
Sample Call:
var varValue = this.Grid00.getHeadValue(0);
Return
Type Description
String 
If a value is set in the cell's text property, the set value is returned.
If the cell's expr property is set, the value with EXPR applied to the text property value is returned.
If the cell's text property is bound to the Dataset object, the column value of the currently selected row in the bound Dataset object is returned.
If the HeadValue value managed inside the Cell is not “undefined”, the HeadValue value is returned.

If a non-existent Cell index is set, "null” is returned.
Remark 
◆ Returning HeadValue value

- If the cell edittype property value is not "none" and the value is changed in the editing state or changed through the setHeadValue method, the HeadValue value managed inside the cell is returned instead of the text property value.

- The initial value of HeadValue managed inside the cell is “undefined”.
  When the Cell text property value is changed or the format of the Grid component is changed, the HeadValue value is changed to “undefined”.
  If the HeadValue value is “undefined”, the text property value is returned when the method is called.

◆ Note on the nCellIdx Parameter

- The head band of Grid is composed of one row, and a row can have several sub rows.
  This does not concern the number of data in the Dataset object.
  In a row with sub rows, the cell index becomes (sub row index * number of columns + col property value of the cell).

- Cells defined in the head band have cell indexes sequentially from the top left to the bottom right.
  For merged cells, the entire merged area has the cell index of the first cell.

- If the cell's col property value has not been changed, the cell index starts at 0.
