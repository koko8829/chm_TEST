Description 
This property is used to set the opacity of a MultiComboTagBoxControl.
Syntax 
MultiComboTagBoxControl.opacity
Setting Syntax 
* XCSS
opacity : 0.8 ;
opacity : 80%;
opacity 
Specify a value between 0.0 and 1.0 or between 0.0% and 100.0%.
A setting of 0 (or 0%) is completely transparent, and a setting of 1 (or 100%) is completely opaque.
Values greater than or equal to 1 are treated as 1, and values less than or equal to 0 are treated as 0. Likewise, values above 100% are treated as 100%, and values below 0% as 0%.

XCSS can only be set as a number or percentage value.
Remark 
- The default value of this property is "undefined", which works as if the property is set to 1.

- If you apply the opacity property to a container component like a Div, it will also apply to the component's child components.

- In the case where a parent and its child have separate opacity values, the opacity of the child will be calculated by factoring in the opacity of the parent. 
For example, a parent and its child have opacity values of 0.5 and 0.4 respectively. In conclusion, the opacity of the child will be 0.2.
