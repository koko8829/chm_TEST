Description 
Form is a basic unit composing a screen and arrangement of each component internally so that the desired screen can be composed.
Remark 
It provides an environment in which a script can be executed.

Move focus Tab 
The focus moves to the next component according to the tab movement order.

If a Form has a Button whose defaultbutton property is set to "true", pressing the enter key will click the Button.

Entering this key combination will scroll a Form to the direction of an entered arrow key when focus falls on the Form or one of its child components and when the Form is scrollable.

Constructor 
Form.init( strName, vLeft, vTop , vWidth, vHeight [, vRight, vBottom, [vMinWidth, [vMaxWidth, [vMinHeight, [vMaxHeight]]]]] )
Parameters Type Description
strName String 
It sets the ID of a Form in the form of a string.
vLeft String, Number 
It specifies the value of the left property of a Form in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the right property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vTop String, Number 
It specifies the value of the top property of a Form in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the bottom property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vWidth String, Number 
It specifies the value of the width property of a Form in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vHeight String, Number 
It specifies the value of the height property of a Form in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vRight String, Number 
It specifies the value of the right property of a Form in pixel or percent (%).

This parameter will be ignored if both the vLeft and vWidth parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the left property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vBottom String, Number 
It specifies the value of the bottom property of a Form in pixel or percent (%).

This parameter will be ignored if both the vTop and vHeight parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the top property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vMinWidth String, Number 
It specifies a Form's minimum width that can be displayed in the Form.
vMaxWidth String, Number 
It specifies a Form's maximum width that can be displayed in the Form.

If this value is smaller than vMinWidth, it will be replaced by the value of vMinWidth.
vMinHeight String, Number 
It specifies a Form's minimum height that can be displayed in the Form.
vMaxHeight String, Number 
It specifies a Form's maximum height that can be displayed in the Form.

If this value is smaller than vMinHeight, it will be replaced by the value of vMinHeight.
Sample Call:
var objComp = new Form();

objComp.init( "Form00", 30, 120, 196, 46 );
objComp.init( "Form00", 30, 120, 196, 46, null, null );
objComp.init( "Form00", null, null, 300, "400px", "80%", 300 );
objComp.init( "Form00", 0, 0, 200, 100, null, null, 300, 500, 200, 500 );
objComp.init( "Form00", "Form22:10", 300, null, null, "Form33:10", "20%", 300, 500, 200, 500 );
 
// Create Object  
var objForm = new Form();  
objForm.init("Form", 30, 120, 196, 46, null, null);
objForm.init("Form", null, null, 300, "400px", "80%", 300);
or
var objForm = new Form("Form", 30, 120, 196, 46, null, null);
var objForm = new Form("Form", null, null, 300, "400px", "80%", 300);
 
// Add Object to Parent Form  
this.addChild("Form", objForm); 
 
// Insert Object to Parent Form  
this.insertChild(1, "Form", objForm); 
 
// Remove Object form Parent Form  
this.removeChild("Form"); 
 
// Show Object  
objForm.show(); 
 
// Destroy Object  
objForm.destroy(); 
objForm = null;
