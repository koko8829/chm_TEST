Property Type
Enum Expr Control Hidden ReadOnly Bind Collection StringResource
Description 
This property is used to set the area where the internal content area of the Div is automatically resized to fit the entire content.
Syntax 
Div.fittocontents[= enumFitTo]
Setting Syntax 
enumFitTo ::= 'none' | 'width' | 'height' | 'both'
 
this.Div00.fittocontents = "width";
"none" 
It maintains the size as designed regardless of internal contents.
"width" 
It adjusts the width of the contents area of a Div for internal contents.
"height" 
It adjusts the height of the contents area of a Div for internal contents.
"both" 
It adjusts the height and width of the contents area of a Div for internal contents.
Remark 
- If the fittocontents property value is not set, it will be set as "none."

- The overall size of the Panel component is determined by reflecting the -nexa-border and -nexa-padding areas and the spacing, horizontalgap and verticalgap property values in the size of the internal content area.

- The size of the internal content area is automatically adjusted only when the form is loaded or the fittocontents property value changes.


◆ Changing the Internal Content with a Script

- If the content of the Panel component is changed by a script, the size of the internal content area will not be automatically adjusted.
When the content of the Panel component is changed by a script and the internal content area needs to be resized, the resetScroll() method must be executed after the execution of the script that changed the content in order to readjust the size.

- When executing the resetScroll() method, the fittocontents property must be a value other than "none" to resize the internal content area.


◆ Properties That Do Not Operate According to the Type Property Value

The type and fittocontents property value settings take precedence over some of the other property value settings.
The properties of the Panel component or components placed in the Panel component shown below will not work.

- type = "horizontal," fittocontents = "width" or "both"
   Panel component: flexwrap, flexmainaxisalign
   Placed components: flexgrow, flexshrink

- type = "horizontal," fittocontents = "height" or "both"
   Panel component: flexcrossaxisalign

- type = "vertical," fittocontents = "height" or "both"
   Panel component: flexwrap, flexmainaxisalign
   Placed components: flexgrow, flexshrink

- type = "vertical," fittocontents = "width" or "both"
   Panel component: flexcrossaxisalign


◆ Operation When the Type Property Value is "Table"

The operation will vary depending on the tabletemplate property value.

- When set in px or % unit: The size of the cell will be adjusted to the size of the tabletemplate.
   A scroll may be appear if the size of the placed component is larger than the cell size.

- When set in * unit: The size of the cell will be  adjusted to the size of the component placed. 
   The tabletemplate proportions will be maintained.

◆ Operation When the Type Property Value Is “Default”

- The internal content area of the Div component will be automatically adjusted based on the component placed in the internal content area of the Div component whose right and bottom boundaries are close to the right and bottom boundaries of the internal content area of the Div component. 
   For example, if the Div component's width property value is 100 and a Button component with a left property value of 20, and width property value of 50 is placed, after which the fittocontents property value is set to "width," the internal content area of the Div component will decrease to the right border of the Button component and the width of the Div component will be displayed as 70.
