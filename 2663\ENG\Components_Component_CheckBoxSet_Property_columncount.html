Description 
This property sets the number of items displayed in a row in the CheckBoxSet component.
Syntax 
CheckBoxSet.columncount[= nColcount]
Setting Syntax 
this.CheckBoxSet00.columncount = 5;
nColcount 
The number of items displayed in a row is set to a number greater than or equal to -1.

When this value is set to 0, the columncount property is not applied and items are displayed according to the rowcount property value.

The setting values are changed according to the direction, rowcount, and fittocontents property values, so refer to the Remark underneath.
Remark 
(1) Action when the property value is set to default

- If the columncount property value is not set, it is applied as 0.
- If columncount, rowcount, and direction property values are not set (i.e. set to the default values), the CheckBoxSet items are displayed in one column. 

(2) Action when columncount and rowcount property values are set to 1 or larger

- When the columncount property value is 1 or larger and the direction property value is set to "horizontal", the rowcount property value is ignored. Also, when the columncount property value is 1 or larger and the rowcount property value is set to 0 or -1, the rowcount property value is ignored.
- When the rowcount property value is 1 or larger and the direction property value is set to "vertical", the columncount property value is ignored. Also, when the rowcount property value is 1 or larger and the columncount property value is set to 0 or -1, the columncount property value is ignored.

(3) Action when the property value is set to -1

- When the columncount property value is 1 or lager and the direction property value is set to "horizontal", the rowcount property value is ignored. If the fittocontents property value is not "none", the CheckBoxSet items are displayed in one column.
