Description 
This object is used to display a path that is connected to a Graphics component.
Remark 
- You can create this object dynamically through a script. You cannot design this object in Nexacro Studio. 

- This object is for displaying connected paths. For displaying unconnected paths, you should use a GraphicsPaths object.

- The moveTo() method applies only once in this object. From the second time, calling of the moveTo() method will be ignored.
Property
Name Description
fillstyle This property is used to color the inside area of a GraphicsPath.
id This property is used to set the identification of a GraphicsPath.
miterlimit This property sets a value used to determine whether the corner of a GraphicsPath becomes a beveled join even if the strokejoin property is set to "miter".
opacity This property is used to set the opacity of a GraphicsPath.
parent This read-only property holds the parent object of a GraphicsPath.
strokecap This property is used to set the cap of the path displayed in a GraphicsPath.
strokejoin This property determines the shape of joins in a GraphicsPath object.
strokepen This property is used to set the style and color of a path in a GraphicsPath.
visible This property is used to determine whether to make a GraphicsPath visible or not.
x This property is used to set the x-coordinate of a position where a GraphicsPath is displayed. The coordinate is calculated based on the position of the parent object.
y This property is used to set the y-coordinate of a position where a GraphicsPath is displayed. The coordinate is calculated based on the position of the parent object.
Method
Name Description
arcTo This method is used to add a path of an elliptical arc to a GraphicsPath. The arc stretches from the current position to the position delivered as arguments.
bezierCurveTo This method is used to add a path of a cubic Bezier curve to a GraphicsPath. The curve stretches from the current position to the position delivered as arguments.
clear This method is used to remove all the paths set on a GraphicsPath.
closePath This method is used to draw a straight line between a beginning and ending points specified in a GraphicsPath object.
destroy This method is used to completely remove a GraphicsPath from a memory.
getBoundRect This method returns the area where a GraphicsPath is displayed, including the border of the GraphicsPath. The area is calculated based on the Graphics component. The border is set by the strokepen property.
getNextSibling This method returns the next object among those under the same parent as a GraphicsPath.
getOffsetBoundRect This method returns the area where a GraphicsPath is displayed, including the border of the GraphicsPath. The area is calculated based on the parent object. The border is set by the strokepen property.
getOffsetRect This method returns the area where a GraphicsPath is displayed, including the border of the GraphicsPath. The area is calculated based on the parent object. The border is set by the strokepen property.
getOffsetRegion This method returns the area where a GraphicsPath is displayed, excluding the border of the GraphicsPath. The area is calculated based on the parent object. The border is set by the strokepen property.
getPathData This method returns the data on the path set in a GraphicsPath with the returned data in the form of a SVG string.
getPreviousSibling This method returns the previous object among those under the same parent as a GraphicsPath.
getRect This method returns the area where a GraphicsPath is displayed, with the area calculated based on the Graphics component.
getRegion This method returns the area where a GraphicsPath is displayed, excluding the border of the GraphicsPath. The area is calculated based on the Graphics component. The border is set by the strokepen property.
hasSegments This method returns whether a GraphicsPath has a path.
isHitTest This method finds the position marked by the coordinates delivered as arguments and returns whether the position is included in the area of a GraphicsPath.
isPointInPath This method returns whether the position of a certain point is included in the area of a GraphicsPath.
lineTo This method is used to add a straight line between the current position and a position delivered as arguments to a GraphicsPath object.
moveBy This method is used to move the beginning point of a GraphicsPath.
moveTo This method is used to set the beginning point of a GraphicsPath object.
quadraticCurveTo This method is used to add a path of a quadratic Bezier curve to a GraphicsPath. The curve stretches from the current position to the position delivered as arguments.
rotate This method is used to rotate a GraphicsPath.
scale This method is used to scale a GraphicsPath up/down.
setClipRect This method is used to set coordinates for clipping a certain region of a GraphicsPath.
setPathData This method is used to set a path in a GraphicsPath by specifying the path data in the form of SVG sting.
setTransform The method to apply the transform-related effect consecutively based on the original location of the GraphicsPath.
shear This method is used to shear a GraphicsPath.
toString This method returns a string representing the type that constitutes a GraphicsPath.
transform This method is used to apply transformational methods to a GraphicsPath.
translate This method is used to move the position of a GraphicsPath.
