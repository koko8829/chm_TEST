Description 
GroupBox is a Component which indicates group shape by inputting title to border area.
Structure
Contents Sizing 
The below table describes how to size a GroupBox and its internal contents.
 
GroupBox Component
Width 
The value of the width property of the GroupBox is applied. The internal contents will be hidden from sight partially if the value is smaller than the width of the internal contents.
Height 
The value of the height property of the GroupBox is applied. The internal contents will be hidden from sight partially if the value is smaller than the height of the internal contents.
 
groupboxtitle Control
Left 
The value of the left property is calculated by factoring in the titlealign and titlegap properties of the GroupBox.
Top 
The value of the top property is calculated by factoring in the titlealign and titlegap properties of the GroupBox.
Width 
If the -nexa-icon-position property of the groupboxtitle control is set to "left" or "right", the width of the control will be calculated by adding the widths of the icon and text. If -nexa-icon-position is set to "top" or "bottom", the bigger one will be applied between the icon and text widths.
Height 
If the -nexa-icon-position property of the groupboxtitle control is set to "left" or "right", the bigger one will be applied between the icon and text heights. If -nexa-icon-position is set to "top" or "bottom", the height of the control will be calculated by adding the heights of the icon and text.
 
icon
Left 
The value of the left property is calculated by factoring in the -nexa-icon-position,-nexa-text-align and -nexa-vertical-align properties.
Top 
The value of the top property is calculated by factoring in the -nexa-icon-position,-nexa-text-align and -nexa-vertical-align properties.
Width 
The width of the icon image is applied.
Height 
The height of the icon image is applied.
 
text
Left 
The value of the left property is calculated by factoring in the -nexa-icon-position,-nexa-text-align and -nexa-vertical-align properties.
Top 
The value of the top property is calculated by factoring in the -nexa-icon-position,-nexa-text-align and -nexa-vertical-align properties.
Width 
The width is calculated automatically according to the text length.
Height 
The height is calculated automatically according to the text size.
 
groupboxcontents Control
Left 
If the titlealign property of the GroupBox has a value beginning with "left", the left property of the groupboxcontents control will be calculated by halving the width of the groupboxtitle control. If the value of titlealign does not begin with "left", the value of the left property of the groupboxcontents control will be 0.
Top 
If the titlealign property of the GroupBox has a value beginning with "top", the top property of the groupboxcontents control will be calculated by halving the height of the groupboxtitle control. If the value of titlealign does not begin with "top", the value of the top property of the groupboxcontents control will be 0.
Width 
If the titlealign property of the GroupBox has a value beginning with "left" or "right", the width of the groupboxcontents control will be calculated by halving the width of the groupboxtitle control and subtracting the result from the width of the Groupbox. If the value of titlealign begins with "top" or "bottom", the value of the width property of the GroupBox will be applied.
Height 
If the titlealign property of the GroupBox has a value beginning with "top" or "bottom", the height of the groupboxcontents control will be calculated by halving the height of the groupboxtitle control and subtracting the result from the height of the Groupbox. If the value of titlealign begins with "left" or "right", the value of the height property of the GroupBox will be applied.

Constructor 
GroupBox.init( strName, vLeft, vTop , vWidth, vHeight [, vRight, vBottom, [vMinWidth, [vMaxWidth, [vMinHeight, [vMaxHeight]]]]] )
Parameters Type Description
strName String 
It sets the ID of a GroupBox in the form of a string.
vLeft String, Number 
It specifies the value of the left property of a GroupBox in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the right property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vTop String, Number 
It specifies the value of the top property of a GroupBox in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the bottom property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vWidth String, Number 
It specifies the value of the width property of a GroupBox in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vHeight String, Number 
It specifies the value of the height property of a GroupBox in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vRight String, Number 
It specifies the value of the right property of a GroupBox in pixel or percent (%).

This parameter will be ignored if both the vLeft and vWidth parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the left property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vBottom String, Number 
It specifies the value of the bottom property of a GroupBox in pixel or percent (%).

This parameter will be ignored if both the vTop and vHeight parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the top property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vMinWidth String, Number 
It specifies a GroupBox's minimum width that can be displayed in the Form.
vMaxWidth String, Number 
It specifies a GroupBox's maximum width that can be displayed in the Form.

If this value is smaller than vMinWidth, it will be replaced by the value of vMinWidth.
vMinHeight String, Number 
It specifies a GroupBox's minimum height that can be displayed in the Form.
vMaxHeight String, Number 
It specifies a GroupBox's maximum height that can be displayed in the Form.

If this value is smaller than vMinHeight, it will be replaced by the value of vMinHeight.
Sample Call:
var objComp = new GroupBox();

objComp.init( "GroupBox00", 30, 120, 196, 46 );
objComp.init( "GroupBox00", 30, 120, 196, 46, null, null );
objComp.init( "GroupBox00", null, null, 300, "400px", "80%", 300 );
objComp.init( "GroupBox00", 0, 0, 200, 100, null, null, 300, 500, 200, 500 );
objComp.init( "GroupBox00", "GroupBox22:10", 300, null, null, "GroupBox33:10", "20%", 300, 500, 200, 500 );
 
// Create Object  
var objGroupBox = new GroupBox();  
objGroupBox.init("GroupBox", 30, 120, 196, 46, null, null);
objGroupBox.init("GroupBox", null, null, 300, "400px", "80%", 300);
or
var objGroupBox = new GroupBox("GroupBox", 30, 120, 196, 46, null, null);
var objGroupBox = new GroupBox("GroupBox", null, null, 300, "400px", "80%", 300);
 
// Add Object to Parent Form  
this.addChild("GroupBox", objGroupBox); 
 
// Insert Object to Parent Form  
this.insertChild(1, "GroupBox", objGroupBox); 
 
// Remove Object form Parent Form  
this.removeChild("GroupBox"); 
 
// Show Object  
objGroupBox.show(); 
 
// Destroy Object  
objGroupBox.destroy(); 
objGroupBox = null;
