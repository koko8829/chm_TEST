Description 
SubCellControl is a control that consists of multiple Grid cells.
Remark 
- You can create a CellSubCellControl by merging cells, which can be done through the "Merge Cells(Having child cells)" menu of the Grid Contents Editor of Nexacro Studio.
  A merged cell belongs to the CellSubCellControl object, separate from the just cell control.

- If you merge cells through the "Merge Cells(A big cell)" menu of the  Grid Contents Editor, the merged cell will belong to the ordinary cell object.
   A merged cell is a single object, so it holds one area and a single set of properties.

- You can set a property of a child cell with a CellSubCellControl through the setSubCellProperty() method.
   You need to use the getSubCellProperty() method to obtain the value of a child cell's property.
   Each child cell has its own index number.

- A child cell of a CellSubCellControl holds almost the same set of properties with a GridCellControl . However, some properties will not be applied if the child cell belongs to the CellSubCellControl.
   There is reason for such properties not to be applicable.
   The reason is that those properties will be applied if the merged cell is split into separate cells again by the splitContentsCell() method of the Grid.

- Some properties of a cell can be bound to a Dataset column or set with an expression.
 For such an expression, you can use the bound Dataset, column ID as well as the functions defined in accessible areas.

- The below list presents the reserved words that represent each designated value.
  > this: the Grid cell where the expression is defined. (It does not indicate the Form.)
  > comp: the Grid that contains the cell
  > dataset: the Dataset bound to the Grid
  > currow: the index number of the row that the expression calculates
  > [Column ID]: the value of the Dataset column bound to the row that the expression calculates
  > The properties and methods of the Dataset bound to the Grid
      ex) "dataset.rowcount", "dataset.colcount", "dataset.rowposition", "dataset.getCount()"
