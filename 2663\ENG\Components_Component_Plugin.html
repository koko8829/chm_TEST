Description 
A Plugin is a wrapper component to use ActiveX or an applet.
Remark 
- An ActiveX control or an applet will be loaded according to property values.

- You cannot use an absolute path and relative path for the properties that require an input of a path.
 You must use an HTTP full URL.

- If you load a Flash-based content, the movie property does not support a path defined in Services of the TypeDefinition (eg. Base::sample.swf).
Structure
Constructor 
Plugin.init( strName, vLeft, vTop , vWidth, vHeight [, vRight, vBottom, [vMinWidth, [vMaxWidth, [vMinHeight, [vMaxHeight]]]]] )
Parameters Type Description
strName String 
It sets the ID of a Plugin in the form of a string.
vLeft String, Number 
It specifies the value of the left property of a Plugin in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the right property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vTop String, Number 
It specifies the value of the top property of a Plugin in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the bottom property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vWidth String, Number 
It specifies the value of the width property of a Plugin in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vHeight String, Number 
It specifies the value of the height property of a Plugin in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vRight String, Number 
It specifies the value of the right property of a Plugin in pixel or percent (%).

This parameter will be ignored if both the vLeft and vWidth parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the left property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vBottom String, Number 
It specifies the value of the bottom property of a Plugin in pixel or percent (%).

This parameter will be ignored if both the vTop and vHeight parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the top property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vMinWidth String, Number 
It specifies a Plugin's minimum width that can be displayed in the Form.
vMaxWidth String, Number 
It specifies a Plugin's maximum width that can be displayed in the Form.

If this value is smaller than vMinWidth, it will be replaced by the value of vMinWidth.
vMinHeight String, Number 
It specifies a Plugin's minimum height that can be displayed in the Form.
vMaxHeight String, Number 
It specifies a Plugin's maximum height that can be displayed in the Form.

If this value is smaller than vMinHeight, it will be replaced by the value of vMinHeight.
Sample Call:
var objComp = new Plugin();

objComp.init( "Plugin00", 30, 120, 196, 46 );
objComp.init( "Plugin00", 30, 120, 196, 46, null, null );
objComp.init( "Plugin00", null, null, 300, "400px", "80%", 300 );
objComp.init( "Plugin00", 0, 0, 200, 100, null, null, 300, 500, 200, 500 );
objComp.init( "Plugin00", "Plugin22:10", 300, null, null, "Plugin33:10", "20%", 300, 500, 200, 500 );
 
// Create Object  
var objPlugin = new Plugin();  
objPlugin.init("Plugin", 30, 120, 196, 46, null, null);
objPlugin.init("Plugin", null, null, 300, "400px", "80%", 300);
or
var objPlugin = new Plugin("Plugin", 30, 120, 196, 46, null, null);
var objPlugin = new Plugin("Plugin", null, null, 300, "400px", "80%", 300);
 
// Add Object to Parent Form  
this.addChild("Plugin", objPlugin); 
 
// Insert Object to Parent Form  
this.insertChild(1, "Plugin", objPlugin); 
 
// Remove Object form Parent Form  
this.removeChild("Plugin"); 
 
// Show Object  
objPlugin.show(); 
 
// Destroy Object  
objPlugin.destroy(); 
objPlugin = null;
Property
Name Description
adjustalpha This property determines whether to re-adjust the alpha value applied to the image of a windowless Plugin.
archive This property is used to set the archive property defined in the object linked to a Plugin.
bottom This property is used to set the bottom coordinate value for displaying the Plugin.
classid This property is used to specify the class identifier (classid) or programmatic identifier (progid) of the object to connect a Plugin to.
classid64 This property is used to specify the class identifier (classid) or programmatic identifier (progid) of the object to connect a Plugin to. This property is applicable when the connected object is 64-bit.
code This property is used to set the codes of the object to link to a Plugin.
codebase This property is used to set the location of the object linked to a Plugin.
contents This property is used to set a certain property of a Plugin. Otherwise, this property is used to save the metadata that will be used in the ActiveX of a Plugin.
enable This property is used to determine whether to enable a Plugin.
flexgrow This property is used to resize the component by dividing the main axis (the horizontal axis, if the type property value is "horizontal" and the vertical axis if the type property value is "vertical") margin within the layout container by the specified ratio.
flexshrink This property is used to readjust the size of the components by the ratio specified so that they do not exceed the container area when they are placed on the main axis (the horizontal axis, if the type property value is "horizontal" and the vertical axis if the type property value is "vertical") within the layout container.
height This property is used to set the height for displaying the Plugin.
id This property is used to set the identification of a Plugin.
initvalueid This property is used to set the ID of an InitValue to be applied to an Plugin.
layoutorder This property is used to set the order in which components are aligned on the main axis (the horizontal axis, if the type property value is "horizontal" and the vertical axis if the type property value is "vertical") within the layout container.
left This property is used to set the left coordinate value for displaying the Plugin.
license This property is used to set the license of the object linked to a Plugin.
lpkpath This property is used to set the ipk path of the object linked to a Plugin.
maxheight This property is used to set the maximum height of a Plugin for the component to be displayed on the screen.
maxwidth This property is used to set the maximum width of a Plugin for the component to be displayed on the screen.
mimetype This property is used to set the type of the object to link to a Plugin.
minheight This property is used to set the minimum height of a Plugin for the component to be displayed on the screen.
minwidth This property is used to set the minimum width of a Plugin for the component to be displayed on the screen.
name This property is used to specify the name of a Plugin.
-nexa-border This property is used to set the border of a Plugin.
parent This read-only property holds the parent object of a Plugin.
pluginpage This property is used to set the location of the object to link to a Plugin.
pluginsrc This property is used to set the data and src properties defined in the object to link to a Plugin.
popupstyle This property determines whether to create a Plugin in the form of a pop-up when the object linked to the Plugin is a window.
positionstep This property is used to specify the index number of a step page where a Plugin is displayed if the parent object uses the step feature.
right This property is used to set the right coordinate value for displaying the Plugin.
tablecellarea This property is used to set the area where the component is aligned within the virtual table cell area in the layout container.
taborder This property is used to define the tab order number of a Plugin. Tab order means the order of moving focus among components when the focus is moved by entering the tab key.
tabstop This property determines whether a Plugin receives focus when you move the focus between components by pressing the tab key.
top This property is used to set the top coordinate value for displaying the Plugin.
usepersistdata The attribute to set whether to use the persistdata.
visible This property is used to determine whether to make a Plugin visible or not.
width This property is used to set the width for displaying the Plugin.
windowed This property determines whether an object to link to a Plugin has a window.
Method
Name Description
callMethod The method to call the method of the object connected to the Plugin.
getOffsetBottom This method is used to obtain the bottom position of a Plugin in pixel, based on the parent component's top side.
getOffsetHeight This method is used to obtain the height of a Plugin in pixel.
getOffsetLeft This method is used to obtain the left position of a Plugin in pixel, based on the parent component's left side.
getOffsetRight This method is used to obtain the right position of a Plugin in pixel, based on the parent component's left side.
getOffsetTop This method is used to obtain the top position of a Plugin in pixel, based on the parent component's top side.
getOffsetWidth This method is used to obtain the width of a Plugin in pixel.
getPixelBottom This method returns the value of a Plugin's bottom property in pixel.
getPixelHeight This method returns the height of a Plugin in pixel.
getPixelLeft This method returns the value of a Plugin's left property in pixel.
getPixelRight This method returns the value of a Plugin's right property in pixel.
getPixelTop This method returns the value of a Plugin's top property in pixel.
getPixelWidth This method returns the width of a Plugin in pixel.
getProperty This method returns the value of a certain property defined in the object linked to a Plugin.
init This method is used to initialize a Plugin after creating it in a script dynamically.
install This method is used to install the object to link to a Plugin.
isInstalled This method is used to check whether the object linked to a Plugin has been installed in the system.
isLoaded This method is used to check whether an object has been successfully created and loaded onto a Plugin.
move This method is used to change the position and size of a Plugin.
moveToNext This method is used to cause a Plugin to be displayed one level higher on the screen than the component delivered as an argument. In this situation, those two components must share the same parent.
moveToPrev This method is used to cause a Plugin to be displayed one level lower on the screen than the component delivered as an argument. In this situation, those two components must share the same parent.
sendToBack This method is used to cause a Plugin to be displayed at the bottom on the screen. This method applies to among the components under the same parent.
sendToNext This method is used to cause a Plugin to be displayed one level lower on the screen. This method applies to among the components under the same parent.
setFocus This method is used to give focus to a Plugin.
setOffsetBottom This method is used to specify the bottom position of a Plugin in pixel, based on the parent component's top position.
setOffsetHeight This method is used to specify the height of a Plugin in pixel.
setOffsetLeft This method is used to specify the left position of a Plugin in pixel, based on the parent component's left position.
setOffsetRight This method is used to specify the right position of a Plugin in pixel, based on the parent component's left position.
setOffsetTop This method is used to specify the top position of a Plugin in pixel, based on the parent component's top position.
setOffsetWidth This method is used to specify the width of a Plugin in pixel.
setProperty This property is used to set the value of a certain property defined in the object linked to a Plugin.
show This method is used to display a Plugin on the screen. The Plugin mentioned here refers to the one that is dynamically created through a script.
