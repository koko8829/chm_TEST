getUTCHours

Description 
This method is used to return the hour drawn from the value of a Date object. The calculation is conducted based on Universal Time Coordinated (UTC).
Syntax 
Date.getUTCHours()
Parameters
Sample Call:
var objDate = new nexacro.Date();

trace(objDate.getUTCHours());
Return
Type Description
Number 
It returns an hour value that ranges from 0 to 23.
Remark 
- If you want to obtain an hour based on the value of a Date object, you need to use the getHours() method.

- The getUTCHours() method returns an integer ranging from 0 to 23, with 0 indicating midnight.
  If this method returns 0, it will mean either that the time is earlier than 1 a.m. or that you did not specify time when creating the Date object.
  If you did not specify time when creating a Date object, an hour, minute and second will be all printed as 0.
Example
Script 
var objDate = new nexacro.Date( 2008, 0, 0, 0, 0, 0 );

trace(objDate.getUTCHours());         // 15
trace(objDate.getHours());               // 0
