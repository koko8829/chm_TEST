open
Supported Environments
[Desktop]
Nexacro Runtime Environment (NRE)
Windows macOS
Web Runtime Environment (WRE)
Edge Chrome Safari Firefox Opera
[Mobile]
Nexacro Runtime Environment (NRE)
Android iOS/iPadOS
Web Runtime Environment (WRE)
Android Default Web Browser iOS/iPadOS Default Web Browser
Description 
This is a method of displaying a dynamically generated ChildFrame as Modeless Window.
Syntax 
nexacro.open( strID, strFormURL, objParentFrame, {objArguList}, strOpenStyle, nLeft, nTop [, nWidth, nHeight [, objOpener [, strExtOpenStyle ]]])
Parameters
Parameters Type Description
strID String 
It specifies the ID of a ChildFrame to display as modeless window.
strFormURL String 
It specifies the URL of a Form that will be loaded onto the ChildFrame displayed as a modeless window.
objParentFrame Object 
This is an object that will serve as the parent of the ChildFrame displayed as a modeless window.

If you specify "null", nothing will be defined as the parent.
objArguList Object 
It specifies a list of variables to add to a ChildFrame that will be displayed as a modeless window, with the format {variable name:variable value, variable name:variable value}.
strOpenStyle String 
You can specify properties of a ChildFrame that will be displayed as a modeless window, with the format "Property1=value Property2=value".

You can set some properties of a ChildFrame and properties are separated by spaces.
You cannot set CSS-related properties (e.g. background, -nexa-border, font).

The value "all" of the dragmovetype property is supported only by the Windows NRE.
The layered property is supported only by the Windows NRE and macOS NRE.
The value "maximize" of the openstatus property is supported only by the Windows NRE.
The values "normal" and "minimize" of the openstatus property is supported only by the Windows NRE.
The titlebar property is supported only by the Windows NRE.
The showstatusbar property is supported only by the Windows NRE.
nLeft Number 
It specifies the coordinates of the left edge of a ChildFrame that will be displayed as a modeless window.
nTop Number 
It specifies the coordinates of the top edge of a ChildFrame that will be displayed as a modeless window..
nWidth Number 
It specifies the width of a ChildFrame that will be displayed as a modeless window..
nHeight Number 
It specifies the height of a ChildFrame that will be displayed as a modeless window..
objOpener Object 
It specifies a Form object that will serve as an opener.

If unspecified, this parameter will take the Form defined by the objParentFrame parameter.
strExtOpenStyle String 
You can specify additional properties of the ChildFrame with the format "Property1=value1 Property2=value2".
Applicable properties vary according to environments and properties are separated by spaces.


* The noactivate property is applicable only to the Windows NRE, and the default value of the property is "false".
If you set the noactivate property to "true", the displayed ChildFrame will not be activated.
If you set noactivate to "false", the displayed ChildFrame will be activated and focused.
The noactivate property is used to prevent focus from going to the displayed ChildFrame.
Sample Call:
var bSucc;
bSucc = nexacro.open("modeless", "Base::Form01.xfdl",  this.getOwnerFrame(), {a:'aaa', b:'bbb'}, "showtitlebar=true showstatusbar=false", 0, 0);
bSucc = nexacro.open("modeless", "Base::Form01.xfdl",  this.getOwnerFrame(), {a:'aaa', b:'bbb'}, "showtitlebar=true showstatusbar=false", 0, 0, 640, 480);
bSucc = nexacro.open("modeless", "Base::Form01.xfdl",  this.getOwnerFrame(), {a:'aaa', b:'bbb'}, "showtitlebar=true showstatusbar=false", 0, 0, 640, 480, this);
bSucc = nexacro.open("modeless", "Base::Form01.xfdl",  this.getOwnerFrame(), {a:'aaa', b:'bbb'}, "showtitlebar=true showstatusbar=false", 0, 0, 640, 480, this, "location=true menubar=true toolbar=true");
bSucc = nexacro.open("modeless", "Base::Form01.xfdl",  this.getOwnerFrame(), {a:'aaa', b:'bbb'}, "showtitlebar=true showstatusbar=false", 0, 0, 640, 480, this, "noactivate=true");
Return
Type Description
Boolean 
If the Modeless Window is displayed as normal, true will be returned. 

If the Modeless Window is not displayed as normal, false will be returned.
Remark 
- When the objParentFrame parameter is set as null, the parent will not be set for the ChildFrame displayed in a popup window, but the popup ChildFrame will be added to the popup list in the window of the MainFrame object.

- If the autosize property value of ChildFrame is true, it will automatically fit to the size of the Form even the nWidth and nHeight parameter values are omitted. 
    Otherwise, the default values of nWidth and nHeight will be 0, so they may not be displayed on the screen. (The default value of the autosize property is true.) 

- Properties subject to environmental constraints in the strOpenStyle parameter are constraints that apply only when created using the open() method.
    Please check the Chm page of each property for the actual support environment for each property.
