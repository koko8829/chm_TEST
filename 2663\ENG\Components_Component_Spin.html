Description 
The Spin is a component used to input number values within a certain range.
Remark 
- Since the value of Spin is a string type, not a number type, you must first convert it to a number for math operation.

- For Spin, you can enter not only an integer but also the number of digits beyond the decimal point.

- You can increase or decrease the number using the up or down arrow of the key using the Spin.

- The Spin can be connected to one value of the bound Dataset.
Structure
Contents Sizing 
The below table describes how to size a Spin and its internal contents.
NULL 
Spin Component
Width 
The value of the width property of the Spin is applied. The internal contents will be hidden from sight partially if the value is smaller than the width of the internal contents.
Height 
The value of the height property of the Spin is applied. The internal contents will be hidden from sight partially if the value is smaller than the height of the internal contents.
NULL 
spinedit Control  * A spinedit control will not appear if the type property is set to "spinonly".
Left 
The width of the spinupbutton control will be applied if the buttonposition property is set to "left". The value of the left property of the spinedit control will be 0 if buttonposition is set to "right".
Top 
The value of the top property is 0.
Width 
The width is calculated by subtracting the width of the spinupbutton control from the width of the Spin.
Height 
The value of the height property of the Spin is applied.
NULL 
spinupbutton Control
Left 
The value of the left property will be 0 if the buttonposition property is set to "left". The left property of the spinupbutton control will be calculated by subtracting the width of the spinupbutton control from the width of the Spin if buttonposition is set to "right".
Top 
The value of the top property is 0.
Width 
The value of the buttonsize property is applied. If buttonsize is not set, the height of the Spin will be applied.
Height 
The height of the control is calculated by halving the height of the Spin.
NULL 
spindownbutton Control

The value of the left property will be 0 if the buttonposition property is set to "left". The left property of the spindownbutton control will be calculated by subtracting the width of the spindownbutton control from the width of the Spin if buttonposition is set to "right".
Top 
The value of the top property is calculated by subtracting the height of the spinupbutton control from the height of the Spin.
Width 
The value of the buttonsize property is applied. If buttonsize is not set, the height of the Spin will be applied.
Height 
The height of the control is calculated by halving the height of the Spin.
Basic Key Action
Action Key Description
Move focus Tab 
The focus moves to the next component according to the tab movement order.
Change value Up/Down arrow key 
Pressing the up/down arrow key will increase/decrease the value of a Spin by the value of increment property.
Accessibility Key Action 
This is a key action to be applied when the accessibility function is activated.
Action Key Description
Move focus Tab 
The focus moves to the next component according to the tab movement order.
Move selection box Up/Down arrow key 
A selection box will move to the previous/next component according to the tab order.
Change value Ctrl + up/down arrow key 
Pressing the Ctrl + up/down arrow key will increase/decrease the value of a Spin by the value of increment property.
Constructor 
Spin.init( strName, vLeft, vTop , vWidth, vHeight [, vRight, vBottom, [vMinWidth, [vMaxWidth, [vMinHeight, [vMaxHeight]]]]] )
Parameters Type Description
strName String 
It sets the ID of a Spin in the form of a string.
vLeft String, Number 
It specifies the value of the left property of a Spin in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the right property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vTop String, Number 
It specifies the value of the top property of a Spin in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the bottom property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vWidth String, Number 
It specifies the value of the width property of a Spin in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vHeight String, Number 
It specifies the value of the height property of a Spin in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vRight String, Number 
It specifies the value of the right property of a Spin in pixel or percent (%).

This parameter will be ignored if both the vLeft and vWidth parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the left property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vBottom String, Number 
It specifies the value of the bottom property of a Spin in pixel or percent (%).

This parameter will be ignored if both the vTop and vHeight parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the top property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vMinWidth String, Number 
It specifies a Spin's minimum width that can be displayed in the Form.
vMaxWidth String, Number 
It specifies a Spin's maximum width that can be displayed in the Form.

If this value is smaller than vMinWidth, it will be replaced by the value of vMinWidth.
vMinHeight String, Number 
It specifies a Spin's minimum height that can be displayed in the Form.
vMaxHeight String, Number 
It specifies a Spin's maximum height that can be displayed in the Form.

If this value is smaller than vMinHeight, it will be replaced by the value of vMinHeight.
