Description 
The CheckBoxSet component is used to select one or multiple values from among multiple candidate values.
Structure
Contents Sizing 
This is the standard for determining the size of the CheckBoxSet component.

- The following is applicable if the fittocontents property value of the CheckBoxSet component is “none”.
 
CheckBoxSet Component
Width 
The width property value of the CheckBoxSet component is applied.
Height 
The height property value of the CheckBoxSet component is applied.
 
checkboxsetitem
Left 
The position of the checkboxsetitem control is determined by the rowcount, columncount, and direction property values, and the value calculated accordingly is applied.
If the rowcount and columncount property values are "-1", the Width values of other checkboxsetitem controls are also included in the calculation.
Top 
The position of the checkboxsetitem control is determined by the rowcount, columncount, and direction property values, and the value calculated accordingly is applied.
Width 
The value obtained by dividing the Width of the CheckBoxSet component by the number of columns determined by the rowcount, columncount, and direction property values is applied.
If the rowcount and columncount property values are "-1", the value obtained by adding the Width of the icon to the Width of the text is applied.
If this is less than the Width of the internal content, the part of the internal content is displayed.
Height 
The value obtained by dividing the Height of the CheckBoxSet component by the number of rows determined by the rowcount, columncount, and direction property values is applied.
If this is less than the Height of the internal content, the part of the internal content is displayed.
 
icon
Left 
The values calculated according to the property values of the -nexa-icon-position, -nexa-text-align, and -nexa-vertical-align of the checkboxsetitem control are applied.
Top 
The values calculated according to the property values of the -nexa-icon-position, -nexa-text-align, and -nexa-vertical-align of the checkboxsetitem control are applied.
Width 
The width of the icon image is applied.
Height 
The height of the icon image is applied.
 
text
Left 
The values calculated according to the property values of the -nexa-icon-position, -nexa-text-align, and -nexa-vertical-align of the checkboxsetitem control are applied.
Top 
The values calculated according to the property values of the -nexa-icon-position, -nexa-text-align, and -nexa-vertical-align of the checkboxsetitem control are applied.
Width 
The value calculated to fit the length of the text is applied.
Height 
The value calculated to fit the text size is applied.
 
- The following is applicable if the fittocontents property value of the CheckBoxSet component is “width”.
 
CheckBoxSet Component

* If both the rowcount and columncount property values of the CheckBoxSet component are "-1", the rowcount and columncount property values are processed as "0".
Width 
The sum of the Widths of the checkboxsetitem control of each column is applied.
 
checkboxsetitem
Left 
The values calculated according to the number of checkboxsetitem controls, the Width value, and the rowcount, columncount, and direction property values of the CheckBoxSet component are applied.
Top 
The values calculated according to the number of checkboxsetitem controls, the Height value, and the rowcount, columncount, and direction property values of the CheckBoxSet component are applied.
Width 
The value obtained by adding the Width of the text to the Width of the icon is applied.
If there is a value greater than the Width value of the current checkboxsetitem control among the width values of checkboxsetitem controls located in the same column, that value is applied.
 
- The following is applicable if the fittocontents property value of the CheckBoxSet component is “height”.
 
CheckBoxSet Component

* If both the rowcount and columncount property values of the CheckBoxSet component are "-1", the rowcount and columncount property values are processed as "0".
Height 
The sum of the Heights of the checkboxsetitem control of each row is applied.
 
checkboxsetitem
Left 
The values calculated according to the number of checkboxsetitem controls, the Width value, and the rowcount, columncount, and direction property values of the CheckBoxSet component are applied.
Top 
The values calculated according to the number of checkboxsetitem controls, the Height value, and the rowcount, columncount, and direction property values of the CheckBoxSet component are applied.
Height 
If the -nexa-icon-position property value of the checkboxsetitem control is "left" or "right", the larger of the icon Height and text Height is applied.
If the -nexa-icon-position property value is "top" or "bottom", the value obtained by adding the Height of the text to the Height of the icon is applied.

Move through the items consecutively.

Constructor 
CheckBoxSet.init( strName, vLeft, vTop , vWidth, vHeight [, vRight, vBottom, [vMinWidth, [vMaxWidth, [vMinHeight, [vMaxHeight]]]]] )
Parameters Type Description
strName String 
It sets the ID of a CheckBoxSet in the form of a string.
vLeft String, Number 
It specifies the value of the left property of a CheckBoxSet in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the right property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vTop String, Number 
It specifies the value of the top property of a CheckBoxSet in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the bottom property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vWidth String, Number 
It specifies the value of the width property of a CheckBoxSet in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vHeight String, Number 
It specifies the value of the height property of a CheckBoxSet in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vRight String, Number 
It specifies the value of the right property of a CheckBoxSet in pixel or percent (%).

This parameter will be ignored if both the vLeft and vWidth parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the left property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vBottom String, Number 
It specifies the value of the bottom property of a CheckBoxSet in pixel or percent (%).

This parameter will be ignored if both the vTop and vHeight parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the top property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vMinWidth String, Number 
It specifies a CheckBoxSet's minimum width that can be displayed in the Form.
vMaxWidth String, Number 
It specifies a CheckBoxSet's maximum width that can be displayed in the Form.

If this value is smaller than vMinWidth, it will be replaced by the value of vMinWidth.
vMinHeight String, Number 
It specifies a CheckBoxSet's minimum height that can be displayed in the Form.
vMaxHeight String, Number 
It specifies a CheckBoxSet's maximum height that can be displayed in the Form.

If this value is smaller than vMinHeight, it will be replaced by the value of vMinHeight.
