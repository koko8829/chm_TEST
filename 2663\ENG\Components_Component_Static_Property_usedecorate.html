Description 
This property determines whether to apply the decoration effect to the text in a Static.
Syntax 
Static.usedecorate[= bDecorate]
Setting Syntax 
bDecorate ::= 'true' | 'false'
 
this.Static00.usedecorate = true; 
this.Static00.text = "<u v='true'><i v='true'>Static00</i></u>";
"true" 
This value indicates that the decoration effect is applied to the text.
"false" 
This value indicates that the decoration effect is not applied to the text.
Remark 
- The default value of this property is "false".

- When you enter the value of the text property directly to source codes, you should replace special characters like "<" and ">" with the characters conforming the HTML rules like "&lt;" and "&gt;".
 A font beginning with @, such as @Gullim, is not supported in HTML.

- You can decorate text by using tags.

- The link function works in the same way as the execBrowser() method. 


â—† Some character input restrictions

- If you use the "<", ">" special characters in the Tag form and use alphanumeric characters outside the specified Tag form, the input value will be deleted.
  For example, if you enter in the form of "<a href=''>", the value will be processed by deleting it.
