ListViewDragEventInfo
Supported Environments
[Desktop]

Windows macOS

Edge Chrome Safari Firefox Opera
[Mobile]
Nexacro Runtime Environment (NRE)
Android iOS/iPadOS
Web Runtime Environment (WRE)
Android Default Web Browser iOS/iPadOS Default Web Browser
Description 
This EventInfo object is delivered to the events that are related to mouse-dragging in the area of a ListView component.
Property
Name Description
altkey This property holds a value indicating whether an event occurred with the Alt key pressed.
bandid This read-only property holds the information on a band that triggered an event.
button This read-only property holds information on a button or touch that fired an event.
canvasx This property holds the x-coordinate of the point where you clicked or touched when an event occurred, calculated based on the area of an entire component.
canvasy This property holds the y-coordinate of the point where you clicked or touched when an event occurred, calculated based on the area of an entire component.
cellid This read-only property holds the ID of a cell that triggered an event.
clientx This property holds the x-coordinate of the point where you clicked or touched when an event occurred, calculated based on the client area of a component.
clienty This property holds the y-coordinate of the point where you clicked or touched when an event occurred, calculated based on the client area of a component.
ctrlkey This property holds a value indicating whether an event occurred with a Control key pressed.
datatype This read-only property holds the location where a dragging action begins.
eventid This read-only property holds the ID of an event.
filelist This read-only property holds an array containing the information on the files that are dragged from outside the application.
fromobject This property holds an event-triggering component.
fromreferenceobject This property holds the component or subcontrol that triggered an event.
metakey The attribute with state on whether the command (or windows) key was entered at the occurrence of an event.
row This read-only property holds the row index number of an item that triggered an event.
screenx This property holds the x-coordinate of the point where the monitor is clicked or touched. The coordinate is calculated based on the monitor area.
screeny This property holds the y-coordinate of the point where the monitor is clicked or touched. The coordinate is calculated based on the monitor area.
shiftkey This property holds a value indicating whether an event occurred with the shift key pressed.
sourceobject This property holds the component where a mouse dragging operation occurred for the first time.
sourcereferenceobject This property holds the component or subcontrol where a mouse dragging operation occurred for the first time.
userdata This is a property whose value can be set by users in a dragging-related event.
