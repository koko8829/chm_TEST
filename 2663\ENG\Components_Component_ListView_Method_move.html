move

Description 
This method is used to change the position and size of a ListView.
Syntax 
ListView.move( vLeft, vTop [, vWidth, vHeight [, vRight, vBottom]] )
Parameters
Parameters Type Description
vLeft String, Number 
This parameter defines the value of the left property of a ListView in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the right property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vTop String, Number 
This parameter defines the value of the top property of a ListView in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the bottom property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vWidth String, Number 
This parameter defines the value of the width property of a ListView in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vHeight String, Number 
This parameter defines the value of the height property of a ListView in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vRight String, Number 
This parameter defines the value of the right property of a ListView in pixel or percent (%).

This parameter will be ignored if both the vLeft and vWidth parameters are all defined.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the left property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vBottom String, Number 
This parameter defines the value of the bottom property of a ListView in pixel or percent (%).

This parameter will be ignored if both the vTop and vHeight parameters are all defined.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the top property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
Sample Call:
this.ListView00.move(10,10);
this.ListView00.move(10,10,100,100);
this.ListView00.move(null, null, 300, "400px", "80%", 300 );
Return
None
Remark 
- If you include a base component in the value of an argument, the position of a ListView will be determined relative to the base component.
 If you want to include a base component in the value of the vLeft, vTop, vWidth, vHeight, vRight or vBottom parameter, the argument must be formatted as "ID:Value".

- A base component specified as an argument must have the same parent as the ListView. Moreover, it must be created earlier than the ListView when the Form is loaded.
 If you do not specify a base component or specify an invalid value, the position of the ListView will be based on the parent.

- If you want to add a base component or an unit as an argument, you must append the value in the form of a string.
