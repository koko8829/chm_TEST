Description 
The ImageViewer is a component used to output the image on the screen.

The button can be produced in the desired shape and used with the ImageViewer.
Structure
Contents Sizing 
The below table describes how to size an ImageViewer and its internal contents.  - The below descriptions are true when the fittocontents property of the ImageViewer is set to "none".
 
ImageViewer Component
Width 
The value of the width property of the ImageViewer is applied. The internal contents will be hidden from sight partially if the value is smaller than the width of the internal contents. If the value is bigger, empty space will be created.
Height 
The value of the height property of the ImageViewer is applied. The internal contents will be hidden from sight partially if the value is smaller than the height of the internal contents. If the value is bigger, empty space will be created.
 
imagearea Control
Left 
The left property is calculated automatically according to the values of the stretch and imagealign properties of the ImageViewer.
Top 
The top property is calculated automatically according to the values of the stretch and imagealign properties of the ImageViewer.
Width 
The width is calculated automatically according to the value of the stretch property of the ImageViewer.
Height 
The height is calculated automatically according to the value of the stretch property of the ImageViewer.
 
imagetext Control
Left 
The value of the left property is 0.
Top 
The value of the top property is 0.
Width 
The width of the ImageViewer is applied.
Height 
The value of the height property of the ImageViewer is applied.
 
- The below descriptions are true when the fittocontents property of the ImageViewer is set to "width".   The size of the image can vary according to the value of the stretch property after the width of the ImageViewer is determined.
 
ImageViewer Component
Width 
The bigger one is applied between the widths of the imagearea control and imagetext control.
 
imagearea Control
Left 
The value of the left property is 0.
Width 
The width of the original image is applied.
 
imagetext Control
Width 
The width is calculated automatically according to the text length. If there are multiple lines of text, the longest line will be a basis.
 
- The below descriptions are true when the fittocontents property of the ImageViewer is set to "height".   The size of the image can vary according to the value of the stretch property after the height of the ImageViewer is determined.
 
ImageViewer Component
Height 
The bigger one is applied between the heights of the imagearea control and imagetext control.
 
imagearea Control
Top 
The value of the top property is 0.
Height 
The height of the original image is applied.
 
imagetext Control
Height 
The height is calculated automatically according to the text size. If there are multiple lines of text, the height of the control will represent the total height of all lines.
 
- The below descriptions are true when the fittocontents property of the ImageViewer is set to "both".
 
ImageViewer Component
Width 
The bigger one is applied between the widths of the imagearea control and imagetext control.
Height 
The bigger one is applied between the heights of the imagearea control and imagetext control.
 
imagearea Control
Left 
The value of the left property is 0.
Top 
The value of the top property is 0.
Width 
The width of the original image is applied.
Height 
The height of the original image is applied.
 
imagetext Control
Width 
The width is calculated automatically according to the text length. If there are multiple lines of text, the longest line will be a basis.
Height 
The height is calculated automatically according to the text size. If there are multiple lines of text, the height of the control will represent the total height of all lines.

Constructor 
ImageViewer.init( strName, vLeft, vTop , vWidth, vHeight [, vRight, vBottom, [vMinWidth, [vMaxWidth, [vMinHeight, [vMaxHeight]]]]] )
Parameters Type Description
strName String 
It sets the ID of a ImageViewer in the form of a string.
vLeft String, Number 
It specifies the value of the left property of a ImageViewer in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the right property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vTop String, Number 
It specifies the value of the top property of a ImageViewer in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the bottom property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vWidth String, Number 
It specifies the value of the width property of a ImageViewer in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vHeight String, Number 
It specifies the value of the height property of a ImageViewer in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vRight String, Number 
It specifies the value of the right property of a ImageViewer in pixel or percent (%).

This parameter will be ignored if both the vLeft and vWidth parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the left property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vBottom String, Number 
It specifies the value of the bottom property of a ImageViewer in pixel or percent (%).

This parameter will be ignored if both the vTop and vHeight parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the top property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vMinWidth String, Number 
It specifies a ImageViewer's minimum width that can be displayed in the Form.
vMaxWidth String, Number 
It specifies a ImageViewer's maximum width that can be displayed in the Form.

If this value is smaller than vMinWidth, it will be replaced by the value of vMinWidth.
vMinHeight String, Number 
It specifies a ImageViewer's minimum height that can be displayed in the Form.
vMaxHeight String, Number 
It specifies a ImageViewer's maximum height that can be displayed in the Form.

If this value is smaller than vMinHeight, it will be replaced by the value of vMinHeight.
Sample Call:
var objComp = new ImageViewer();

objComp.init( "ImageViewer00", 30, 120, 196, 46 );
objComp.init( "ImageViewer00", 30, 120, 196, 46, null, null );
objComp.init( "ImageViewer00", null, null, 300, "400px", "80%", 300 );
objComp.init( "ImageViewer00", 0, 0, 200, 100, null, null, 300, 500, 200, 500 );
objComp.init( "ImageViewer00", "ImageViewer22:10", 300, null, null, "ImageViewer33:10", "20%", 300, 500, 200, 500 );
 
// Create Object  
var objImageViewer = new ImageViewer();  
objImageViewer.init("ImageViewer", 30, 120, 196, 46, null, null);
objImageViewer.init("ImageViewer", null, null, 300, "400px", "80%", 300);
or
var objImageViewer = new ImageViewer("ImageViewer", 30, 120, 196, 46, null, null);
var objImageViewer = new ImageViewer("ImageViewer", null, null, 300, "400px", "80%", 300);
 
// Add Object to Parent Form  
this.addChild("ImageViewer", objImageViewer); 
 
// Insert Object to Parent Form  
this.insertChild(1, "ImageViewer", objImageViewer); 
 
// Remove Object form Parent Form  
this.removeChild("ImageViewer"); 
 
// Show Object  
objImageViewer.show(); 
 
// Destroy Object  
objImageViewer.destroy(); 
objImageViewer = null;
