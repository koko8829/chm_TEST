Description 
Mobile device's contact data
Property
Name Description
addresses This property is used to set a list of addresses in a ContactAddress object. The list takes the form of an array.
birthday This property is used to set the data on the birthday of a contact.
categories This property is used to set a list of categories in a ContactField object. The list takes the form of an array.
contactname This read-only property includes sub-properties that hold the name of a contact.
emails This property is used to set a list of email addresses in a ContactField object. The list takes the form of an array.
id This property is used to set the identification of a Contact.
ims This property is used to set the data on instant messengers in a ContactIM object. The list takes the form of an array.
name This property is used to specify the name of a Contact.
nickname This property is used to set the nickname of a contact.
note This property is used to set the data on the memo of a contact.
organizations This property is used to set a list of affiliated organizations in a ContactOrganization object. The list takes the form of an array.
phonenumbers This property is used to set a list of phone numbers in a ContactField object. The list takes the form of an array.
photos This property is used to set a list of photos in a ContactPhoto object. The list takes the form of an array.
uniqueid This read-only property holds the ID of a contact.
urls This property is used to set a list of web page URLs in a ContactField object. The list takes the form of an array.
