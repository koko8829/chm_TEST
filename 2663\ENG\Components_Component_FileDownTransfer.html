FileDownTransfer
Supported Environments

Edge Chrome Safari Firefox Opera
[Mobile]
Nexacro Runtime Environment (NRE)
Android iOS/iPadOS
Web Runtime Environment (WRE)
Android Default Web Browser iOS/iPadOS Default Web Browser
Description 
This is an invisible object designed to download files from the server.
Remark 
- Files are downloaded from the server to a designated location in the "POST" manner by using the HTTP/HTTPS protocol.

- The onsuccess and onerror events do not occur in web browsers.
Constructor 
FileDownTransfer( strName, objParent )
 
// Create Object  
var objFileDown = new nexacro.FileDownTransfer( "FileDownTransfer00", this );  

// Add Object to Parent Form  
this.addChild( "FileDownTransfer00", objFileDown );

// Set Object 
objFileDown.url = "http://localhost/Test.txt";
objFileDown.setEventHandler( "onsuccess", this.FileDownTransfer00_onsuccess, this);

// Remove Object form Parent
var objFileDown = this.removeChild( "FileDownTransfer00" );
 
// Destroy Object
objFileDown.destroy();
objFileDown = null;
Property
Name Description
downloadfilename This property is used to define a file name that will appear by default in a FileDialog opened for file downloading.
id This property is used to set the identification of a FileDownTransfer.
name This property is used to specify the name of a FileDownTransfer.
postdatalist This read-only property holds the data that will be sent to the server. The data, which takes the form of a collection, is used to request file download.
url In a FileDownTransfer, this property is used to designate the name of a file to download and its location in the server.
Method
Name Description
addEvent This method is used to declare a certain event and then add it to a FileDownTransfer.
addEventHandler This method is used to add a handler to a certain event of a FileDownTransfer.
addEventHandlerLookup This method is used to find a function and add it to a certain event of a FileDownTransfer as a handler.
clearEventHandler This method is used to delete all the event handlers registered in a certain event.
clearPostDataList This method is used to remove all the entries of the data set by the postdatalist property.
destroy This method is used to delete a FileDownTransfer that is dynamically created through a script.
download This method is used to download a certain file from the server in the "POST" manner by using the HTTP/HTTPS protocol.
findEventHandler This method finds a handler in a certain event of a FileDownTransfer and returns the index of the handler.
getEventHandler This method returns a specific handler from a certain event of a FileDownTransfer by specifying the index number of the handler.
getPostData This method returns the entry of a certain key from the data set by the postdatalist property.
insertEventHandler This method is used to add a handler to a certain event of a FileDownTransfer at a specified position.
removeEvent This method is used to remove a certain event that is declared in a FileDownTransfer.
removeEventHandler This method is used to remove a handler in a certain event of a FileDownTransfer.
removeEventHandlerLookup This method is used to find a handler in a certain event of a FileDownTransfer and remove it.
removePostData This method is used to remove the entry of a certain key from the data set by the postdatalist property.
setEventHandler This method is used to change the first handler in a certain event of a FileDownTransfer.
setEventHandlerLookup This method finds a function and uses it to replace the first handler in a certain event of a FileDownTransfer.
setPostData This method is used to add data to the postdatalist property, whose data will be sent to the server to request file download.
Event
Name Description
onerror This event occurs when file download fails or an error code is returned.
onsuccess This event occurs when a success code is delivered after the downloading of files.
