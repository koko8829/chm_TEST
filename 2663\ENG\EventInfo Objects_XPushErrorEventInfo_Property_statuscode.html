Description 
This property holds the code of an error that occurred.
Syntax 
XPushErrorEventInfo.statuscode
Setting Syntax
-100 
An XPush server has sent "Byec" or been terminated according to the protocol.
-101 
Communications have been timed out.
-200 
Command Auth Error. An authentication error has occurred to the current IP.
-201 
Access has failed to all the XPush servers registered in the iplist property.
-202 
Authentication error for a data transfer channel.
-300 
Access to a server has failed.
-301 
Packet transmission (send/recv) error has occurred.
-401 
Request for the disconnection from an XPush server has failed.
-501 
Request for suspending messages from an XPush server has failed.
-601 
Request for resuming the message reception from an XPush server has failed.
-701 
Request for registering/removing a topic in/from an XPush server has failed.
-801 
Information on the version of a message format is not matched.
-901 
Response to a received reliable message has failed.
-902 
Request for unreceived messages has failed.
-1001 
It has been failed to request the registration of the information on a mobile device token in an XPush server.
-1002 
Request for registering a user topic in an XPush server (registerUser) has failed.
-1003 
Request for removing a user topic from an XPush server (unregisterUser) has failed.
-1004 
Request for the information on unreceived messages has failed.
-1005 
Topic information request failed.
-1072 
It suggests that you have attempted to register (RGST) the same device as an existing device.
-1092 
It suggests that you have attempted to register (ADUI) the same message topic as an existing topic.
