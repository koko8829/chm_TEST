Property Type
Enum Expr Control Hidden ReadOnly Bind Collection StringResource
Description 
This property is used to determine the character types permitted as an input to a Grid cell whose edittype property is set to "text".
Syntax 
GridCellControl.editinputtype[= strInputtype]
Setting Syntax 
strInputtype ::= 'normal' | <type>

<type> ::= <inputtype> | <inputtype> ',' <type>
<inputtype> ::= 'alpha' | 'english' | 'digit' | 'number' | 'numberandenglish' | 'comma' | 'dot' | 'sign' | 'space' | 'symbol' | 'half' | 'full'

* You can specify multiple types by dividing them with commas (,).
 
this.Grid00.setCellProperty( "body", 0, "editinputtype", "english" );                         // English letters are permitted.
this.Grid00.setCellProperty( "body", 0, "editinputtype", "english,number" );             // English letters and numbers are permitted.
this.Grid00.setCellProperty( "body", 0, "editinputtype", "english,number,space" );   // English letters, numbers and white spaces are permitted.
"normal" 
It permits all types of characters.
"alpha" 
You can enter lower-case letters (a-z) and upper-case letters (A-Z).
"english" 
You can enter lower-case letters (a-z) and upper-case letters (A-Z).
"digit" 
You can enter numbers (0-9).
"number" 
You can enter numbers (0-9), periods (.), commas (,) and minuses (-).
"numberandenglish" 
You can enter lower-case letters (a-z), upper-case letters (A-Z), numbers (0-9), periods (.), commas (,) and minuses (-).

Setting this value is the same as setting "number" and "english" together.
"comma" 
You can enter commas (,).
"dot" 
You can enter periods (.).
"sign" 
You can enter pluses (+) and minuses (-).
"space" 
You can enter white spaces (" ").
"symbol" 
You can enter the following symbols: !,",#,$,%,@,&,',*,/,\,:,;,^,`,{,},[,],(,),|,~,=,_,<,>,?

You can enter all symbols except for the ones defined by the "comma", "dot", "sign" and "space" values.
"half" 
You can enter half-width characters.
"full" 
You can enter full-width characters.
Remark 
- The default value of this property is "normal".

- This property is applied only when you enter a value by yourself.
  If a value is entered through a script, this property will not be applied to the value.

- You can enter only the character types set by this property. You cannot enter other types.

- If this property does not include "full" as its value, you can enter only half-width characters.

-  Properties that restrict input are applied in order of editinputmode, editinputfilter and editinputtype.


◆ Restrictions in the mobile environment

- If this property is set in a way that allows only numerical input (eg. "digit", "number", "tel", "dot"), the tel type and number type keypads will be applied by default to Android and iOS/iPadOS respectively.
 However, the text type keyboard will appear in iOS/iPadOS if the value of an sub cell contains a character other than a number.

- If this property is set in a way that allows alphabetical input (eg. "numberandenglish" or "english"), the text type keyboard will appear both in Android and in iOS/iPadOS.
