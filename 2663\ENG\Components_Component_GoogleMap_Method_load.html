Description 
This method is used to load a map on a GoogleMap.
Syntax 
GoogleMap.load( [bUseSensor[, nLatitude, nLongitude[, constViewMode[, nZoomLevel]]] )
Parameters
Parameters Type Description
bUseSensor Boolean 
This parameter determines whether to use the GPS information.

For "true", a loaded map will be determined based on the GPS information.
For "false", GPS information will not be used.

The default value is "false".

* If this parameter is set to "true" with the mobile device not permitting the use of GPS information, the onerror event will occur.
   If this parameter is set to "true" with the mobile device permitting the use of GPS information, a dialog for permission may appear once or more.
nLatitude Number 
This is a number representing the latitude of a loaded map.

The default value is 0.
nLongitude Number 
This is a number representing the longitude of a loaded map.

The default value is 0.
constViewMode Number 
This parameter is used to set the map type.

For "0", the GoogleMap displays a normal road map.
For "1", the GoogleMap displays a satellite image.
For "2", the GoogleMap displays a satellite image with a transparent layer of major roads.
For "3", the GoogleMap displays terrain information with a transparent layer of major roads. Terrain information includes topography and vegetation.

The default value is 0.
nZoomLevel Number 
This is an integer larger than 0, representing the zoom level of a loaded map.
Return
Type Description
Boolean 
If parameters are invalid, this method will return "false".

Otherwise, this method will return "true", indicating the loading has been successful.
Remark 
- If the loading of a map is successful, the onload event will occur.
Otherwise, the onerror event will occur.

- If this method is called to a GoogleMap already loaded with a map, this method will reload a map.

- After the loading of a map, the centerlocation, viewmode and zoomlevel properties are set automatically.

- The view mode and zoom level vary according to locations. If you set values that are not allowed in a certain location, the map may not show properly.
 Click on the below link to learn the maximum zoom levels of a GoogleMap, allowed by each location.
"https://google-developers.appspot.com/maps/documentation/javascript/examples/control-positioning?hl=ko"

- To use a GoogleMap in the Android NRE, you need to insert the below definition to the build.gradle file.
  implementation 'com.google.android.gms:play-services-maps:16.0.0'
  implementation 'com.google.android.gms:play-services-location:16.0.0'
