Property Type
Enum Expr Control Hidden ReadOnly Bind Collection StringResource
Description 
This read-only property holds the detailed information on an event-firing cell.
Syntax 
GridClickEventInfo.clickitem
Setting Syntax
"control" 
This property holds this value when you click a cell whose displaytype property is set to one of the following values: "buttoncontrol", "calendarcontrol", "checkboxcontrol", "combocontrol", "editcontrol", "maskeditcontrol", "progressbarcontrol", radioitemcontrol" and "textareacontrol".
"image" 
This property holds this value when you click a cell whose displaytype property is set to "imgaecontrol".
"treeitembutton" 
This property holds this value when you click the Collapse/Expand button of a cell whose displaytype property is set to "treeitemcontrol".
"treeitemcheckbox" 
This property holds this value when you click the checkbox of a cell whose displaytype property is set to "treeitemcontrol".
"expanddown" 
This property holds this value when you click the Collapse/Expand button that displayed on the right-hand side of a cell.
Remark 
- This property will have an empty string as its value if the cause of the event is not related to the value "control", "cellimage", "treeitembutton", "treeitemcheckbox" or "expanddown".

- This property does not have a value at the time of event occurrence to guarantee the performance of the application.
  This property fetches a value when the property is accessed in a script. Therefore, this property always seems to have an empty value in the Watch Tab of a debugging program.
  Since frequent access to this property can cause a slowdown, you are recommended to use this property by allocating it to a variable.
