Description 
This property is used to set the shape of the corners of a TreeItemIconControl.
Syntax 
TreeItemIconControl.set_border-radius-control() // set
TreeItemIconControl.border-radius-control // get
Setting Syntax 
border-radius ::= <horizontal-radius>{4} ['/' <vertical-radius>{4}]
 
* XCSS
border-radius : 10px;
border-radius : 5px 6px 10px / 5px 10px;
<horizontal-radius> 
Widths of rounded corners.

A maximum of four values can be entered in pixel, divided by spaces.

One value: the value applies to all four corners.
Two values: first value applies to top-left and bottom-right corner, and the second value applies to top-right and bottom-left corner.
Three values: first value applies to top-left and top-right, second value applies to bottom-left, and third value applies to bottom-right corner.
Four values: first value applies to top-left, second value applies to top-right, third value applies to bottom-right, and fourth value applies to bottom-left corner.
<vertical-radius> 
Heights of rounded corners.

A maximum of 4 values can be entered in pixel, divided by spaces.

One value: the value applies to all four corners.
Two values: first value applies to top-left and bottom-right corner, and the second value applies to top-right and bottom-left corner.
Three values: first value applies to top-left and top-right, second value applies to bottom-left, and third value applies to bottom-right corner.
Four values: first value applies to top-left, second value applies to top-right, third value applies to bottom-right, and fourth value applies to bottom-left corner.
Remark 
- If you do not set this property, the property's value will be undefined with the property working as if its value is 0px.

- The <horizontal-radius> value and <vertical-radius> value are divided by a slash (/).
