Property Type
Enum Expr Control Hidden ReadOnly Bind Collection StringResource
Description 
This property is used to set a language applied to a GoogleMap.
Syntax 
GoogleMap.language[= strLang]
Setting Syntax 
this.GoogleMap00.language = "ko";
strLang 
This is a string that represents the code of a language applied to the GoogleMap.
Remark 
- If you do not set this property, the language set by the browser will be applied.
