Description 
The ListBox is a component used to select one or multiple values among several candidate values.

The ListBox is also used to output multiple selected values.
The data of the ListBox is used through the InnerBind Dataset.
Structure
Contents Sizing 
The below table describes how to size a ListBox.
NULL 
ListBox Component  * An hscrollbar control will appear if the width of the ListBox is smaller than the longest width among listboxitem controls. * A vscrollbar control will appear if the height of the ListBox is smaller than the total height of all listboxitem controls.
Width 
The value of the width property of the ListBox is applied.
Height 
The value of the height property of the ListBox is applied.
NULL 
listboxitem Control
Left 
The value of the left property is 0.
Top 
The value of the top property is calculated by factoring in the number of listboxitem controls and the height of the listboxitem control.
Width 
The width adopts the longest text among the items displayed in the listboxitem control.
Height 
The value of the itemheight property of the ListBox is applied. If itemheight is not set, the height will be calculated according to the text size.
Basic Key Action
Action Key Description
Move focus Tab 
The focus moves to the next component according to the tab movement order.
Move between items Up/Down arrow key 
Focus moves between items. 
An item will be selected automatically if it receives focus. 

If you press the up or down arrow key in the first or last item respectively, focus will move to the last or first item respectively.
Scroll ListBox Ctrl + arrow keys 
Scrolls to the direction of the arrow key entered when vscrollbar is enabled.
Scroll ListBox (page) PageUp/PageDown 
Scrolls to page units (number of items displayed on the screen) when vscrollbar is enabled.
The item in the moved position is automatically selected.
Select multiple items Shift + Up/Down arrow key 
Multiple items can be selected when the multiselect property is set to "true".
Entering an up or down arrow key with the shift key pressed selects the item at the starting point plus the above or below item consecutively.
Accessibility Key Action 
This is a key action to be applied when the accessibility function is activated.
Action Key Description
Move focus Tab 
The focus moves to the next component according to the tab movement order.
Move selection box Up/Down arrow key 
A selection box moves between items. 
The item in the moved position is not automatically selected.
It must press the space or enter key to select the item in the moved position.

If you press the up or down arrow key in the first or last item respectively, a selection box will move to the previous or next component according to the tab order respectively.
Determine selection Space 
Pressing the space bar determines the selection of an item and apply it to the value property.
Scroll ListBox Ctrl + arrow keys 
Scrolls to the direction of the arrow key entered when vscrollbar is enabled.
Scroll ListBox (page) PageUp/PageDown 
Scrolls to page units (number of items displayed on the screen) when vscrollbar is enabled.
The item in the moved position is not automatically selected.
It must press the space or enter key to select the item in the moved position.
Constructor 
ListBox.init( strName, vLeft, vTop , vWidth, vHeight [, vRight, vBottom, [vMinWidth, [vMaxWidth, [vMinHeight, [vMaxHeight]]]]] )
Parameters Type Description
strName String 
It sets the ID of a ListBox in the form of a string.
vLeft String, Number 
It specifies the value of the left property of a ListBox in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the right property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vTop String, Number 
It specifies the value of the top property of a ListBox in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the bottom property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vWidth String, Number 
It specifies the value of the width property of a ListBox in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vHeight String, Number 
It specifies the value of the height property of a ListBox in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vRight String, Number 
It specifies the value of the right property of a ListBox in pixel or percent (%).

This parameter will be ignored if both the vLeft and vWidth parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the left property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vBottom String, Number 
It specifies the value of the bottom property of a ListBox in pixel or percent (%).

This parameter will be ignored if both the vTop and vHeight parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the top property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vMinWidth String, Number 
It specifies a ListBox's minimum width that can be displayed in the Form.
vMaxWidth String, Number 
It specifies a ListBox's maximum width that can be displayed in the Form.

If this value is smaller than vMinWidth, it will be replaced by the value of vMinWidth.
vMinHeight String, Number 
It specifies a ListBox's minimum height that can be displayed in the Form.
vMaxHeight String, Number 
It specifies a ListBox's maximum height that can be displayed in the Form.

If this value is smaller than vMinHeight, it will be replaced by the value of vMinHeight.
Sample Call:
var objComp = new ListBox();

objComp.init( "ListBox00", 30, 120, 196, 46 );
objComp.init( "ListBox00", 30, 120, 196, 46, null, null );
objComp.init( "ListBox00", null, null, 300, "400px", "80%", 300 );
objComp.init( "ListBox00", 0, 0, 200, 100, null, null, 300, 500, 200, 500 );
objComp.init( "ListBox00", "ListBox22:10", 300, null, null, "ListBox33:10", "20%", 300, 500, 200, 500 );
 
// Create Object  
var objListBox = new ListBox();  
objListBox.init("ListBox", 30, 120, 196, 46, null, null);
objListBox.init("ListBox", null, null, 300, "400px", "80%", 300);
or
var objListBox = new ListBox("ListBox", 30, 120, 196, 46, null, null);
var objListBox = new ListBox("ListBox", null, null, 300, "400px", "80%", 300);
 
// Add Object to Parent Form  
this.addChild("ListBox", objListBox); 
 
// Insert Object to Parent Form  
this.insertChild(1, "ListBox", objListBox); 
 
// Remove Object form Parent Form  
this.removeChild("ListBox"); 
 
// Show Object  
objListBox.show(); 
 
// Destroy Object  
objListBox.destroy(); 
objListBox = null;
