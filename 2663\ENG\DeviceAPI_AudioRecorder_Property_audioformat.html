Property Type

Description 
This property is used to define the extension of a file to which recorded audio data is saved in a mobile device.
Setting Syntax 
enumFormat ::= 'WAV' | 'OGG' | 'MP3' | 'AMR_NB' | 'AMR_WB'
 
this.AudioRecoder00.audioformat = "WAV";
this.AudioRecoder00.audioformat = "MP3";
this.AudioRecoder00.audioformat = "OGG";
this.AudioRecoder00.audioformat = "AMR_NB";
this.AudioRecoder00.audioformat = "AMR_WB";
enumFormat 
This is the format of recorded audio data.

You can select one of the following values: "WAV", "OGG", "MP3", "AMR_NB" and "AMR_WB".
Remark 
- The default value of this property is "MP3" in the Android NRE and "WAV" in the iOS/iPadOS.
