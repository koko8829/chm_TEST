Property Type
Enum Expr Control Hidden ReadOnly Bind Collection StringResource
Description 
This read-only property holds an integer indicating the cause of an error in a BluetoothLE object.
Syntax 
BluetoothLEErrorEventInfo.reason
Setting Syntax 
var nReason = e.reason;
10 
This value suggests that the error occurred when a scan was initiated by the scanStart() method.
11 
This value suggests that the error occurred when a scan finished due to the timeout set by the scanStart() method.
12 
This value suggests that the error occurred when the device that provides a certain service was found by the calling of the scanStart() method.
20 
This value suggests that the error occurred when a scan was stopped by the scanStop() method.
30 
This value suggests that the error occurred when a device was connected by the connect() method.
40 
This value suggests that the error occurred when a device was disconnected by the disconnect() method.
50 
This value suggests that the error occurred when a request was made by the discoverService() method to obtain the list of services provided by the device.
51 
This value suggests that the error occurred when a service list requested by the discoverService() method was received. The list is provided by the device.
60 
This value suggests that the error occurred when a notification was requested by the subscribe() method.
61 
This value suggests that the error occurred when a notification requested by the subscribe() method was received.
70 
This value suggests that the error occurred when notification was cancelled by the unsubscribe() method.
80 
This value suggests that the error occurred when a characteristic of a certain service was requested by the readCharacteristic() method.
90 
This value suggests that the error occurred when a characteristics of a certain service was saved by the writeCharacteristic() method.
