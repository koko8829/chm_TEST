IconControl
Supported Environments
[Desktop]
Nexacro Runtime Environment (NRE)
Windows macOS
Web Runtime Environment (WRE)
Edge Chrome Safari Firefox Opera
[Mobile]
Nexacro Runtime Environment (NRE)
Android iOS/iPadOS
Web Runtime Environment (WRE)
Android Default Web Browser iOS/iPadOS Default Web Browser
Description 
This is a control object that can display an icon.
Remark 
- This object is used as a control that composes a composite component.
Property
Name Description
background The attribute to set the background area of the IconControl.
border-radius This property is used to set the shape of the corners of a IconControl.
bottom This property is used to specify the value of the point that indicates the position of a IconControl's bottom side.
cursor This property is used to set the type of a mouse pointer that appears when the mouse cursor is located in a IconControl.
enable This property is used to determine whether to enable a IconControl.
enableevent This property is used to determine whether to trigger events in a IconControl.
height This property is used to specify the height of a IconControl.
left This property is used to specify the value of the point that indicates the position of a IconControl's left side.
name This property is used to specify the name of a IconControl.
-nexa-border This property is used to set the border of a IconControl.
-nexa-edge This property is used to set the image of the edges that are displayed inside the border of an IconControl.
-nexa-icon This property is used to set the image of an icon displayed in an IconControl object.
-nexa-padding This property is used to set the padding, an inner margin, of a IconControl.
-nexa-rtl-background-image This property is used to set an image that will be displayed as the background of a IconControl. This property is applicable when the rtl property of the IconControl is set to "true".
-nexa-rtl-edge-image This property is used to set an edge image that is displayed inside the border of a IconControl. This property is applicable when the rtl property of the IconControl is set to "true".
-nexa-rtl-icon This property is used to set an icon image when the rtl property of a IconControl is set to "true".
-nexa-text-align This property is used to set the horizontal alignment of the text or contents displayed in a IconControl.
-nexa-vertical-align This property is used to set the vertical alignment of the text or contents displayed in a IconControl.
opacity This property is used to set the opacity of a IconControl.
right This property is used to specify the value of the point that indicates the position of a IconControl's right side.
top This property is used to specify the value of the point that indicates the position of a IconControl's top side.
width This property is used to specify the width of a IconControl.
Status

disabled status 
This is the status in which a component is disabled.

This is the status in which a mouse cursor hovers over a component or a certain item of the component.
enabled status 
This is the status in which a component is enabled.
