Property Type
Enum Expr Control Hidden ReadOnly Bind Collection StringResource
Description 
The attribute to set the condition to move from the final or the first component to a different frame within the form when moving the focus by entering tap key or shift+tap key.
Syntax 
FrameSet.tabkeycirculation[= enumTabkeyCirculation]
Setting Syntax 
enumTabkeyCirculation ::= 'all' | 'all, no_minimize' | 'active_only' | 'auto'
 
this.FrameSet00.tabkeycirculation = "auto";
"all" 
Circulates the entire Frame inside the FrameSet (Default)
"all,no_minimize" 
Circulates the Frame except the minimized window.
"active_only" 
Circulates only the window in an active state.
"auto" 
The action defers according to the parameter when executing the arrange method.
- When the parameter is "cascade": Operates as "active_only".
- When the parameter is not "cascade": Operates as "all".
Remark 
- It is applied as "all" if the tabkeycirculation attribute value is not set.
- The focus does not move between frames when the tabkeycirculation attribute value is set as “form,cycle” or “form,nocycle” in the Environment.
