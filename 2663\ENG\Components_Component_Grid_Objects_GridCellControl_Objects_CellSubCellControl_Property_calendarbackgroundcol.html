Property Type

Description 
The attribute to set the column of the dataset object set in the calendardataset attribute to display a background color on a specific date on the calendar with a calendar-type SubCell.
Syntax 
CellSubCellControl.calendarbackgroundcol[= strColumnID]
Setting Syntax 
this.Grid00.setSubCellProperty( "body", 0, 0, "calendarbackgroundcol", "column0");
strColumnID 
Set the ID of the Column with the background color defined in strings.
Remark 
- Only the column of the Dataset object in the calendardataset attribute can be set.
- The background color set in the calendarbackgroundcol on the specific date set in the calendardatecol is applied.
