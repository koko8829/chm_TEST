textareascrollbartype
Supported Environments

Property Type
Enum Expr Control Hidden ReadOnly Bind Collection StringResource
Description 
This property determines how scrollbars work in the editable field when the edittype property of a Grid sub-cell is set to"textarea".
Syntax 
CellSubCellControl.textareascrollbartype[= strBartype]
Setting Syntax 
strBartype ::= <scrolltype> | <pairtype>

<scrolltype> ::= <desktype> | <mobiletype>
<pairtype> ::= <desktype> ' ' <desktype> | <mobiletype> ' ' <mobiletype>

<desktype> ::= 'default' | 'none' | 'fixed' | 'auto'
<mobiletype> ::= 'default' | 'none' | 'indicator' | 'autoindicator'
 
this.Grid00.setSubCellProperty( "body", 0, 0, "textareascrollbartype", "none" );
this.Grid00.setSubCellProperty( "body", 0, 0, "textareascrollbartype", "default autoindicator" );
this.Grid00.setSubCellProperty( "body", 0, 0, "textareascrollbartype", "auto fixed" );
this.Grid00.setSubCellProperty( "body", 0, 0, "textareascrollbartype", "indicator none" );
<scrolltype> 
If you specify just one value for the textareascrollbartype property, the value will apply to both horizontal and vertical scrollbars.
<pairtype> 
You can specify two values for textareascrollbartype by dividing them with a whitespace.

The first and second values will apply to the horizontal and vertical scrollbars respectively.
"default" 
The "auto" option will apply in the desktop computer environment.

The "autoindicator" option will apply in the mobile environment.
"none" 
Scrollbars or indicator bars will not be displayed.
"fixed" 
Scrollbars are always displayed.
"auto" 
Scrollbars will be displayed automatically only when needed.
"indicator" 
Scroll indicator bars are displayed automatically when necessary.
"autoindicator" 
Scroll indicator bars are displayed only in the event of scrolling.
Remark 
- The "default" option will apply if this property is not specified.

- This property is applied only when the sub cell is editable.
 Scrollbars will not be displayed necessarily even if the displaytype property is set to "textareacontrol".

- You can apply a desktop-only option and mobile-only option simultaneously. However, it is not recommendable.
   ex) this.Grid00.setSubCellProperty( "body", 0, "textareascrollbartype", "fixed autoindicator" );
