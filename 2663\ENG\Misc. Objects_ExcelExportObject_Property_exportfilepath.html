Description 
This property is used to set an absolute path where an exported file is created.
Setting Syntax 
this.ExcelExportObject00.exportfilepath = "C:\\";
this.ExcelExportObject00.exportfilepath = "C:\\Users\\[UserID]\\Downloads";
this.ExcelExportObject00.exportfilepath = "%MYDOCUMENT%";
strFilePath 
This is a string that represents an absolute path where an exported file is created. You can also use an alias instead of an absolute path.

You can specify only a path to which the access is permitted by the filesecurelevel property of the Environment object.
Note that a file will not be created if the access to the path is not permitted or the path does not exist.
Remark 
- If this property is set, an Excel file will be exported to the specified path without a path-selection dialog popping up, with the file named after the value of the exportfilename property.

- Access will be allowed to every path if the filesecurelevel property of the Environment object, which is a sandbox option, is set to"all".
 In this case, a file will be created only when the user approves the access after he or she is asked for the permission.
