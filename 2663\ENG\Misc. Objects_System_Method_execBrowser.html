execBrowser
Supported Environments
[Desktop]
Nexacro Runtime Environment (NRE)
Windows macOS
Web Runtime Environment (WRE)
Edge Chrome Safari Firefox Opera
[Mobile]
Nexacro Runtime Environment (NRE)
Android iOS/iPadOS
Web Runtime Environment (WRE)
Android Default Web Browser iOS/iPadOS Default Web Browser
Description 
This is a method of launching the browser (or app) according to the delivered string information.
Syntax 
system.execBrowser( strUrl )
Parameters
Parameters Type Description
strUrl String 
Set the URL path or information needed for executing dialing and mailing in strings.
Strings for executing files or apps can be set in Android NRE and iOS/iPadOS NRE. Refer to the Remark item.
Sample Call:
system.execBrowser("http://www.dummy.com");
system.execBrowser("mailto:test@dummy.com");
system.execBrowser("tel:2025550145");
Return
None

â€˜- In the Windows OS NRE, the Edge browser will be launched instead of the default browser set in the system.
   Internet Explorer may run on Windows 10 if the latest update has not been applied or on Windows 8.1 or lower.
   To run the default browser set on the system, use the execDefaultBrowser method.

- When a method is executed in WRE, it is executed in a new tab (or new window) in the web browser.

- In Android NRE and iOS/iPadOS NRE, you can use the string shown below to access specific files or run installed apps.
   What is supported by URL Scheme varies depending on the operating system and the connected app. Please refer to the documented guide on the operating system or applicable app. 
   Custom URL Scheme supported by certain apps will not work if the app is not installed.

  // Relative path
  system.execBrowser("/storage/emulated/0/Download/test.mp3");

  // Protocol (ftp, file, etc.)
  system.execBrowser("file:///storage/emulated/0/Download/test.mp3");

  // URL Scheme
  system.execBrowser("Intent://command?parameter=option&version=1");
  system.execBrowser("android-app://com.android.chrome"); 
  system.execBrowser( "fb://profile" );
  system.execBrowser( "twitter://user?screen_name=unity3d" );
