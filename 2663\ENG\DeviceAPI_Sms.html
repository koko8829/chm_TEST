Description 
As an object that supports Short Message Service (SMS), an Sms deals with the process of sending and receiving SMS messages.
Property
Name Description
id This property is used to set the identification of a Sms.
message This property is used to set a message that will be sent to a receiver through a mobile device.
name This property is used to specify the name of a Sms.
parent This read-only property holds the parent object of a Sms.
phonenumber This property is used to set the phone number of a person to whom a message will be sent through a mobile phone.
Method
Name Description
deleteMessage This method is used to delete a specified message from the SMS field in a mobile device.
readMessageList This method returns the list of SMSs saved in a mobile device.
recvStart This method is used to cause an event to occur when a mobile device receives an SMS.
recvStop This method is used to cancel the event occurrence set by the recvStart() method.
sendMessage This method is used to display a screen to write an SMS message in a mobile device.
Event
Name Description
cansendmessage This event occurs when a mobile device sends an SMS.
ondeletemessage This event occurs when an SMS saved in a mobile device is deleted.
onerror This event occurs when an SMS-related action fails to be performed in a mobile devices.
onreadmessagelist This event occurs when the list of SMSs saved in a mobile device is searched.
onrecvmessage This event occurs when a mobile device receives an SMS.
onsendmessage This event occurs after a mobile device sends an SMS.
