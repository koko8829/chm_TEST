Property Type
Enum Expr Control Hidden ReadOnly Bind Collection StringResource
Description 
This property is used to set the area where the internal content area of the Button is automatically resized to fit the entire content.
Syntax 
Button.fittocontents[= enumFitTo]
Setting Syntax 
enumFitTo ::= 'none' | 'width' | 'height' | 'both'
 
this.Button00.fittocontents = "width";
"none" 
It maintains the size as designed regardless of internal contents.
"width" 
It adjusts the width of the contents area of a Button for internal contents.
"height" 
It adjusts the height of the contents area of a Button for internal contents.
"both" 
It adjusts the height and width of the contents area of a Button for internal contents.
Remark 
- If the fittocontents property value is not set, it will be set as "none."

- The overall size of the Button is determined by adding the -nexa-border and -nexa-padding areas to the size of the internal content area.

- The size of the internal content area will be automatically adjusted only when the form is loaded and the fittocontents property value is changed.
    If the content of the Button is changed by a bound property or script, the size of the internal content area will not be automatically adjusted.

- When the content of the Button is changed by a bound property and the internal content area needs to be resized,
    the size must be readjusted by executing the resetScroll() method in the onbindingvaluechanged event of the Form.

- When the content of the Button is changed by a script and the internal content area needs to be resized,
    the size must be readjusted by executing the resetScroll() method after the script that changes the content is run.. 

- In the case of ImageViewer, if the image is loaded after the form has been loaded, the internal content area will not be automatically resized. 
    The resetScroll() method must be executed in the onload event of ImageViewer to adjust the size.

- When executing the resetScroll() method, the fittocontents property must be a value other than "none" for the internal content area to be resized.
