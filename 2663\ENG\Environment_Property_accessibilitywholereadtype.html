accessibilitywholereadtype

Property Type
Enum Expr Control Hidden ReadOnly Bind Collection StringResource
Description 
This property is used to determine whether to conduct the accessibility features of a whole Form when it is loaded.
Syntax 
Environment.accessibilitywholereadtype[= enumReadType]
Setting Syntax 
enumReadType ::= 'none' | 'load' | 'change' | 'load,change'
 
var objEnv = nexacro.getEnvironment();

objEnv.accessibilitywholereadhotkey = "load";
objEnv.accessibilitywholereadhotkey = "load,change";
"none" 
The accessibility features of a whole Form are not conducted automatically when it is loaded.

Even when a container component (Div,PopupDiv or TabPage) changes its linked Form, the entire accessibility features of the new Form will not be conducted automatically.

The accessibility features of a whole Form will not be conducted even if you press the hotkey specified as the value of the accessibilitywholereadhotkey property.
"load" 
The accessibility features of a whole Form are conducted automatically when it is loaded. Moreover, the Form will have focus.

Even when a container component (Div,PopupDiv or TabPage) changes its linked Form, the entire accessibility features of the new Form will not be conducted automatically.

The accessibility features of a whole Form will be conducted if you press the hotkey specified as the value of the accessibilitywholereadhotkey property.
"change" 
The accessibility features of a whole Form are not conducted automatically when it is loaded.

When a container component (Div,PopupDiv or TabPage) changes its linked Form, the entire accessibility features of the new Form will be conducted automatically.

The accessibility features of a whole Form will be conducted if you press the hotkey specified as the value of the accessibilitywholereadhotkey property.
"load,change" 
The accessibility features of a whole Form are conducted automatically when it is loaded. Moreover, the Form will have focus.

When a container component (Div,PopupDiv or TabPage) changes its linked Form, the entire accessibility features of the new Form will be conducted automatically.

The accessibility features of a whole Form will be conducted if you press the hotkey specified as the value of the accessibilitywholereadhotkey property.
Remark 
- The default value of this property is "none".

- This property is applicable only when the enableaccessibility property is set to "true" .

- If you want to cancel in the process of conducting the accessibility features of a whole Form, you need to follow the cancellation function of the screen reader program.
