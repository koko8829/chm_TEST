Description 
MultiLineTextField is a componentfor receiving multi-line string inputs from the user.
Remark 
- To receive inputs of single-line strings and check their validity, the TextField component must be used.
Structure
Contents Sizing 
This is the basis for determining the size of the MultiLineTextField component.
 
MultiLineTextField Component
Width 
The value of the width property of the MultiLineTextField component will be applied. 
If it is smaller than the width of the internal content, the internal content will only be partially displayed.
Height 
The value of the height property of the MultiLineTextField component will be applied. If it is smaller than the height of the internal content, the internal content will be truncated.
 
label Control
Width 
The automatically calculated width value of the label control text will be applied.
Height 
The automatically calculated height value of the label control text will be applied.
 
charcounttext Control
Left 
The width of the MultiLineTextField component minus the width of the charcounttext control will be applied.
Top 
The height of the MultiLineTextField component minus the height of the charcounttext control will be applied.
Width 
The automatically calculated width value of the charcounttext control text will be applied.
Height 
The automatically calculated height of the charcounttext control text will be applied.
 
helpertext Control
Left 
0 will be applied.
Top 
The height of the MultiLineTextField component minus the height of the helpertext control will be applied.
Width 
The width of the MultiLineTextField component minus the width of the charcounttext control will be applied.
Height 
The height of the MultiLineTextField component minus the height of the helpertext and charcounttext controls, whichever is greater, will be applied.
 
nulltext Control
Left 
0 will be applied.
Top 
0 will be applied.
Width 
The width of the MultiLineTextField component minus the sum of the widths of the prefixtext, suffixtext, leadingbutton, and trailingbutton controls will be applied.
Height 
The height of the MultiLineTextField component will be applied.
 
box control
Left 
0 will be applied.
Top 
The height of the MultiLineTextField component minus the height of the area reserved for the floating of the label control will be applied.
Width 
The width of the MultiLineTextField component will be applied.
Height 
The height of the MultiLineTextField component minus the height of the area reserved for the floating of the label control and the height of the helpertext control will be applied.
Basic Key Action
Action Key Description
Move focus Tab 
The focus will move to the next component according to the tab sequence.
Move caret Arrow keys 
When the caret is displayed, the caret will be moved in the direction of the arrow key entered.
Select text Shift + arrow keys 
When the caret is displayed, text will be selected in the direction of the arrow key entered.
Move scroll Ctrl + arrow keys 
When there is a scroll in the MultiLineTextField component, the scroll will move in the direction of the arrow key entered.
Constructor 
MultiLineTextField.init( strName, vLeft, vTop , vWidth, vHeight [, vRight, vBottom, [vMinWidth, [vMaxWidth, [vMinHeight, [vMaxHeight]]]]] )
Parameters Type Description
strName String 
It sets the ID of a MultiLineTextField in the form of a string.
vLeft String, Number 
It specifies the value of the left property of a MultiLineTextField in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the right property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vTop String, Number 
It specifies the value of the top property of a MultiLineTextField in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the bottom property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vWidth String, Number 
It specifies the value of the width property of a MultiLineTextField in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vHeight String, Number 
It specifies the value of the height property of a MultiLineTextField in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vRight String, Number 
It specifies the value of the right property of a MultiLineTextField in pixel or percent (%).

This parameter will be ignored if both the vLeft and vWidth parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the left property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vBottom String, Number 
It specifies the value of the bottom property of a MultiLineTextField in pixel or percent (%).

This parameter will be ignored if both the vTop and vHeight parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the top property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vMinWidth String, Number 
It specifies a MultiLineTextField's minimum width that can be displayed in the Form.
vMaxWidth String, Number 
It specifies a MultiLineTextField's maximum width that can be displayed in the Form.

If this value is smaller than vMinWidth, it will be replaced by the value of vMinWidth.
vMinHeight String, Number 
It specifies a MultiLineTextField's minimum height that can be displayed in the Form.
vMaxHeight String, Number 
It specifies a MultiLineTextField's maximum height that can be displayed in the Form.

If this value is smaller than vMinHeight, it will be replaced by the value of vMinHeight.

var objComp = new MultiLineTextField();

objComp.init( "MultiLineTextField00", 30, 120, 196, 46 );
objComp.init( "MultiLineTextField00", 30, 120, 196, 46, null, null );
objComp.init( "MultiLineTextField00", null, null, 300, "400px", "80%", 300 );
objComp.init( "MultiLineTextField00", 0, 0, 200, 100, null, null, 300, 500, 200, 500 );
objComp.init( "MultiLineTextField00", "MultiLineTextField22:10", 300, null, null, "MultiLineTextField33:10", "20%", 300, 500, 200, 500 );
 
// Create Object  
var objMultiLineTextField = new MultiLineTextField();  
objMultiLineTextField.init("MultiLineTextField", 30, 120, 196, 46, null, null);
objMultiLineTextField.init("MultiLineTextField", null, null, 300, "400px", "80%", 300);
or
var objMultiLineTextField = new MultiLineTextField("MultiLineTextField", 30, 120, 196, 46, null, null);
var objMultiLineTextField = new MultiLineTextField("MultiLineTextField", null, null, 300, "400px", "80%", 300);
 
// Add Object to Parent Form  
this.addChild("MultiLineTextField", objMultiLineTextField); 
 
// Insert Object to Parent Form  
this.insertChild(1, "MultiLineTextField", objMultiLineTextField); 
 
// Remove Object form Parent Form  
this.removeChild("MultiLineTextField"); 
 
// Show Object  
objMultiLineTextField.show(); 
 
// Destroy Object  
objMultiLineTextField.destroy(); 
objMultiLineTextField = null;
