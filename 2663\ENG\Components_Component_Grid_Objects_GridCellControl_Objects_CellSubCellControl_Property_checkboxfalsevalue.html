Description 
This property is used to set a string that causes a checkbox to be unchecked. The checkbox here is represented by a Grid sub-cell.
Syntax 
CellSubCellControl.checkboxfalsevalue[= strValue]
Setting Syntax 
this.Grid00.setSubCellProperty( "body", 0, 0, "checkboxfalsevalue", "F");
strValue 
This a string that causes the checkbox to be unchecked.

If the string matches the value of the sub-cell, the checkbox will be considered to have "false" as its value and thus be unchecked.
Remark 
- The default value of this property is "false" or "0".

- This property applies to the checkbox displayed in a Grid sub-cell whose edittype is set to "checkbox".
In addition, this property applies to the checkbox displayed in a Grid sub-cell whose displaytype is set to "checkboxcontrol".

- If you deselect the checkbox of a sub-cell, the value of this property will be saved to the bound Dataset column.

- If the checkboxtruevalue and checkboxfalsevalue properties have the same value, the former property will override the latter.
