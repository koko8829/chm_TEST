Description 
MultiCombo is a component used to select multiple items from a preset list of items.

Constructor 
MultiCombo.init( strName, vLeft, vTop , vWidth, vHeight [, vRight, vBottom, [vMinWidth, [vMaxWidth, [vMinHeight, [vMaxHeight]]]]] )
Parameters Type Description
strName String 
It sets the ID of a MultiCombo in the form of a string.
vLeft String, Number 
It specifies the value of the left property of a MultiCombo in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the right property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vTop String, Number 
It specifies the value of the top property of a MultiCombo in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the bottom property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vWidth String, Number 
It specifies the value of the width property of a MultiCombo in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vHeight String, Number 
It specifies the value of the height property of a MultiCombo in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vRight String, Number 
It specifies the value of the right property of a MultiCombo in pixel or percent (%).

This parameter will be ignored if both the vLeft and vWidth parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the left property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vBottom String, Number 
It specifies the value of the bottom property of a MultiCombo in pixel or percent (%).

This parameter will be ignored if both the vTop and vHeight parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the top property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vMinWidth String, Number 
It specifies a MultiCombo's minimum width that can be displayed in the Form.
vMaxWidth String, Number 
It specifies a MultiCombo's maximum width that can be displayed in the Form.

If this value is smaller than vMinWidth, it will be replaced by the value of vMinWidth.
vMinHeight String, Number 
It specifies a MultiCombo's minimum height that can be displayed in the Form.
vMaxHeight String, Number 
It specifies a MultiCombo's maximum height that can be displayed in the Form.

If this value is smaller than vMinHeight, it will be replaced by the value of vMinHeight.

oninput This event occurs when you attempt to enter a value to a MultiCombo.
onitemchanged This event occurs after the items in the MultiCombo component is changed.
onitemclick This event occurs when you click an item in the list.
onkeydown It is Event to occur when a Key of a keyboard is pressed while Focus is in MultiCombo.
onkeyup This event occurs when you release a key with a MultiCombo focused.
onkillfocus This event occurs when focus moves out of a MultiCombo.
onlbuttondown This event occurs when you press the left button of a mouse within the area of a MultiCombo.
onlbuttonup This event occurs when you release the left button of a mouse within the area of a MultiCombo.
onmouseenter This event occurs when a mouse pointer enters a MultiCombo.
onmouseleave This event occurs when a mouse pointer moves out of a MultiCombo.
onmousemove This event occurs when a mouse pointer moves within a MultiCombo.
onmousewheel This event occurs when you rotates your mouse wheel within a MultiCombo.
onmove This event occurs when the position of a MultiCombo changes.
onrbuttondown The event occurring when clicking with the right mouse button within the MultiCombo area.
onrbuttonup This event occurs when you release the right button of a mouse within the area of a MultiCombo.
onsetfocus It is Event to occur when Focus is set to MultiCombo.
onsize This event occurs when the size of a MultiCombo changes.
ontouchend This event occurs when you touch and release the screen within the area of a MultiCombo.
ontouchmove This event occurs when a touch point is moved along the touch surface within a MultiCombo.
ontouchstart This event occurs when a touch point is placed on the touch surface within a MultiCombo.
