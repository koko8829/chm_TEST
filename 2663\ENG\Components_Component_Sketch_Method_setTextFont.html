setTextFont
Supported Environments
[Desktop]
Nexacro Runtime Environment (NRE)
Windows macOS
Web Runtime Environment (WRE)
Edge Chrome Safari Firefox Opera
[Mobile]
Nexacro Runtime Environment (NRE)
Android iOS/iPadOS
Web Runtime Environment (WRE)
Android Default Web Browser iOS/iPadOS Default Web Browser
Description 
This method is used to set the font used in the editable field of a Sketch.
Syntax 
Sketch.setTextFont( strTextFont )


strTextFont ::= [<font-style>] [<font-weight>] <font-size> ['/'<line-height>] <font-family>

<font-style> ::= 'normal' | 'italic'
<font-weight> ::= 'normal' | 'bold' | '100' | '200' | '300' | '400' | '500' | '600' | '700' | '800' | '900'
<font-size> ::= <nSize> 'px'|'pt'
<line-height> ::= 'normal' | <nValue> | <nPixel>'px' | <nPercent>'%'
<font-family> ::= <font-name> [',' <font-name>]*
Parameters
Parameters Type Description
<font-style> String 
It sets the style of a font.

Nothing applies if you specify the font as "normal".
The text is shown in italics if you specify the font as "italic".
<font-weight> String 
It sets the weight of a font.

Nothing applies if you specify the font as "normal".
The text is shown in bold if you specify the font as "bold".

If you specify a number ranging from 100 to 900, the thickness of the font will be as much as specified.
If you want to specify a number, you must predefine a weight that matches the specified number so that the relevant font is ready to provide the relevant weight.
<font-size> String 
It sets the size of a font in pixel or point.

You must specify "px" or "pt" depending on the unit you intend.
<nSize> String 
It sets the size of a font in integer, with a point as a unit.

You cannot use a negative number.
<line-height> String 
It specifies line spacing if text is displayed in multiple lines.

You cannot specify a negative number.

If you specify "normal", this option will not have a particular value.
If you specify "normal", line spacing will vary according to web browsers.
If you specify "normal", the Nexacro Runtime Environment will multiply the font size by 1.15 (i.e. the value of <nValue> is 1.15).
<nValue> String 
It specifies a value that will be multiplied by a font size.

The result will be applied as the line spacing of the font.
<nPixel> String 
If specifies line spacing in pixel.

You cannot omit the unit "px".
<nPercent> String 
It specifies a percentage that will be multiplied by a font size.

The result will be applied as the line spacing of the font.
You cannot omit the unit "%".
<font-family> String 
You can designate a list of fonts to be applied to a Sketch.

You can designate multiple fonts by dividing them with commas (,).
Fonts will be applied under a fallback system.
If the system does not support the first font, for example, it tries the next font, and so on.
The system will apply its default font if it can support none of the designated fonts.
<font-name> String 
It designated a font to apply to a Sketch by name.
Sample Call:
this.Sketch00.setTextFont( "bold 12pt/30px arial, sans-serif" );
Return
None
Remark 
- If a font size is changed by this property, the line spacing can be adjusted depending on the value of <line-height>.

- In the case that you specify a number for the <font-weight> value, 400 is the same as normal, and 700 is the same as bold.
 If the font does not support a certain numerical value for its <font-weight>. the weight will be determined between "normal" or "bold" by judging whether the numerical input is close to 400 or 700.
