Description 
This object is used to display multiple paths in a GraphicsControl object.
Remark 
- You can create this object dynamically in a script while you cannot design this object in Nexacro Studio. 

- You can display both connected paths and unconnected paths.

- To display multiple paths, you need to set path data in a GraphicsPaths object. Otherwise, you need to add a GraphicsPath object.
Property
Name Description
fillstyle This property is used to color the inside area of a GraphicsPaths.
id This property is used to set the identification of a GraphicsPaths.
miterlimit This property sets a value used to determine whether the corner of a GraphicsPaths becomes a beveled join even if the strokejoin property is set to "miter".
opacity This property is used to set the opacity of a GraphicsPaths.
parent This read-only property holds the parent object of a GraphicsPaths.
strokecap This property is used to set the caps of the paths displayed in a GraphicsPaths.
strokejoin This property determines the shape of joins in a GraphicsPaths object.
strokepen This property is used to set the style and color of a path in a GraphicsPaths.
visible This property is used to determine whether to make a GraphicsPaths visible or not.
x This property is used to set the x-coordinate of a position where a GraphicsPaths is displayed. The coordinate is calculated based on the position of the parent object.
y This property is used to set the y-coordinate of a position where a GraphicsPaths is displayed. The coordinate is calculated based on the position of the parent object.
Method
Name Description
addChild This method is used to add a GraphicsPath to a GraphicsPaths.
clear This method is used to remove an object added to a GraphicsPaths completely from a memory.
closePath This method is used to draw a straight line between a beginning and ending points specified in a GraphicsPaths object.
destroy This method is used to completely remove a GraphicsPaths from a memory.
getBoundRect This method returns the area where a GraphicsPaths is displayed, including the border of the GraphicsPaths. The area is calculated based on the Graphics component. The border is set by the strokepen property.
getChildByIndex This method returns the GraphicsPath of a certain index number from a GraphicsPaths.
getFirstChild This method returns the first object among those that have been added to a GraphicsPaths.
getLastChild This method returns the last object among those that have been added to a GraphicsPaths.
getNextSibling This method returns the next object among those under the same parent as a GraphicsPaths.
getObjectByID This method searches the objects added to a GraphicsPaths and returns the object of a certain ID among them.
getObjects This method returns an array containing the objects added to a GraphicsPaths.
getOffsetBoundRect This method returns the area where a GraphicsPaths is displayed, including the border of the GraphicsPaths. The area is calculated based on the parent object. The border is set by the strokepen property.
getOffsetRect This method returns the area where a GraphicsPaths is displayed, with the area calculated based on the parent object.
getOffsetRegion This method returns the area where a GraphicsPaths is displayed, excluding the border of the GraphicsPaths. The area is calculated based on the parent object. The border is set by the strokepen property.
getPathData This method returns the data on the path set in a GraphicsPaths with the returned data in the form of a SVG string.
getPreviousSibling This method returns the previous object among those under the same parent as a GraphicsPaths.
getRect This method returns the area where a GraphicsPaths is displayed, with the area calculated based on the Graphics component.
getRegion This method returns the area where a GraphicsPaths is displayed, excluding the border of the GraphicsPaths. The area is calculated based on the Graphics component. The border is set by the strokepen property.
hasChild This method returns whether a GraphicsPaths has any GraphicsPath object.
indexOf This method finds the object of a certain ID among the objects added to a GraphicsPaths and then returns the index number of the found object.
insertChild This method is used to insert a GraphicPath object as a child to a certain position in a GraphicsPaths component.
isHitTest This method finds the position marked by the coordinates delivered as arguments and returns whether the position is included in the area of a GraphicsPaths.
removeChild This method is used to remove the GraphicsPath object of a certain ID from a GraphicsPaths.
rotate This method is used to rotate a GraphicsPaths.
scale This method is used to scale a GraphicsPaths up/down.
setClipRect This method is used to set coordinates for clipping a certain region of a GraphicsPaths.
setPathData This method is used to set a path in a GraphicsPaths by specifying the path data in the form of SVG sting.
setTransform The method to apply the transform-related effect consecutively based on the original location of the GraphicsPaths.
shear This method is used to shear a GraphicsPaths.
toString This method returns a string representing the type that constitutes a GraphicsPaths.
transform This method is used to apply transformational methods to a GraphicsPaths.
translate This method is used to move the position of a GraphicsPaths.
