Description 
This method is used to rotate a GraphicsText.
Syntax 
GraphicsText.rotate( nAngle [, nCenterX, nCenterY] );
Parameters
Parameters Type Description
nAngle Number 
This is a number indicating the angle at which the GraphicsText rotates.

If the value is a positive number, the GraphicsText will rotate clockwise.
If the value is a negative number, the GraphicsText will rotate counterclockwise.
nCenterX Number 
This is a number indicating an x-coordinate that serves as an axis of the rotation.
The value is calculated based on the parent object.

The default value is 0.
nCenterY Number 
This is a number indicating a y-coordinate that serves as an axis of the rotation.
The value is calculated based on the parent object.

The default value is 0.
Return
None
Remark 
- The execution of this method does not change position-related properties (eg. x and y).
   As for the methods that return the information on areas such as getRect(), they return the areas to which the rotate() method has applied.

- This method is executed based on the position where a GraphicsText is displayed.
   Accordingly, results of rotating a GraphicsText will be accumulated if this method is executed multiple times.

- For a GraphicsGroup, the effect of rotation applies to its child objects equally.

- A GraphicsText must have its parent if this method is to be executed successfully.
   That is, you should add a GraphicsText to its parent by using the addChild() method before the execution of this method.
