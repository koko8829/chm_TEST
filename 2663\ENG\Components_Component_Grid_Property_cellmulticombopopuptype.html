Property Type

Description 
This property sets how the item list is displayed when the cell is in the MultiCombo format.
Syntax 
Grid.cellmulticombopopuptype[= enumType]
Setting Syntax 
enumType ::= 'none' | 'normal' | 'center'
 
this.Grid00.cellmulticombopopuptype = "center";
"none" 
The item list is not displayed on the screen.
"normal" 
The general rules for displaying a list of items are followed.
"center" 
A list of items is displayed in the center of the screen where the application is running.
Remark 
- If the cellmulticombopopuptype property value is not set, it is applied as “normal".

- If the cell’s multicombopopuptype property value is not set, the Grid’s cellmulticombopopuptype property value is applied.

- This property should be used limitedly in special cases such as the mobile environment.

- If there are multiple monitors, the list of items is displayed based on the monitor on which the Grid is displayed.
