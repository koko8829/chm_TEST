Property Type
Enum Expr Control Hidden ReadOnly Bind Collection StringResource
Description 
This read-only property holds an integer indicating the cause of an event in a BluetoothLE object.
Syntax 
BluetoothLEEventInfo.reason
Setting Syntax 
var nReason = e.reason;
10 
This value suggests that the cause of an event is a scan initiated by the scanStart() method.
11 
This value suggests that the cause of an event is a scan that finished due to the timeout set by the scanStart() method.
12 
This value suggests that the cause of an event is the discovery of  the device that provides a certain service. The scan here is conducted by the scanStart() method.
20 
This value suggests that the cause of an event is the termination of a scan that was stopped by the scanStop() method.
30 
This value suggests that the cause of an event is connection to a device taken by the connect() method.
40 
This value suggests that the cause of an event is disconnection from a device taken by the disconnect() method.
50 
This value suggests that the cause of an event is a request made by the discoverService() method to obtain the list of services provided by the device.
51 
This value suggests that the cause of an event is the reception of a service list requested by the discoverService() method. The list is provided by the device.
60 
This value suggests that the cause of an event is a request for a notification made by the subscribe() method.
61 
This value suggests that the cause of an event is the reception of a notification requested by the subscribe() method.
70 
This value suggests that the cause of an event is the cancellation of notification. The cancellation here is conducted by the unsubscribe() method.
80 
This value suggests that the cause of an event is a request for a characteristic of a certain service. The request here is made by the readCharacteristic() method.
90 
This value suggests that the cause of an event is the saving of a characteristics of a certain service, conducted by the writeCharacteristic() method.
