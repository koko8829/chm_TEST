Description 
This is an invisible object designed to upload files to the server.
Remark 
- Files are uploaded to the server in the "POST" manner by using the HTTP/HTTPS protocol.

- This component sends the following to the server: one or more files set by the filelist property as well as the data set by the postdatalist property.

- The onprogress event will occur while files are uploaded. The interval between event occurrences varies according to browsers.

Property
Name Description
filelist This read-only property holds a collection that contains the information on the files to be uploaded from a FileUpTransfer to the server.
id This property is used to set the identification of a FileUpTransfer.
name This property is used to specify the name of a FileUpTransfer.
postdatalist This read-only property holds the data that will be sent to the server. The data, which takes the form of a collection, is used to request file upload.
url This property is used to define a server location onto which a FileUpTransfer will upload files.
Method
Name Description
addEvent This method is used to declare a certain event and then add it to a FileUpTransfer.
addEventHandler This method is used to add a handler to a certain event of a FileUpTransfer.
addEventHandlerLookup This method is used to find a function and add it to a certain event of a FileUpTransfer as a handler.
addFile This method is used to add a file item to the end position of the filelist property.
clearEventHandler This method is used to delete all the event handlers registered in a certain event.
clearFileList This method is used to delete all the file items held by the filelist property.
clearPostDataList This method is used to remove all the entries of the data set by the postdatalist property.
destroy This method is used to delete a FileUpTransfer that is dynamically created through a script.
existFile This method returns the existence of a file item that matches a VirtualFile object delivered as an argument. The file item here is held by the filelist property.
findEventHandler This method finds a handler in a certain event of a FileUpTransfer and returns the index of the handler.
getEventHandler This method returns a specific handler from a certain event of a FileUpTransfer by specifying the index number of the handler.
getFileArrayByFileName This method returns an array whose elements are VirtualFile objects that match specified file names. The specified files are among the file items held by the filelist property.
getIndexArrayByFileName This method returns an array whose elements are index numbers of the file items that match specified file names. The file items here are held by the filelist property.
getPostData This method returns the entry of a certain key from the data set by the postdatalist property.
insertEventHandler This method is used to add a handler to a certain event of a FileUpTransfer at a specified position.
removeEvent This method is used to remove a certain event that is declared in a FileUpTransfer.
removeEventHandler This method is used to remove a handler in a certain event of a FileUpTransfer.
removeEventHandlerLookup This method is used to find a handler in a certain event of a FileUpTransfer and remove it.
removeFile This method is used to remove a file item that matches a VirtualFile object delivered as an argument. The file item here is held by the filelist property.
removeFileByIndex This method is used to remove a file item whose index number matches a number delivered as an argument. The file item here is held by the filelist property.
removePostData This method is used to remove the entry of a certain key from the data set by the postdatalist property.
setEventHandler This method is used to change the first handler in a certain event of a FileUpTransfer.
setEventHandlerLookup This method finds a function and uses it to replace the first handler in a certain event of a FileUpTransfer.
setFile This method is used to change a file item that is held by the filelist property. The index number of the item is delivered as an argument.
setPostData This method is used to add data to the postdatalist property, whose data will be sent to the server to request file upload.
upload This method is used to upload the files set by the filelist property in the "POST" manner by using the HTTP/HTTPS protocol.
Event
Name Description
onerror This event occurs when file upload fails or an error code is returned.
onprogress This event occurs at regular intervals while files are uploaded.
onsuccess This event occurs when a success code is delivered after the uploading of files.
