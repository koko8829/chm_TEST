Description 
The PopupDiv is a component used to compose multiple subscreens on one screen.

The screen can be composed independently or connected with the main screen if composed with the PopupDiv.
The PopupDiv is similar to the Div component but automatically disappears once it loses the focus.
Remark 
- The PopupDiv can make screen composition easy and increase the reusability of a form by dividing the screen.
  However, if the number of stages of the PopupDiv increases, the screen composition slows down and the initialization time increases.

- It is efficient to use the PopupDiv with a depth of around 2 to 3 stages.
  It is faster to use the PopupDiv with 2 to 3 stages on one screen than with 4 to 5 stages when composing a screen.
Structure
Contents Sizing 
The below table describes how to size a PopupDiv.
NULL 
PopupDiv Component  * A vscrollbar and hscrollbar controls will appear if child components are placed out of the PopupDiv's area.
Left 
The related argument of the trackPopup or trackPopupByComponent method will be applied. If the applied position of the PopupDiv's left side is placed outside the area where the PopupDiv is supposed to appear, the left property will be re-calculated so that it can appear within the area.
Top 
The related argument of the trackPopup or trackPopupByComponent method will be applied. If the applied position of the PopupDiv's top side is placed outside the area where the PopupDiv is supposed to appear, the top property will be re-calculated so that it can appear within the area.
Width 
The related argument of the trackPopup or trackPopupByComponent method is applied. If no related argument is specified, the width will adopt the value specified at the design stage. If the applied width is bigger than the area where the PopupDiv is displayed, the width will be re-adjusted accordingly.
Height 
The related argument of the trackPopup or trackPopupByComponent method is applied. If no related argument is specified, the height will adopt the value specified at the design stage. If the applied height is bigger than the area where the PopupDiv is displayed, the height will be re-adjusted accordingly.
NULL 
Form
Left 
The value of the left property is 0.
Top 
The value of the top property is 0.
Width 
The width of the PopupDiv is applied.
Height 
The value of the height property of the PopupDiv is applied.

Constructor 
PopupDiv.init( strName, vLeft, vTop , vWidth, vHeight [, vRight, vBottom, [vMinWidth, [vMaxWidth, [vMinHeight, [vMaxHeight]]]]] )
Parameters Type Description
strName String 
It sets the ID of a PopupDiv in the form of a string.
vLeft String, Number 
It specifies the value of the left property of a PopupDiv in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the right property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vTop String, Number 
It specifies the value of the top property of a PopupDiv in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the bottom property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vWidth String, Number 
It specifies the value of the width property of a PopupDiv in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vHeight String, Number 
It specifies the value of the height property of a PopupDiv in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vRight String, Number 
It specifies the value of the right property of a PopupDiv in pixel or percent (%).

This parameter will be ignored if both the vLeft and vWidth parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the left property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vBottom String, Number 
It specifies the value of the bottom property of a PopupDiv in pixel or percent (%).

This parameter will be ignored if both the vTop and vHeight parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the top property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vMinWidth String, Number 
It specifies a PopupDiv's minimum width that can be displayed in the Form.
vMaxWidth String, Number 
It specifies a PopupDiv's maximum width that can be displayed in the Form.

If this value is smaller than vMinWidth, it will be replaced by the value of vMinWidth.
vMinHeight String, Number 
It specifies a PopupDiv's minimum height that can be displayed in the Form.
vMaxHeight String, Number 
It specifies a PopupDiv's maximum height that can be displayed in the Form.

If this value is smaller than vMinHeight, it will be replaced by the value of vMinHeight.
Sample Call:
var objComp = new PopupDiv();

objComp.init( "PopupDiv00", 30, 120, 196, 46 );
objComp.init( "PopupDiv00", 30, 120, 196, 46, null, null );
objComp.init( "PopupDiv00", null, null, 300, "400px", "80%", 300 );
objComp.init( "PopupDiv00", 0, 0, 200, 100, null, null, 300, 500, 200, 500 );
objComp.init( "PopupDiv00", "PopupDiv22:10", 300, null, null, "PopupDiv33:10", "20%", 300, 500, 200, 500 );
 
// Create Object  
var objPopupDiv = new PopupDiv();  
objPopupDiv.init("PopupDiv", 30, 120, 196, 46, null, null);
objPopupDiv.init("PopupDiv", null, null, 300, "400px", "80%", 300);
or
var objPopupDiv = new PopupDiv("PopupDiv", 30, 120, 196, 46, null, null);
var objPopupDiv = new PopupDiv("PopupDiv", null, null, 300, "400px", "80%", 300);
 
// Add Object to Parent Form  
this.addChild("PopupDiv", objPopupDiv); 
 
// Insert Object to Parent Form  
this.insertChild(1, "PopupDiv", objPopupDiv); 
 
// Remove Object form Parent Form  
this.removeChild("PopupDiv"); 
 
// Show Object  
objPopupDiv.show(); 
 
// Destroy Object  
objPopupDiv.destroy(); 
objPopupDiv = null;
