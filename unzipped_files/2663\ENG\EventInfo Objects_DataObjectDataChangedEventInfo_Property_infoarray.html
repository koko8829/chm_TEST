Property Type

Description 
Gets an array of type Infoarray containing the change information.
Remark 
For each property of Infoarray, refer to the table below:

	

		Property
		Type
		Description
	


	

		
data

		
Object

		
changed data

	


	

		
dataobjectpath

		
String

		
The value of the dataobjectpath property set in the Dataset object where the change occurred.

	


	

		
index

		
Number

		
The Row Index value of the Dataset object where the change occurred.

	


	

		
sourcedatasetid

		
String

		
The value of the Dataset object id property where the change occurred.

	


	

		
type

		
String

		
The state in which the change occurred
- update: Change the Column value of a specific Row
- add: Add new data (when executing addRow method of Dataset object)
- insert: Add new data (when insertRow method of Dataset object is executed)
- delete: delete data (when executing deleteRow method of Dataset object)

	


	

		
uid

		
String

		
Unique identification id
- It is used to delete specific data using the removeChangedInfo method.
