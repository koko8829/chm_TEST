Description 
TextField is a component for receiving single-line string inputs from the user.
Remark 
- As for the TextField component, the input format can be specified by setting the inputtype or pattern property value.
   You can validate input values based on their input format.

- To receive inputs of multi-line strings, the MultiLineTextField component must be used.
Structure
Contents Sizing 
This is the basis for determining the size of the TextField component.
 
TextField Component
Width 
The value of the width property of the TextField component will be applied. 
If it is smaller than the width of the internal content, the internal content will only be partially displayed.
Height 
The value of the height property of the TextField component will be applied. 
If it is smaller than the height of the internal content, the internal content will only be partially displayed. 
label Control
Width 
The automatically calculated width value of the label control text will be applied.
Height 
The automatically calculated height value of the label control text will be applied.
 
charcounttext Control
Left 
The width of the TextField component minus the width of the charcounttext control will be applied.
Top 
The height of the TextField component minus the height of the charcounttext control will be applied.
Width 
The automatically calculated width value of the charcounttext control text will be applied.
Height 
The automatically calculated height of the charcounttext control text will be applied.
 
helpertext Control
Left 
0 will be applied.
Top 
The height of the TextField component minus the height of the helpertext control will be applied.
Width 
The width of the TextField component minus the width of the charcounttext control will be applied.
Height 
The height of the TextField component minus the height of the helpertext and charcounttext controls, whichever is greater, will be applied.
 
nulltext Control
Left 
0 will be applied.
Top 
0 will be applied.
Width 
The width of the TextField component minus the sum of the widths of the prefixtext, postfixtext, leadingbutton, and trailingbutton controls will be applied.
Height 
The height of the TextField component will be applied.
 
box control
Left 
0 will be applied.
Top 
The height of the TextField component minus the height of the area reserved for the floating of the label control will be applied.
Width 
The width of the TextField component will be applied.
Height 
The height of the TextField component minus the height of the area reserved for the floating of the label control and the height of the helpertext control will be applied.
 
prefixtext control
Left 
0 will be applied.
Top 
0 will be applied.
Width 
The automatically calculated width value of the prefixtext control will be applied.
Height 
The height of the TextField component will be applied.
 
postfixtext control
Left 
The width of the TextField component minus the sum of the widths of the postfixtext and trailingbutton controls will be applied.
Top 
0 will be applied.
Width 
The automatically calculated width value of the postfixtext control will be applied.
Height 
The height of the TextField component will be applied. 
leadingbutton control
Left 
0 will be applied.
Top 
0 will be applied.
Width 
The value of the iconbuttonsize property will be applied. 
If there is no value for the iconbuttonsize property, the height of the TextField component will be applied.
Height 
The value of the iconbuttonsize property will be applied. 
If there is no value for the iconbuttonsize property, the height of the TextField component will be applied.
 
trailingbutton control
Left 
The width of the TextField component minus the width of the trailingbutton control will be applied.
Top 
0 will be applied.
Width 
The value of the iconbuttonsize property will be applied. If there is no value for the iconbuttonsize property, the height of the TextField component will be applied.
Height 
The value of the iconbuttonsize property will be applied. If there is no value for the iconbuttonsize property, the height of the TextField component will be applied.
Basic Key Action
Action Key Description
Move focus Tab 
The focus will move to the next component according to the tab sequence.
Move caret Left and right arrow keys 
When the caret is displayed, the caret will be moved left/right.
Select text Shift + left and right arrow keys 
When the caret is displayed, the input text will be selected in the left/right direction.
Constructor 
TextField.init( strName, vLeft, vTop , vWidth, vHeight [, vRight, vBottom, [vMinWidth, [vMaxWidth, [vMinHeight, [vMaxHeight]]]]] )
Parameters Type Description
strName String 
It sets the ID of a TextField in the form of a string.
vLeft String, Number 
It specifies the value of the left property of a TextField in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the right property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vTop String, Number 
It specifies the value of the top property of a TextField in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the bottom property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vWidth String, Number 
It specifies the value of the width property of a TextField in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vHeight String, Number 
It specifies the value of the height property of a TextField in pixel or percent (%).

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be specified in pixel while the base component will be ignored.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vRight String, Number 
It specifies the value of the right property of a TextField in pixel or percent (%).

This parameter will be ignored if both the vLeft and vWidth parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the left property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the width property of the base component.
vBottom String, Number 
It specifies the value of the bottom property of a TextField in pixel or percent (%).

This parameter will be ignored if both the vTop and vHeight parameters are all specified.

* When you include a base component in the argument:
If you use a pixel as an unit, this parameter will be determined based on the top property of the base component.
If you use a percentage (%) as an unit, this parameter will be determined based on the height property of the base component.
vMinWidth String, Number 
It specifies a TextField's minimum width that can be displayed in the Form.
vMaxWidth String, Number 
It specifies a TextField's maximum width that can be displayed in the Form.

If this value is smaller than vMinWidth, it will be replaced by the value of vMinWidth.
vMinHeight String, Number 
It specifies a TextField's minimum height that can be displayed in the Form.
vMaxHeight String, Number 
It specifies a TextField's maximum height that can be displayed in the Form.

If this value is smaller than vMinHeight, it will be replaced by the value of vMinHeight.

var objComp = new TextField();

objComp.init( "TextField00", 30, 120, 196, 46 );
objComp.init( "TextField00", 30, 120, 196, 46, null, null );
objComp.init( "TextField00", null, null, 300, "400px", "80%", 300 );
objComp.init( "TextField00", 0, 0, 200, 100, null, null, 300, 500, 200, 500 );
objComp.init( "TextField00", "TextField22:10", 300, null, null, "TextField33:10", "20%", 300, 500, 200, 500 );
 
// Create Object  
var objTextField = new TextField();  
objTextField.init("TextField", 30, 120, 196, 46, null, null);
objTextField.init("TextField", null, null, 300, "400px", "80%", 300);
or
var objTextField = new TextField("TextField", 30, 120, 196, 46, null, null);
var objTextField = new TextField("TextField", null, null, 300, "400px", "80%", 300);
 
// Add Object to Parent Form  
this.addChild("TextField", objTextField); 
 
// Insert Object to Parent Form  
this.insertChild(1, "TextField", objTextField); 
 
// Remove Object form Parent Form  
this.removeChild("TextField"); 
 
// Show Object  
objTextField.show(); 
 
// Destroy Object  
objTextField.destroy(); 
objTextField = null;
