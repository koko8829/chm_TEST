Description 
This is an object that calls, creates, and manages data.
It supports REST API call function and one-way binding with Dataset objects.
Remark 
- When data are loaded successfully, events will occur in the following order:
request: DataObject.onsuccess -> DataObject.onload -> Dataset.onload
load: DataObject.onload -> Dataset.onload

- When data communication fails, events will occur in the following order:
Environment.onerror (Return false) -> DataObject.onerror
Environment.onerror (Return true)
Property
Name Description
data The attribute in an object form with data information.
enableevent This property is used to determine whether to trigger events in a DataObject.
id This property is used to set the identification of a DataObject.
name This property is used to specify the name of a DataObject.
parent This read-only property holds the parent object of a DataObject.
preload The attribute to set the point to load data when setting the url attribute value.
url The attribute to set the URL to call data.
Method
Name Description
addEvent This method is used to declare a certain event and then add it to a DataObject.
addEventHandler This method is used to add a handler to a certain event of a DataObject.
addEventHandlerLookup This method is used to find a function and add it to a certain event of a DataObject as a handler.
clearChangedInfo This method deletes all change information.
clearEventHandler This method is used to delete all the event handlers registered in a certain event.
destroy This method is used to delete a DataObject that is dynamically created through a script.
findEventHandler This method finds a handler in a certain event of a DataObject and returns the index of the handler.
getAllResponseHeaders This method returns response header information after calling the request method.
getChangedInfo This method returns the saved change information.
getEventHandler This method returns a specific handler from a certain event of a DataObject by specifying the index number of the handler.
getObjectByPath The method to return the data object filtered by jsonpath value.
getResponse This method returns response data information after calling the request method.
getResponseHeader This method returns a specific response header value after calling the request method.
insertEventHandler This method is used to add a handler to a certain event of a DataObject at a specified position.
load The method to load data from the service set to the url attribute.
removeChangedInfo This method is used to remove the item information corresponding to a specific uid from the change information.
removeEvent This method is used to remove a certain event that is declared in a DataObject.
removeEventHandler This method is used to remove a handler in a certain event of a DataObject.
removeEventHandlerLookup This method is used to find a handler in a certain event of a DataObject and remove it.
request The method to process communication according to the set attribute value.
serializeToString The method to return data as string.
setEventHadlerLookup This method finds a function and uses it to replace the first handler in a certain event of a DataObject.
setEventHandler This method is used to change the first handler in a certain event of a DataObject.
Event
Name Description
ondatachanged This event occurs when the data of the DataObject object is changed.
onerror This event occurs when communication fails after calling the load or request method.
onload The event occurring upon completion of data loading.
onloaddatasets This is an event that occurs after data loading of Dataset object is processed when DataObejct object is set to the binddataobject property value of Dataset object.
onsuccess The event occurring when communication is successful after calling the request method
onvaluechanged This event occurs when the value of the DataObject object (the value corresponding to a specific key in JSON data) is changed.
