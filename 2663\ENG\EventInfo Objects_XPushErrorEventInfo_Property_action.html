Property Type
Enum Expr Control Hidden ReadOnly Bind Collection StringResource
Description 
This property holds a code that indicating the cause of the event that occurred.
Syntax 
XPushErrorEventInfo.action
Setting Syntax 
var nCode = e.action ;
"0" 
This value indicates that an X-PUSH server has failed to be connected by the connect() method.

This value is treated as the constant "nexacro.XPushAction.AUTH".
"1" 
This value indicates that an X-PUSH server has failed to be disconnected by the disconnect() method.

This value is treated as the constant "nexacro.XPushAction.BYEC".
"2" 
This value indicates that a topic has failed to be registered by the subscribe() method.

This value is treated as the constant "nexacro.XPushAction.ADDF".
"3" 
This value indicates that a topic has failed to be removed by the unsubscribe() method.

This value is treated as the constant "nexacro.XPushAction.DELF".
"4" 
This value indicates that a reliable messaging message has failed to be requested by the requestMessage() method.

This value is treated as the constant "nexacro.XPushAction.REQD".
"5" 
This value indicates that the requestMessage() method has failed to respond to an unreceived reliable messaging message.

This value is treated as the constant "nexacro.XPushAction.RECT".
"6" 
This value indicates that a device token has failed to be registered in an X-PUSH server by the requestMessage() method.

This value is treated as the constant "nexacro.XPushAction.RGST".
"7" 
This value indicates that a device token has failed to be unregistered from an X-PUSH server by the unregisterDevice() method.

This value is treated as the constant "nexacro.XPushAction.UNRG".
"8" 
This value indicates that the registerTopic() method has failed to register a topic for reliable messaging.

This value is treated as the constant "nexacro.XPushAction.ADUI".
"9" 
This value indicates that the unregisterTopic() method has failed to unregister a topic for reliable messaging.

This value is treated as the constant "nexacro.XPushAction.UNUI".
"10" 
This value indicates that requestMessageCount() method has failed to request the number of reliable messaging messages that are not received yet.

This value is treated as the constant "nexacro.XPushAction.MSGC".
"11" 
This value indicates that the getRegisteredTopics and getRegisteredTopicsWithUserID method has failed to search a topic information.
This value is treated as the constant "nexacro.XPushAction.RTID".
Example
Source 
this.XPush00_onerror = function(obj:XPush,  e:XPushErrorEventInfo)
{
   if(e.action == nexacro.XPushAction.ADDF)
   {
      alert("function == subscribe()");
   }
}
