Description 
As a frame subordinate to a MainFrame, a TileFrameSet is a basic unit for arranging Form screens in an application.

A TileFrameSet can include one or more of FrameSets, HFrameSets, VFrameSets, TileFrameSets and ChildFrames internally.
A TileFrameSet automatically arranges its subordinate frames horizontally or vertically.
Structure
Constructor 
TileFrameSet.init( strName, nLeft, nTop, nWidth, nHeight [, nRight, nBottom] )
Parameters Type Description
strName String 
This is a string that represents the ID of a TileFrameSet.
nLeft Number 
This parameter is used to set the left property of the TileFrameSet based on the top-left side of the parent frame.
The value can be expressed in pixel or percent (%).
nTop Number 
This parameter is used to set the top property of the TileFrameSet based on the top-left side of the parent frame.
The value can be expressed in pixel or percent (%).
nWidth Number 
This parameter is used to set the width property of the TileFrameSet.
The value can be expressed in pixel or percent (%).
nHeight Number 
This parameter is used to set the height property of the TileFrameSet.
The value can be expressed in pixel or percent (%).
nRight Number 
This parameter is used to set the right property of the TileFrameSet based on the bottom-right side of the parent frame.
The value can be expressed in pixel or percent (%).
If you set the nLeft and nWidth parameters all, this parameter will be ignored.
nBottom Number 
This parameter is used to set the bottom property of the TileFrameSet based on the bottom-right side of the parent frame.
The value can be expressed in pixel or percent (%).
If you set the nTop and nHeight parameters all, this parameter will be ignored.
Sample Call:
var obFrame = new TileFrameSet();

obFrame.init( "TileFrameSet00", 30, 120, 196, 46, null, null);
obFrame.init( "TileFrameSet00", null, null, 300, "400px", "80%", 300);
 
// Create Object  
var objTileFrameSet = new TileFrameSet();  
objTileFrameSet.init("TileFrameSet", 30, 120, 196, 46, null, null);
objTileFrameSet.init("TileFrameSet", null, null, 300, "400px", "80%", 300);
or
var objTileFrameSet = new TileFrameSet("TileFrameSet", 30, 120, 196, 46, null, null);
var objTileFrameSet = new TileFrameSet("TileFrameSet", null, null, 300, "400px", "80%", 300);
 
// Add Object to Parent Frame  
this.addChild("TileFrameSet", objTileFrameSet); 
 
// Insert Object to Parent Frame  
this.insertChild(1, "TileFrameSet", objTileFrameSet); 
 
// Remove Object form Parent Frame  
this.removeChild("TileFrameSet"); 
 
// Show Object  
objTileFrameSet.show(); 
 
// Destroy Object  
objTileFrameSet.destroy(); 
objTileFrameSet = null;
