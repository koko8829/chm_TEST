Description 
This object is used to work with dates and times.
Remark 
- The following methods are provided by Nexacro on top of the methods subordinate to the Script object: toLocaleDateString(), toLocaleString() and toLocaleTimeString ().

- As this object is provided by Nexacro, you must specify the class "nexacro" when you write a code to create this object.

- If you do not set the time when creating a Date object, the hour, minute and second will be all set at 0.

- With the datatyperule set to "2.0", you must set "undefined" as a parameter to create a Date object that has an invalid value.
Constructor 
new nexacro.Date( [nYear [, nMonth [, nDay [, nHours [, nMinutes [, nSeconds [,nMillisecond]]]]]]] )
Parameters Type Description
nYear Number 
It is a four-digit value representing the year.

If you specify a value ranging from 0 to 99, the value will be added to 1900.
nMonth Number 
This is a code value representing a month. The value ranges from 0 (January) to 11 (December).
nDay Number 
This is a code value representing a day. The value ranges from 1 to 31.
nHours Number 
This is a code value representing an hour. 

The value ranges from 0 (midnight) to 23 (11p.m.).
nMinutes Number 
This is a code value representing a minute. The value ranges from 0 to 59.
nSeconds Number 
This is a code value representing a second. The value ranges from 0 to 59.
nMillisecond Number 
This is a code value representing a millisecond. The value ranges from 0 to 999. 
// Create Object (Valid Value)
var objDate1 = new nexacro.Date();
var objDate2 = new nexacro.Date( 2000,1,1 );
var objDate3 = new nexacro.Date( 2000,1,1,9,0,0,0 );

// Create Object (Invalid Value)
var objDate4 = new nexacro.Date( undefined );
