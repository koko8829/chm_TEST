query

Description 
This method is used to search for the contacts that satisfy certain search conditions. This method is applicable to mobile devices.
Syntax 
ContactSet.query( strQueryInfo [, nResultCnt] )
Parameters
Parameters Type Description
strQueryInfo String 
This is a string representing the conditions for the contact search. The conditions are derived from the contact information.

The argument is formatted as "field name:search word;field name:search word;...".
You can use wild card characters like ? and * in search words.
nResultCnt Number 
This is the number of contacts to return among the contacts that satisfy the conditions.

If you specify -1, all the satisfying contacts will be returned.
Sample Call:
this.ContactSet00.query( "contactname:John;phonenumbers:*2214", 5 );
Return
Type Description
Boolean 
If this method is executed successfully, this method will return "true".

If this method fails to be executed or the parameter is invalidly set, this method will return "false".
Remark 
- If a contact search is successful in a mobile device, the onsuccess event will occur.
  Otherwise, the onerror event will occur.
