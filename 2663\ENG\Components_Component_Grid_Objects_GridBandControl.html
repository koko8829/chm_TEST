Description 
This object is used to specify Grid bands.

A Grid consists of the head, body and summary bands. Each band has its own list of cells.
The head band constitutes the cells put at the top of a Grid while the body band prints the rows of a Dataset bound to the Grid. The summary band is comprised of the cells placed at the bottom of the Grid.
Remark 
- If the style of the head band is not defined in the CSS files or in themes, the head band will adopt the style of the body band.
 If the style of the summary band is not defined in the CSS files or in themes, the summary band will adopt the style of the head band.
 If the styles of the head and summary bands are not defined in the CSS files or in themes, both bands will adopt the style of the body band.
