Property Type
Enum Expr Control Hidden ReadOnly Bind Collection StringResource
Description 
This read-only property holds an integer that indicates the cause of the event.
Syntax 
DSRowsetChangeEventInfo.reason
Setting Syntax 
var nReason = e.reason;
Remark 
- This property indicates the cause of the onrowsetchanged event.
