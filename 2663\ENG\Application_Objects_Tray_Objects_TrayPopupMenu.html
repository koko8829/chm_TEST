TrayPopupMenu

Android Default Web Browser iOS/iPadOS Default Web Browser
Description 
This object represents a pop-up menu of a Tray.
This object uses a pop-up menu provided as a system API.
Property
Name Description
captioncolumn This property is used to specify a column of a dataset that is designated as an innerdataset.The column contains the texts of pop-up menu items.
checkboxcolumn The attribute to set the column defining whether to display the check image on the sub-menu item.
enablecolumn This property is used to designate a Dataset column that defines whether to activate each pop-up menu item. The Dataset must be set by the innerdataset property.
hotkeycolumn This property is used to designate a dataset column for the hotkeys for menu items. The dataset is bound to the innerdataset property.
iconcolumn The attribute to set the column defining the icon information displayed on the pop-up menu item.
id This property is used to set the identification of a TrayPopupMenu.
idcolumn This property is used to specify a column of a dataset that is designated as an innerdataset. The column contains the IDs of pop-up menu items.
innerdataset This property is used to specify the ID of a dataset that defines tray pop-up menu items.
levelcolumn This property is used to specify a column of a dataset that is designated as an innerdataset. The column contains the levels of pop-up menu items.
userdatacolumn This property is used to designate a Dataset column to process user data. The Dataset must be set by the innerdataset property.
Method
Name Description
trackPopup This method is used to display a Tray popup menu at the location of a mouse cursor.
Event
Name Description
oninnerdatachanged This event occurs when there has been a change in the data of a Dataset that is bound to the innerdataset property of a TrayPopupMenu.
onmenuclick This event occurs when you click a menu item on the Tray with the left button of a mouse.
