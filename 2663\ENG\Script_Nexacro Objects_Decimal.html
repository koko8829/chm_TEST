Description 
This object is used to handle decimal type data.
Remark 
- For calculation, a Decimal object must use provided methods instead of the following arithmetic operators: +, -, * and /.

- This object is provided to process a Dataset column of the "BIGDECIMAL" type.

- This object can handle a maximum of 24 digits.
