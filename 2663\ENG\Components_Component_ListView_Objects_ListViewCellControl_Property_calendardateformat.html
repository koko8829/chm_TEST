Description 
This property is used to set the format of a displayed date when the displaytype property of a cell is set to "date" or "calendarcontrol".
Syntax 
ListViewCellControl.calendardateformat[= strFormat]
Setting Syntax 
strFormat ::= <maskchar> | 'LONGDATE' | 'SHORTDATE'

<maskchar> ::= [<Year>] [<Month>] [<Day>] [<Week>] [<Hour>] [<Minute>] [<Second>] [<MiliSec>]


* <maskchar> is a combination of mask characters for receiving a date input. The elements can be arranged in different orders while each of them can be omitted.
 
this.ListView00.setCellProperty( "body", "cell00", "calendardateformat", "yyyy-MM-dd ddd"); 

this.ListView00.setCellProperty( "body", "cell00", "calendardateformat", "yyyy-MM-dd ddd HH:mm:ss");
<Year> 
It specifies a mask for displaying a year.

"yy" for displaying the last two digits of a year
"yyyy" for displaying all the four digits of a year
<Month> 
It specifies a mask for displaying a month.

If it specifies "M", the cell applies the one-digit format for January to September while using the two-digit format for October to December.
If it specifies "MM", the cell applies the two-digit format for all the months while January to September begin with 0.
If it specifies "MMMM", the cell displays the month in the format defined by Nexacro.
<Day> 
It specifies a mask for displaying a day.

If it specifies "d", the cell applies the one-digit format for the first to ninth day while using the two-digit format for the remaining days.
If it specifies "dd", the cell applies the two-digit format for all the days while the first to ninth
 day begin with 0.
<Week> 
It specifies a mask for displaying a weekday.

"ddd" for displaying a abbreviated weekday name
"dddd" for displaying a weekday name that is fully spelled out
<Hour> 
It specifies a mask for displaying an hour.

If it specifies "h", the cell applies the one-digit format for one to nine while using the two-digit format for the remaining hours. It is based on the 12-hour format.
If it specifies "hh", the cell applies the two-digit format for all the hours while one to nine begin with 0. It is based on the 12-hour format.
If it specifies "H", the cell applies the one-digit format for one to nine while using the two-digit format for the remaining hours. It is based on the 24-hour format.
If it specifies "HH", the cell applies the two-digit format for all the hours while one to nine begin with 0. It is based on the 24-hour format.
<Minute> 
It specifies a mask for displaying a minute.

If it specifies "m", the cell applies the one-digit format for one to nine while using the two-digit format for the remaining minutes.
If it specifies "mm", the cell applies the two-digit format for all the minutes while one to nine begin with 0.
<Second> 
It specifies a mask for displaying a second.

If it specifies "s", the cell applies the one-digit format for one to nine while using the two-digit format for the remaining seconds.
If it specifies "ss", the cell applies the two-digit format for all the seconds while one to nine begin with 0.
<MiliSec> 
It specifies a mask for displaying the millisecond.

If it specifies "sss", the cell applies the three-digit format for all the milliseconds while one to ninety-nine begin with 0.
"LONGDATE" 
It derives the long date formats from the regional settings defined by Nexacro.
"SHORTDATE" 
It derives the short date formats from the regional settings defined by Nexacro.
Remark 
- The default value of this property is "yyyy-MM-dd ddd".

- You should use the calendareditformat property to specify the date format applied to an editable cell.

- If you specify characters not defined as a mask, those will be displayed as they are while they are uneditable.
