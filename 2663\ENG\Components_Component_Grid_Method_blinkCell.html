Description 
This method is used to render an effect where a certain cell of a Grid blinks for a specified duration.
Syntax 
Grid.blinkCell( nRow, vCell, nKeepMSec, nBlinkCnt );
Parameters
Parameters Type Description
nRow Number 
This is the index number of a data row that the blinking effect will apply to.

The method will not be executed if you set a negative, invalid or non-existent index number.
vCell String, Number 
This is the index number or ID of a cell that the blinking effect will apply to.

If you specify an index number, you can apply the blinking effect only to one cell.
If you want to specify more than one cell, you need to divide them with commas (,).

The method will not be executed if you set a negative number, non-existent index number, empty value or non-existent ID.
nKeepMSec Number 
This is a duration for the blinking effect, specified in millisecond.

If you specify 0, this method will cancel the blinking effect set for the cell designated by the nRow and vCell parameters.

The method will not be executed if you set a negative number or invalid value.
nBlinkCnt Number 
This parameter indicates how many times a cell blinks during a specified duration.

If you specify 0, this method will cancel the blinking effect set for the cell designated by the nRow and vCell parameters.

The method will not be executed if you set a negative number or invalid value.
Sample Call:
* XCSS
.Grid .body .row .cell[userstatus=blinked]
{
	background : #125dae;
	color : #ffffff;
}

* Script
this.Grid00.blinkCell( 0, 0, 10000, 20 );
this.Grid00.blinkCell( 0,"Column0,Column1", 10000, 20 );

for( var iCnt = 0; iCnt < this.Grid00.getCellCount( "body" ); iCnt++ )
{
	this.Grid00.blinkCell( 0, iCnt, 10000, 20 );
}
Return
None
Remark 
- To use the blinking effect, you need to define the "blinked" userstatus in a CSS file and then abort it.
Accordingly, you must define the "blinked" userstatus in a CSS file first to render the blinking effects.

- The blinking effect is rendered by evenly dividing the value of the nKeepMSec parameter by the value of the nBlinkCnt parameter. nKeepMSec and nBlinkCnt indicate the duration and times of blinking respectively.
 For example, assume that you set nKeepMSec and nBlinkCnt at 10,000 and 5 respectively. As a result, the "blinked" userstatus will be suspended for 1,000 milliseconds and then be applied for 1,000 milliseconds. Such a cycle will be repeated five times.

- The blinkCell() method sets the entire duration and frequency of the blinking effect.
Meanwhile, the blinkCellByInterval() method sets the frequency of the blinking effect and determines the interval between the occasions of the effect.

- When this method is executed, it cancels the existing definition of the blinking effect applied to the cell designated by the nRow and vCell parameters. Then, it applies the new definition.
 This method applies to each single cell.

- This method can apply to each cell.
 If you want to apply the blinking effect to an entire row, you should specify the IDs of the whole cells by dividing them with commas(,).
 Otherwise, you should call this method as many times as the number of the whole cells by using a For statement, as seen in the above sample call.
