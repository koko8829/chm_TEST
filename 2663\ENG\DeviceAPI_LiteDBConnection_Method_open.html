Description 
This is a method of connecting a database to LiteDBConnection.
Syntax 
LiteDBConnection.open( [strDataSource [,constOpenFlag]] )
Parameters
Parameters Type Description
strDataSource String 
This is a string representing the location of a database that will be connected.

The location is specified by using an absolute path or alias.
If you do not set this parameter, the value of the datasource property will be applied instead.
constOpenFlag Constant 
Set the options for connecting to the database.

In case LiteDBConnection.openReadWrite or 0 is set, the database will be connected to enable reading and writing.
In case LiteDBConnection.openReadWriteCreate or 1 is set, the database will be connected to enable reading, writing, and creating files. 

If the value is omitted, the openflag property value will be applied.
Sample Call:
this.LiteDBConnection00.open();
this.LiteDBConnection00.open("%USERAPP%datasource");
this.LiteDBConnection00.open("%USERAPP%datasource", LiteDBConnection.openReadWrite);
Return
Type Description
Boolean 
If the method execution is successful, true will be returned.

If the parameter value is set incorrectly, false will be returned.
Remark 
- An onsuccess event occurs when the database connection is successful.
    If the database connection fails, an onerror event occurs. 

- When the open() method is executed on a database that is already connected and open, it will be processed normally and an onsuccess event occurs.

- This is the actual Alias path that can be used in strDataSource.
