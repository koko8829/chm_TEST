read

Description 
This method is used to receive data from a connected server.
Syntax 
TCPClientSocket.read( nSize [, nTimeOut [, strCharSet]] )
Parameters
Parameters Type Description
nSize Number 
It is the number of bytes, presenting the maximum size of incoming data.

A specified value will be applied as the maximum size of incoming data while actually-received data can be smaller than the value.

If you specify 0, this method will only check whether there is incoming data.
If you specify a negative number, the calling of this method will be treated as a failure.
nTimeOut Number 
It is the number of seconds, representing the timeout for data reception.

If you specify 0 or smaller, the timeout will be infinite.

The default value is 0.
strCharSet String 
This parameter is used to define a character set that will be applied when data is received.

If you do not omit this parameter, the character set designated by the system will be applied.

It is recommended to use this parameter only when you need to change the character set.
Sample Call:
var arrRecvData;

arrRecvData = this.TCPClientSocket00.read( 1000 );
arrRecvData = this.TCPClientSocket00.read( 1000, 3 );
arrRecvData = this.TCPClientSocket00.read( 1000, 3, "utf-8" );
Return
Type Description
Array 
This method returns an array that contains a value indicating whether the method has succeeded in receiving data. The array also includes received data.

As the first element, a Boolean value indicates whether the method has succeeded in receiving data.
Received data is included as the second element.
Remark 
- The onerror event will occur if data reception fails.

- A returned array will have the second element only when this method succeeds in data reception and the received data is not empty.

- If you want to receive data by line, you need to use the readLine() method.
