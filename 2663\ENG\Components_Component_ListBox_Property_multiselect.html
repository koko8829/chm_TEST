Description 
This property is used to enable multiple selection in a LisxBox.
Syntax 
ListBox.multiselect[= bMultiselect]
Setting Syntax 
bMultiselect ::= 'true' | 'false'
true 
You can select multiple items from the list.

You can select multiple items by using the Ctrl or Shift key.
When you select multiple items, the index and value properties will have the last-selected item for their values.
When the value property is bound to a Dataset, the index and value properties will have an item selected in the Dataset.
false 
You can select only one item from the list.

The index and value properties will have the currently-selected item for their values.
When the value property is bound to a Dataset, the index and value properties will have an item selected in the ListBox.
Dragging that begins in the ListBox is valid only within the component.
Remark 
- The default value of this property "false".

- If a script or Dataset binding causes the change in the selected items with the multiselect property set to "true", the change will not apply to the screen.
 Therefore, note that there may be difference between the value of this property and selected items displayed on the screen.

â—† Restrictions according to the usage environment

In environments where the input device is touch, such as mobile, the multiselect function is partially supported.
- Bluetooth keyboard
  It can be selected multiple items with a combination of the direction key and shift key.
- Mouse + Bluetooth keyboard
  In the Android operating system, it can be selected multiple items by pressing Ctrl and Shift keys and clicking on the mouse.
(It is not applicable to iOS)
- Script
  When the multiselect property value is true, it can select multiple items using the setSelect method.

Multiple items cannot be selected in the following cases:
- Touch the screen
  Multiple items cannot be selected (limited by touch operation characteristics).
- Mouse
  Multiple items cannot be selected by mouse drag action.
- Touch screen + Bluetooth keyboard
  When pressing Ctrl and Shift keys and selecting an item through a touch action, multiple items cannot be selected because the key input status cannot be checked.
