TCPClientSocket

[Mobile]
Nexacro Runtime Environment (NRE)

Web Runtime Environment (WRE)
Android Default Web Browser iOS/iPadOS Default Web Browser
Description 
This is a client component that enables data transactions with a server by using the TCP.
Remark 
- This component serves as a socket for TCP communications, which adopts the one-on-one connection method.
Property
Name Description
address This read-only property holds the address of a connected server.
errorcode This read-only property holds an error code that indicates the type of the failure of a task requested by a TCPClientSocket component.
errormsg This read-only property holds an error message about the failure of a task requested by a TCPClientSocket component.
isopen This read-only property holds a value indicating whether a socket is connected to a server.
name This property is used to specify the name of a TCPClientSocket.
port This read-only property holds the port of a connected server.
Method
Name Description
addEvent This method is used to declare a certain event and then add it to a TCPClientSocket.
addEventHandler This method is used to add a handler to a certain event of a TCPClientSocket.
addEventHandlerLookup This method is used to find a function and add it to a certain event of a TCPClientSocket as a handler.
clearEventHandler This method is used to delete all the event handlers registered in a certain event of a TCPClientSocket.
close This method is used to request a server to disconnect from a socket
findEventHandler This method finds a handler in a certain event of a TCPClientSocket and returns the index of the handler.
getEventHandler This method returns a specific handler from a certain event of a TCPClientSocket by specifying the index number of the handler.
insertEventHandler This method is used to add a handler to a certain event of a TCPClientSocket at a specified position.
open This method is used to request a server to connect with a socket by using TCP.
read This method is used to receive data from a connected server.
readLine This method is used to receive data from a connected server by line.
removeEvent This method is used to remove a certain event that is declared in a TCPClientSocket.
removeEventHandler This method is used to remove a handler in a certain event of a TCPClientSocket.
removeEventHandlerLookup This method is used to find a handler in a certain event of a TCPClientSocket and remove it.
setEventHandler This method is used to change the first handler in a certain event of a TCPClientSocket.
setEventHandlerLookup This method finds a function and uses it to replace the first handler in a certain event of a TCPClientSocket.
write This method is used to send data to a connected server.
Event
Name Description
ondataarrived This event occurs when a connected server has data that you can receive.
onerror This event occurs when a task requested by a TCPClientSocket fails.
onsuccess This event occurs when a task requested by a TCPClientSocket succeeds.
